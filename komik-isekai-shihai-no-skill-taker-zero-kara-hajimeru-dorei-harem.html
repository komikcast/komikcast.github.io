Pembahasan

Angka Jepangnya : san ichi no zero ichi no ni zero zero desu.

Bentuk angkanya : 31-01-200

Cara baca dalam bahasa Indonesia : tiga satu nol satu dua nol nol.

_______________

san ichi no zero ichi no ni zero zero desu :

Masing-masing angka dalam bentuk kanjinya.

三 (san) = tiga = 3.一 (ichi) = satu = 1.零 (zero) = nol = 0.二 (ni) = dua = 2.

Adanya kata no yang terletak diantara tengah-tengah angka sebenarnya itu melambangkan tanda strip "—" dan tidak memiliki arti. Suka digunakan ketika kita menyebutkan suatu nomor, seperti nomor telepon, dll.

Adanya partikel です (desu) yang berada pada akhir setelah menyebutkan angka, karena memang fungsi partikel ini sebagai penekanan dan bentuk akhiran dalam kalimat.

_______________

8. Zero-ni-nana-go-no-nana-hachi-hachi-no-kyuu-roku-nana-desu.


Zero ni nana go no nana hachi hachi no kyuu roku nana desu artinya dalam bahasa Indonesia adalah nol dua tujuh lima tujuh delapan delapan sembilan enam tujuh (0275-788-967).

Pembahasan

Denwa Bangou「 でんわ ばんご 」

Arti nomor telepon dalam bahasa Jepang ialah denwa bangou. Materi kali ini membahas mengenai cara menyebutkan angka-angka atau menyebutkan nomor-nomor untuk nomor telepon dalam bahasa Jepang.

Seperti halnya yang terdapat dalam jawaban diatas,

Zero ni nana go no nana hachi hachi no kyuu roku nana desu artinya adalah 0275-788-967. Jika penulisan nomor teleponnya dalam bentuk hiragana yaitu :

ぜろ に なな ご の なな はち はち の きゅ ろく なな です (Zero ni nana go no nana hachi hachi no kyuu roku nana desu).

Pola kalimat yang digunakan untuk menyatakan denwa bangou adalah sebagai berikut :

S + partikel の (no) + denwa bangou + partikel は (wa) + menyebut no. telp + partikel です (desu).

Contoh :

わたしのでんわ ばんごは ぜろ に なな ご の なな はち はち の きゅ ろく なな です

watashi no denwa bangou wa zero ni nana go no nana hachi hachi no kyuu roku nana desu

Nomor telepon aku adalah 0275-788-967.

---------

NOTE :

Adanya salah satu huruf hiragana の (no) diantara angka-angka nomor telepon tersebut merupakan jenis fungsi untuk mengartikan tanda "–".

Nama-nama angka dalam bahasa Jepang adalah berikut ini (1-10) :

ぜろ (zero) : 0いち (ichi) : 1に (ni) : 2さん (san) : 3よん (yon) : 4ご (go) : 5ろく (roku) : 6なな (nana) : 7はち (hachi) : 8きゅ (kyu) : 9じゅう (juu) : 10

13. Digunakan untuk menujukan benda yang dekat dengan pembicara dare dore sore kore kareDigunakan untuk menujukan benda yang dekat dengan pembicara dare dore sore kore kare


Kata yang digunakan untuk menunjukkan benda yang dekat dengan pembicara adalah Kore yang artinya adalah "Ini".

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Pembahasan :

Kata tunjuk dalam Bahasa Jepang disebut dengan Shiji-shi. Kegunaan nya yaitu untuk menunjukkan sesuatu. Kata tunjuk dalam Bahasa Jepang dibagi menjadi 4 berdasarkan awalan bunyi tiap kata nya. Awalan bunyi kata tunjuk dalam Bahasa Jepang adalah Ko-So-A-Do.

Kata tunjuk berawalan Ko : Digunakan untuk menunjukkan sesuatu yang dekat dengan pembicara. Kata tunjuk dengan awalan bunyi ini umumnya diartikan dengan "Ini", "Sini", dan sebagainya.

Kata tunjuk berawalan So : Digunakan untuk menunjukkan sesuatu yang jauh dengan pembicara, tapi dekat dengan lawan bicara. Kata tunjuk dengan awalan bunyi ini umumnya diartikan dengan "Itu", "Situ", dan sebagainya.

Kata tunjuk berawalan A : Digunakan untuk menunjukkan sesuatu yang jauh dengan pembicara juga lawan bicara. Kata tunjuk dengan awalan bunyi ini kalau diartikan kurang lebih mirip seperti yang berawalan So, yaitu dengan "Itu", "Sana", dan sebagainya.

Kata tunjuk berawalan Do : Digunakan untuk menanyakan sesuatu yang ditunjukkan. Kata tunjuk dengan awalan bunyi ini umumnya diartikan dengan "Mana", "Yang mana", dan sebagainya.

- - - - - - - - -

Kore / Sore / Are / Dore

Keempat kata tersebut merupakan kata tunjuk benda, yang di mana Kore berarti "Ini", Sore dan Are berarti "Itu", dan Dore berarti "Yang mana". Penggunaan kata-kata nya seperti yang sudah saya jelaskan di atas, yaitu dibedakan dari awalan bunyi nya. Perbedaan penggunaannya tidak saya jelaskan lagi masing-masing, namun intinya keempat kata tunjuk tersebut digunakan untuk menunjuk benda.

Dare

Kata ini tidak termasuk ke kata tunjuk, melainkan merupakan kata tanya yang memiliki arti "Siapa".

Kare

Kata ini juga tidak termasuk ke kata tunjuk, melainkan merupakan kata ganti orang ketiga yang memiliki arti "Dia". Yang dimaksud dia di sini hanya untuk laki-laki saja. Jika untuk perempuan menggunakan Kanojo.

18. karendaa wa dore desuka artinya​


カレンダわ どれです か? [ Karendaa wa dore desuka] artinya "Kalender yang mana?"

Pembahasan

Kalender adalah sebuah hal yang selalu kita lihat untuk melihat tanggal besok, lusa, hari ini dan lain sebagainya. Jika diberikan tugas terjadwal, biasanya diberikan batas akhir sesuai dengan kalender. Kalender ini berdasarkan terhadap tahun Masehi yang dimana diukur dari revolusi bumi.

==== ! ====

カレンダわ どれです か? [ Karendaa wa dore desu ka? ]

Artinya : "Kalender yang mana?"

•> カレンダ [Karendaa] artinya kalender.

•> わ [wa] digunakan sebagai penjelas dari suatu kalimat.

•> どれ [Dore] artinya yang mana. Digunakan untuk menanyakan sesuatu yang memiliki rupa yang sama namun memiliki perbedaan yang mencolok. Misalnya pita biru dan merah.

•> です [Desu] digunakan sebagai bentuk kalimat itu positif.

•> か [Ka] digunakan sebagai bentuk kalimat tersebut adalah tanda tanya.

angka = romaji = hiragana = kanji

1= ichi  = いち = 一

2= ni  = に = 二

3= san  = さん = 三

4= shi/ yon  = し、よん = 四

37. Apa bahasa jepangnya Hai! namaku shiha, aku kelas enam a, aku cantik, cerdas, dan imut, terimakasih


N5 Level A || Bab: - & 9

Pembahasan:
• (aisatsu)
• kata sifat.

=== Jawaban ===
↪初めまして! 僕の名前はシハです。僕は六年生です。僕が綺麗や利口や可愛いの子です。宜しく御願いします (Hajimemashite! Boku no namae wa shiha desu. Boku wa roku nen sei desu. Boku ga kirei ya rikou ya kawaii no ko desu. Yoroshiku onegaishimasu!): Perkenalkan! Namaku Shiha. Aku kelas 6. Aku anak yang cantik, pintar, dan imut. Terimakasih!

❇ Partikel:
↪の (no): partikel menyatakan kepemilikan
↪は (wa): partikel menyatakan subjek
↪が (ga): partikel menyatakan penegasan dan kata sifat.
↪や (ya): dan. (Pengganti partikel "to" yang memiliki arti yang sama).

❇ Penjelasan:
↪Karena "Halo" dalam bahasa Jepang ditentukan dengan waktu, maka di ganti dengan "Hajimemashite". Tetapi, jika waktunya di tentukan, "Halo" bisa diganti menjadi:
• Ohayou (selamat pagi)
• Konnichiwa (selamat siang)
• Konbanwa (selamat malam)
("Oyasuminasai" dan "Moshi-moshi" tidak berlaku ✖)
↪ Diakhiran menggunakan "宜しく御願いします (yoroshiku onegashimasu)" karena makna yang terkandung di sana adalah >>> "Mohon bantuannnya (terimakasih)".

❇ Kata sifat:
↪綺麗 (kirei): cantik, indah.
↪利口 (rikou): cerdas / pintar (boleh diganti menjadi Atama ga ii / jouzu / nou ga ii)
↪可愛い (kawaii): lucu, imut.

•| Catatan |•
↪ Untuk nama oran luar Jepang, ditulis menggunakan huruf katakana.
