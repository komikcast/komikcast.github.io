4. Nihon no gakkou wa nangatsu kara nangatsu made desuka.


nihon no gakkou = sekolah jepang
nangatsu = bulan apa (kata untuk menanyakan bulan)
kara = dari
made = sampai
("kara" dan "made" sering digunakan untuk menanyakan atau menyatakan jangka waktu)
jadi kalo di buat kata menjadi "nihon no gakkou wa nangatsu kara nangatsu made" berarti => "sekolah di jepang dari bulan apa sampai bulan apa?"

12. Tri koro dharmo berpusat di


jakarta

smoga membantuGerakan Pemuda Tri Koro Dharmo Golongan pemuda merasa tidak puas pada organisasi Budi Utomo yang condong menjadi perkumpulan kaum tua. Golongan muda menyadari bahwa mereka harus mempunyai perkumpulan sendiri yang dapat mendidik para pemuda memenuhi kewajibannya pada lalu hari. Keinginan ini terwujud dengan terbentuknya Tri Koro Dharmo di Jakarta pada tanggal 7 Maret 1915. Pendiri Tri Koro Dharmo adalah dr. R. Satiman Wiryosanjoyo, Kadarman, dan Sunardi. Anggota Tri Koro Dharma terdiri atas anak sekolah menengah dari Jawa dan Madura. Tri Koro Dharmo adalah organisasi gerakan pemuda pertama. Perkumpulan ini mengelola sebuah majalah yang bernama Tri Koro Dharmo yang terbit pertama kali pada tanggal 10 November 1915. Perkumpulan Tri Koro Dharmo sudah mencapai Jawa Raya dengan cara memperkukuh rasa persatuan antara pemuda Jawa, Sunda, Madura, Bali, dan Lombok. Tri Koro Dharmo dapat diartikan tiga tujuan mulia yang terdiri atas sakti, budhi, dan bakti. Lewat kongresnya yang pertama di Solo pada tahun 1918, nama Trikoro Dharmo diubah menjadi Jong Java. Hal ini dimaksudkan agar para pemuda di luar Pulau Jawa, tata sosialnya berdasar budaya Jawa akan mau, memasuki Jong Java. Kegiatan Jong Java berkisar pada masalah sosial dan kebudayaan, misalnya pemberantasan buta huruf, kepanduan, kesenian. Jong Java tidak ikut terjun dalam dunia politik dan tidak pula mencampuri urusan agama tertentu. Bahkan para anggotanya dilarang menjalankan politik atau menjadi anggota partai politik. Akan tetapi, sejak tahun 1942, sebab pengaruh gerakan radikal, maka Syamsuridjal (ketuanya) mengusulkan agar anggota yang sudah berusia 18 tahun diberi kebebasan berpolitik dan agar Jong Java memasukkan program memajukan agama Islam. Usul ini ditolak, akibatnya para anggotanya yang menghendaki tujuan ke dalam dunia politik dan ingin memajukan agama Islam mendirikan Jong Islamieten Bond. Organisasi ini dipimpin Haji Agus Salim.

21. kodomo no koro kara Hinata ga sukideshita artinya apa?jng asal auto report​


Arti dari 子供の頃からヒナタが好きでした ( kodomo no koro kara Hinata ga sukideshita ) → Saya menyukai Hinata sejak saya masih anak-anak

Pembahasan

Menyatakan kesukaan dalam bahasa Jepang biasanya anda bisa menyebutkan “Aishiteiru , aishiteimasu , Suki , dan Daisuki” kalimat-kalimat tersebut memiliki arti yang berbagai Pula .

子供の頃殻ヒナタが好きでした ( kodomo no koro kara Hinata ga sukideshita ) → Saya menyukai Hinata sejak saya masih anak-anak

Terjemah Per-Kata;

子供 ( Kodomo ) → Memiliki arti "Anak-anak"

の ( No ) → Partikel yang menunjukkan Kepemilikan

頃 ( Koro ) → Memiliki arti "Sekitar"

から ( Kara ) → Memiliki arti "Dari/Sejak"

ヒナタ ( Hinata ) → Nama ditulis menggunakan sistem katakana

が ( Ga ) → Merupakan Partikel

好き ( Suki ) → Memiliki arti suka

でした ( Deshita ) → Merupakan Bentuk lampau dari "Desu" memiliki arti 'Dulu/Bekas'

Nb; kalimat sedikit membingungkan dikarenakan penggunaan 'Deshita' jika deshita yang ditulis dalam kalimat , yang artinya kalimat tersebut memiliki arti dulunya / bekas

__________________

Lalu bagaimana menyatakan kesukaan dalam bahasa Jepang , berikut merupakan beberapa contohnya;

Suka dalam bahasa Jepang adalah ‘Suki’ , Untuk menyatakan Rasa suka ( Pria ) : Suki Dayo ( Wanita ) : Suki Yo

Menyatakan Suka :

1.私はあなたが好きです

( Watashi wa Anata ga Suki Desu )

- Saya suka anda ( Formal )

2.私はあなたが好きだよ

( Watashi wa Anata ga Suki dayo )

- Saya suka Kamu ( Pria )

3. 私はあなた が好きよ

( Watashi wa Anata ga Suki yo )

- Saya suka Kamu ( Wanita )

4. 私はあなたを愛しています

( Watashi wa Anata o Aishiteimasu )

- Saya Cinta kamu

→ Jika kata ''Kamu" ingin diubah maka tunggal diubah pada kata Anata menjadi kata yang akan diubah

Bertanya :

彼は何がすきですか?

( Kare wa Nani ga Suki Desu Ka ? )

- Dia ( L ) suka apa ?

あなたは本が好きですか?

( Anata wa Hon ga Suki Desu Ka ? )

- Apakah Kamu suka buku ?

Menjawab :

• はい、私がすきです!

( Hai, Watashi ga Suki Desu ! )

Ya , Saya Suka !

• はい、私が大好きです!

( Hai , watashi ga Daisuki desu )

Ya , Saya sangat Suka

"愛してる"

Aishiteru Yang Artinya Yaitu Aku Cinta Kamu , Ai 愛 Yang Artinya Yaitu Cinta [ Jarang digunakan oleh anak Remaja , karena memiliki arti yang sangat Mendalam ]

"愛しています"

Aishiteimasu Yang Artinya Yaitu Aku Cinta Kamu , Ai 愛 Yang Artinya Yaitu Cinta

"好き"

Suki Memiliki arti yaitu Suka

Kalimat Yang Menyatakan Kesukaan

Tambah : 好きだよ ( Suki Dayo ) Aku Suka Kamu : Untuk Laki-laki

好きよ ( Suki Yo ) : Aku Suka Kamu : Untuk Perempuan

"大好き"

Dai Suki Yang Memiliki arti yaitu Sangat Suka Dai Yang artinya Besar , Dan Suki Yang Artinya Suka
