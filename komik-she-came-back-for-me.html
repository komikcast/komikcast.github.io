2. - I had waiting for the package when she came. - I hadn't waiting for the package when she came. - Had you waiting for the package when your brother came?Bantu buatin sentence patternnya kaka​


Jawaban:

1.Subject-Verb-Object-When-Clause: "I had been waiting for the package when she came."

2.Subject-Auxiliary-Not-Verb-Object-When-Clause: "I hadn't been waiting for the package when she came."

3.Auxiliary-Subject-Verb-Object-When-Clause: "Had you been waiting for the package when your brother came?"

Jawaban:

1. Subject-Verb-Object-When-Clause :

•I had been waiting for the package when she came.

2. Subject-Had-not-Verb-Object-When-Clause :

•I hadn't been waiting for the package when she came.

3. Had-Subject-Verb-Object-When-Clause-Question :

•Had you been waiting for the package when your brother came?


3. -I had waiting for the package when she came. - I hadn't waiting for the package when she came. - Had you waiting for the package when your brother came?bantu buatin pattern nya bang​


The correct grammar for these sentences would be:

•I was waiting for the package when she came. (Past continuous tense)

•I hadn't been waiting for the package when she came. (Past perfect continuous tense)

•Had you been waiting for the package when your brother came? (Past perfect continuous tense, question form)

The pattern for each sentence would be:

Subject + past continuous tense (was/were + present participle) + object + when + past simple tense (subject + came)

Subject + hadn't + past participle of "be" (been) + present participle + object + when + past simple tense (subject + came)

Had + subject + been + present participle + object + when + past simple tense (subject + came) + question mark

Jawaban:

Kalimat 1:

Subject (I) + past perfect (had been waiting) + object (for the package) + when clause (she came)

Kalimat 2:

Subject (I) + negative past perfect (hadn't been waiting) + object (for the package) + when clause (she came)

Kalimat 3:

Auxiliary (Had) + subject (you) + past participle (been waiting) + object (for the package) + when clause (your brother came)

2. He asked me, "How do you make a great event?"

3. Kalimat tanya yang membutuhkan jawaban yes/no, menggunakan if/whether.

4. Pada kalimat larangan, bentuk reported speech-nya adalat  not + to + verb 1.

28. - he was waiting for the package when she came. - he wasn't waiting for the package when she came. - Were you waiting for the package when your brother came? Tolong bikinin sentence patern nya kaka​


Kalimat 1: Subyek + Past Continuous Tense (was waiting) + Prepositional Phrase (for the package) + Conjunction (when) + Subyek Lain + Simple Past Tense (came).

Contoh: He was waiting for the package at the front porch when she came.

Kalimat 2: Subyek + Past Continuous Tense (wasn't waiting) + Prepositional Phrase (for the package) + Conjunction (when) + Subyek Lain + Simple Past Tense (came).

Contoh: He wasn't waiting for the package at the post office when she came.

Kalimat 3: Question Word (Were) + Subyek + Past Continuous Tense (waiting) + Prepositional Phrase (for the package) + Conjunction (when) + Possessive Pronoun (your) + Noun (brother) + Simple Past Tense (came)?

Contoh: Were you waiting for the package at home when your brother came?
