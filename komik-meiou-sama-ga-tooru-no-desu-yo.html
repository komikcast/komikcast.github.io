Terjemahan :

Halo. Nama saya Norio Tanaka. Kelas 1 SMA. Saya suka olahraga. Paling suka sepak bola. Itu menarik. Tolong e-mail saya dalam Bahasa Jepang. Senang berkenalan denganmu.

Pembahasan

Perkenalan diri dalam Bahasa Jepang disebut dengan 自己紹介 (Jikoshoukai). Dalam perkenalan diri, biasa terdapat pembuka dan penutup. Di antara pembuka dan penutup ini, kalian bisa mengenalkan nama, kelas, hal yang disuka, dll.

1). Pembuka

Pembuka Jikoshoukai biasa menggunakan beberapa ungkapan di bawah ini.

Hajimemashlte (Perkenalkan)Doumo (Halo)Konnichiwa (Halo)...

2). Nama, kelas, hal yang disuka, dll

Untuk memperkenalkan nama, dapat menggunakan pola kalimat berikut.

=> Watashi no namae wa [...] desu.

=> Nama saya [...].

Untuk memberitahu kelas berapa, dapat menggunakan pola kalimat berikut.

=> [Tingkatan] [1/2/3] nensei desu.

=> Kelas [1/2/3] [Tingkatan].

Untuk memberitahu hal yang disuka, dapat menggunakan pola kalimat berikut.

=> Watashi wa [...] ga suki desu.

=> Saya suka [...].

3). Penutup

Penutup Jikoshoukai biasa menggunakan beberapa ungkapan di bawah ini.

YoroshikuYoroshiku onegaishimasuDouzo yoroshiku...

Artinya kurang lebih sama, yaitu "senang berkenalan denganmu" atau "mohon bantuannya".

*Kata "Saya" di sini tergantung dari siapa subjek yang dibahasnya. Jika ini diucapkan oleh si pembicara dan menujukan dirinya sendiri, maka dapat menggunakan "Saya". Tapi jika ditujukan ke orang lain, maka menjadi "Dia". Tapi agar aman, lebih baik tidak usah menyebutkan subjeknya, karena memang di kalimat aslinya juga tidak menyebutkan subjek.

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

PEMBAHASAN

POLA KALIMAT

=> [Subjek] wa [KB] ga suki desu

=> [Subjek] wa [KK] koto/no ga suki desu

Kedua pola kalimat di atas digunakan untuk menyatakan kesukaan terhadap suatu benda atau hal. Bila pola kalimat tersebut diartikan, maka artinya seperti "[Subjek] suka [KB/KK]". Perlu diingat bahwa jika yang disuka adalah sebuah kegiatan (kata kerja), maka perlu menambahkan 'Koto' atau 'No' setelah kata kerja nya. Fungsi dari 'Koto' dan 'No' ini yaitu untuk mengubah si kata kerja menjadi kata benda.

CONTOH KALIMAT

1). Watashi wa nihongo ga suki desu.

    = Saya suka Bahasa Jepang.

2). Kare wa anime o miru no ga suki desu.

    = Dia (lk) suka menonton anime.

3). Naruse wa utau koto ga suki desu.

    = Narse suka menyanyi.

Sebagai tambahan saja, jika subjek sudah diketahui oleh lawan bicara, maka tidak perlu menyebutkan '[Subjek] wa' lagi. Langsung saja dengan '[KB] ga suki desu' atau '[KK] koto/no ga suki desu'.
