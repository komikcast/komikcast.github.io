25. If i (know)...............that you were ill,i(go)...........to see you


~ Answer:knew; would go

.~ If I knew that you were ill, I would go to see you. ~.

~ Introduction:

Conditional sentence artinya kalimat kondisional merupakan suatu pengibaratan terhadap suatu pekerjaan/keberadaan yang ditentang atau tidak sesuai oleh faktual, hanya diharapkan dapat terjadi menurut ekspektasi. Pada kalimat kondisional terdiri atas tiga tipe yaitu:

Conditional type 1—pengibaratan pada masa kini (simple present)Conditional type 2—pengibaratan pada masa lalu (simple past)Conditional type 3—pengibaratan pada peristiwa yang benar-benar sudah terjadi pada masa lalu (past perfect)

Serta ada conditional type 0 dimana ini menunjukkan suatu pengibaratan pada peristiwa faktual.

~ Discussion:

Berikut kunci untuk membuktikan tipe kalimat kondisional:

Type 1

if + simple present + simple future

Type 2

if + simple past + past future

Type 3

if + past perfect + past future perfect

Pernyataan if I ... that you were ill menunjukkan bentuk dari simple past maka if + simple past merupakan bentuk dari type 2 sehingga:

know menjadi simple past verb yaitu knewgo dengan dilengkapi would menjadi would go dinyatakan ke dalam bentuk past future

—> jawabannya: knew; would go
