3. Apa bedanya "ni hen bang" sama "ni hen pang"?


Jawaban:

ni hen bang dan ni hen pang memiliki dua arti yang berbeda. arti dari ni hen bang adalah 'kamu sangat hebat'. sedangkan, arti dari ni hen pang adalah 'kamu sangat gendut'.

- 很 = sangat

- 棒 = hebat

arti： kamu sangat hebat

你很胖 （Ni Hen Pang）

- 你 = kamu

- 很 = sangat

- 胖  = gendut

arti： kamu sangat gendut

18. Denwa bangou ha nanban desuka. “Rei hachi ichi san no ni rei go roku no ichi nana ni go desu “. Penulisan nomor telpon dalam bahasa indonesia yang benar adalah…. 0813-2056-1725       0831-2506-7152        0813-5206-1725       0831-2056-1725       0813-5206-7152 ​


Penulisan nomor telepon dalam bahasa Indonesia yang benar adalah 0813-2056-1725 (A).

Pembahasan

Denwa bangou wa nanban desuka?

"Rei hachi ichi san no ni rei go roku no ichi nana ni go desu."

Artinya dalam bahasa Indonesia :

Nomor teleponnya berapa?

"Nol delapan satu tiga dua nol lima enam satu tujuh dua lima."

______

Berikut jika angka nomor telepon 0813-2056-1725 ditulis dengan versi hiragana adalah : れい はち さん の に れい ご ろく の いち なな に ご

Denwa bangou「電話番号」

Adalah nomor telepon.

Jika dalam bahasa Jepang ketika ingin menanyakan nomor telepon kepada orang lain maka menggunakan kalimat :

Anata no denwa bangou wa nanban desuka? (Nomor telepon mu berapa?).

Bisa juga dengan :

Denwa bangou wa nanban desuka? (Nomor teleponnya berapa?).

Kemudian, ketika ingin memberikan nomor telepon diri sendiri kepada seseorang maka harus memahami hal berikut ini :

- Mengetahui angka-angka dasar dalam bahasa Jepang.

Berikut nama-nama angka 1-10 dalam bahasa Jepang adalah :

零・れい・ゼロ (rei/zero) : nol. 一・いち (ichi) : satu二・に (ni) : dua.三・さん (san) : tiga.四・よん (yon) : empat.五・ご (go) : lima.六・ろく (roku) : enam.七・なな (nana) : tujuh.八・はち (hachi) : delapan.九・きゅう (kyuu) : sembilan.十・じゅう (juu) : sepuluh.

- Menggunakan huruf の (no) di antara-antara nomor dalam telepon.

Penggunaan huruf tersebut adalah memiliki fungsi sebagai tanda strip (-).

- Diakhiri dengan kata です (desu), tujuannya untuk menghaluskan.

Denwa bangou wa nanban desuka. “Rei hachi ichi san no ni rei go roku no ichi nana ni go desu“. Penulisan nomor telepon dalam Bahasa Indonesia yang benar adalah 0813-2056-1725 (A).

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Pembahasan :

Cara penyebutan nomor telepon dalam Bahasa Jepang itu cukup mudah. Kalian hanya perlu menghafalkan angka 0 sampai 9 saja. Penyebutan angka yang digunakan pada penyebutan nomor telepon dalam Bahasa Jepang adalah sebagai berikut.

•  0 = Rei / Zero       •  5 = Go

•  1 = Ichi                  •  6 = Roku

•  2 = Ni                   •  7 = Nana

•  3 = San                 •  8 = Hachi

•  4 = Yon                •  9 = Kyuu

Dalam nomor telepon itu biasa terdapat tanda strip ( - ) yang memisahkan antara beberapa angka. Tanda ini dibaca No. Setelah selesai menyebutkan nomor teleponnya, ditambahkan kata Desu untuk memperhalusnya.

- - - - - - -

Contoh :

0812-3456-7890

Rei hachi ichi ni no san yon go roku no nana hachi kyuu rei desu

0812 = Rei hachi ichi ni3456 = San yon go roku7890 = Nana hachi kyuu rei

0198-7654-3210

Rei ichi kyuu hachi no nana roku go yon no san ni ichi rei desu

0198 = Rei ichi kyuu hachi7654 = Nana roku go yon3210 = San ni ichi rei

0813-2056-1725

Rei hachi ichi san no ni rei go roku no ichi nana ni go desu

0813 = Rei hachi ichi san2056 = Ni rei go roku1725 = Ichi nana ni go

- - - - - - -

Untuk menanyakan nomor telepon seseorang, kita bisa mengatakan 'Anata no denwa bangou wa nanban desuka' yang artinya "Berapa nomor telepon anda?". Sedangkan untuk menjawabnya, bisa dengan dengan mengatakan 'Watashi no denwa bangou wa (...) desu' yang artinya "Nomor telepon saya (...)".

31. Simpangan naku dari data : 1,2,3,4,5


Banyaknya data ( n ) = 5
Jumlah seluruh data 1 + 2 + 3 + 4 + 5 = 15
Kemudian dibagi banyaknya data (n) untuk  mencari rata-rata 15 : 5 = 3

Kemudian mencari selisih antara rata-rata dengan setiap data (1 - 3) + (2 - 3) + (3 - 3) + (4 - 3) + (5 - 3)
Hasilnya dipangkat 2 lalu di jumlah (- 2 )2 + (- 1 )2 + ( 0 )2 + ( 1 )2 + ( 2 )2  = 4 + 1 + 0 + 1 + 4 = 10
Lalu dibagi banyaknya data (n) 10 : 5 = 2
Lalu beri tanda akar √2 = 1,4
Jadi simpangan bakunya adalah  √2 atau 1,4
