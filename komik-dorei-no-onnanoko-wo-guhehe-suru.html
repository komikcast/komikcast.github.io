6. ななさんを大切にしる ( nana san wo taisetsu ni suru ) artinya..​


ななさんを大切にする

Nana-san wo taisetsu ni suru

Menjaga Nana

Penjelasan :

Taisetsu berarti penting atau berharga, dan pola Ni suru digunakan untuk menjadikan sesuatu menjadi si kata benda yang dituju. Pola nya adalah 'KB ni suru', yang kurang lebih artinya menjadikan KB. Pada kata 'Taisetsu ni suru' ini berarti menjadikan penting, menjadikan berharga, mementingkan, atau menghargai. Tapi dalam konteks kalimat ini dapat diartikan dengan menjaga. Jadi maksud dari kalimat 'Nana-san wo taisetsu ni suru' yaitu seperti menjadikan penting si Nana.

11. karendaa wa dore desuka artinya​


カレンダわ どれです か? [ Karendaa wa dore desuka] artinya "Kalender yang mana?"

Pembahasan

Kalender adalah sebuah hal yang selalu kita lihat untuk melihat tanggal besok, lusa, hari ini dan lain sebagainya. Jika diberikan tugas terjadwal, biasanya diberikan batas akhir sesuai dengan kalender. Kalender ini berdasarkan terhadap tahun Masehi yang dimana diukur dari revolusi bumi.

==== ! ====

カレンダわ どれです か? [ Karendaa wa dore desu ka? ]

Artinya : "Kalender yang mana?"

•> カレンダ [Karendaa] artinya kalender.

•> わ [wa] digunakan sebagai penjelas dari suatu kalimat.

•> どれ [Dore] artinya yang mana. Digunakan untuk menanyakan sesuatu yang memiliki rupa yang sama namun memiliki perbedaan yang mencolok. Misalnya pita biru dan merah.

•> です [Desu] digunakan sebagai bentuk kalimat itu positif.

•> か [Ka] digunakan sebagai bentuk kalimat tersebut adalah tanda tanya.

20. 1. Desu/ima/amari/wa/atsukunai/Jakaruta. 2. Nihon/dewa arimasen/amari/wa/shizuka. 3. Kakimasu/repooto/wo/de/nihongo. 4. Jesica san/wa/o/agemasu/ni/tokei/watashi. 5. Dea san/dore/wa/kaban/no/ka/desu.


Susunan kalimat-kalimat tersebut yang benar adalah sebagai berikut.

Jakaruta wa ima amari atsukunai desu. (Jakarta sekarang tidak terlalu panas)Nihon wa amari shizuka dewa arimasen. (Jepang tidak terlalu sepi)Nihongo de repooto wo kakimasu. (Menulis laporan dengan Bahasa Jepang)Watashi wa Jesica-san ni tokei o agemasu. (Saya memberi Jesica jam)Dea-san no kaban wa dore desu ka. (Tas milik Dea yang mana?)

Pembahasan

Kata keterangan amari dalam Bahasa Indonesia memiliki arti "tidak terlalu", "tidak begitu", "jarang", dsb. Amari ini diikuti dengan kata sifat atau kata kerja, tapi dalam bentuk negatif. Pola nya adalah sebagai berikut.

=> Amari + [KK/KS negatif]

=> Tidak terlalu [KK/KS]

Contoh kalimat :

Amari oishikunai = Tidak terlalu enakAmari oyoganai = Tidak banyak berenangAmari tokui janai = Tidak begitu pandai

- - - - -

Kata kerja agemasu merupakan bentuk sopan (masu) dari ageru. Artinya dalam Bahasa Indonesia adalah "memberi". Kata ini digunakan saat si pembicara yang memberikan sesuatu atau berbuat baik ke orang lain.

Jika si pembicara yang diberikan sesuatu, maka kata yang digunakan adalah kureru. Ageru dan kureru artinya sama, tapi penggunaannya saja yang berbeda.

Kita akan bahas bersama sama tentang grammarnya.

Waga na (我がな) adalah frasa non formal yang berarti aku sendiri. Nah, Megumin harus ditulis dengan katakana menjadi メグミン. Partikel は digunakan sebagai penanda subjeknya yaitu Megumin.

Nariwai (kanji : 生業) merupakan frasa yang berarti mata pencaharian lalu toshi (とし) kata singkat dari tosh|te (として) yang artinya sebagai. Jadi, artinya menjadi ber mata pencaharian/bekerja sebagai... . Namun, karena ini didunia fiktif anime maka artinya menjadi "seorang" (sudah melambangkan pekerjaan). Arch Wizard juga merupakan kata serapan sehingga ditulis katakana menjadi アークウィザード .

Saikyou ( さいきょうい) merupakan frasa yang artinya pemulihan, tertinggi, terkuat. Partikel の (no) sebagai partikel kepemilikan. Kougeki (kanji : 攻撃) berarti penyerangan, serangan. Mahou (kanji : 魔法) berarti sihir. Jadi, arti saikyou no kougeki no mahou adalah mempunyai serangan sihir terkuat.

Bakuretsu Mahou (kanji : 爆烈魔法) berarti sihir ledakan dimana bakutetsu = ledakan dan mahou = sihir. Ayatsuru (操る) adalah kata jamak dari 操 (ayatsu) yang artinya manipulasi sehingga artinya menjadi memanipulasi. Mo (も) adalah partikel yang bersifat mempertegas. Jadi, artinya adalah "sihir ledakan" yang memanipulasi!

Kesimpulan

Waga na wa Megumin + Akuwizado wo nariwai toshi + saikyou no kougeki no mahou + bakuretsu mahou + ayatsuru

= Aku sendiri Megumin. Seorang Arch Wizard yang mempunyai serangan sihir terkuat yaitu sihir ledakan yang dapat memanipulasi

35. Digunakan untuk menujukan benda yang dekat dengan pembicara dare dore sore kore kareDigunakan untuk menujukan benda yang dekat dengan pembicara dare dore sore kore kare


Kata yang digunakan untuk menunjukkan benda yang dekat dengan pembicara adalah Kore yang artinya adalah "Ini".

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Pembahasan :

Kata tunjuk dalam Bahasa Jepang disebut dengan Shiji-shi. Kegunaan nya yaitu untuk menunjukkan sesuatu. Kata tunjuk dalam Bahasa Jepang dibagi menjadi 4 berdasarkan awalan bunyi tiap kata nya. Awalan bunyi kata tunjuk dalam Bahasa Jepang adalah Ko-So-A-Do.

Kata tunjuk berawalan Ko : Digunakan untuk menunjukkan sesuatu yang dekat dengan pembicara. Kata tunjuk dengan awalan bunyi ini umumnya diartikan dengan "Ini", "Sini", dan sebagainya.

Kata tunjuk berawalan So : Digunakan untuk menunjukkan sesuatu yang jauh dengan pembicara, tapi dekat dengan lawan bicara. Kata tunjuk dengan awalan bunyi ini umumnya diartikan dengan "Itu", "Situ", dan sebagainya.

Kata tunjuk berawalan A : Digunakan untuk menunjukkan sesuatu yang jauh dengan pembicara juga lawan bicara. Kata tunjuk dengan awalan bunyi ini kalau diartikan kurang lebih mirip seperti yang berawalan So, yaitu dengan "Itu", "Sana", dan sebagainya.

Kata tunjuk berawalan Do : Digunakan untuk menanyakan sesuatu yang ditunjukkan. Kata tunjuk dengan awalan bunyi ini umumnya diartikan dengan "Mana", "Yang mana", dan sebagainya.

- - - - - - - - -

Kore / Sore / Are / Dore

Keempat kata tersebut merupakan kata tunjuk benda, yang di mana Kore berarti "Ini", Sore dan Are berarti "Itu", dan Dore berarti "Yang mana". Penggunaan kata-kata nya seperti yang sudah saya jelaskan di atas, yaitu dibedakan dari awalan bunyi nya. Perbedaan penggunaannya tidak saya jelaskan lagi masing-masing, namun intinya keempat kata tunjuk tersebut digunakan untuk menunjuk benda.

Dare

Kata ini tidak termasuk ke kata tunjuk, melainkan merupakan kata tanya yang memiliki arti "Siapa".

Kare

Kata ini juga tidak termasuk ke kata tunjuk, melainkan merupakan kata ganti orang ketiga yang memiliki arti "Dia". Yang dimaksud dia di sini hanya untuk laki-laki saja. Jika untuk perempuan menggunakan Kanojo.
