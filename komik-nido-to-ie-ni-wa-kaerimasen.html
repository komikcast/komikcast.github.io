15. bedanya 'ie' sama 'iie' itu apa? :(


家 / いえ (ie) : rumah.いいえ (iie) : gak, tidak, bukan.

Pembahasan

–› ie「 家 / いえ 」

Kata 家 –› ditulis dengan versi kanji, sedangkan いえ –› ditulis dengan versi hiragana.

Artinya yaitu rumah (tempat tinggal), penggunaan kata 家 (ie) ini termasuk umum sehingga bisa digunakan untuk menunjukkan kepemilikan rumah diri sendiri atau juga rumah orang lain.

Bentuk kata 家 / いえ (ie) ini termasuk jenis kata benda (meishi).

Contoh penggunaanya adalah sebagai berikut :

私の家 (watashi no ie) : rumahku.友達の家です (tomodachi no ie desu) : rumah temanku.私は彼女の家が好きです (watashi wa kanojo no ie ga suki desu) : aku suka rumah dia (pr).

–› iie「 いいえ 」

Ditulis dengan bentuk hiragana. Artinya dalam bahasa Indonesia berarti tidak, gak, bukan. Jenis pernyataan yang bersifat negatif.

Kata いいえ (iie) umum dipakai ketika menjawab pertanyaan yang dimana pertanyaan tersebut bisa dijawab iya/tidak.

Contoh penggunaan kata いいえ (iie) :

= アニメは好きですか? (anime wa suki desuka?) : apa kamu suka anime?

いいえ、アニメは好きではありません (iie, anime suki de wa arimasen) : gak, aku gak suka anime.

= わかりますか? (wakarimasu ka?) : kamu ngerti tidak?

いいえ、わかりません (iie, wakarimasen) : tidak, tidak ngerti.

Jawaban :

Bedanya Ie dengan Iie

いえ (Ie) = Versi pendek dari 'Iie', namun bisa juga memiliki arti lain, tergantung dari Kanji nya.いいえ (Iie) = Memiiki arti "Tidak", "Nggak", "Bukan", dsb.

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Pembahasan :

Kosakata Bahasa Jepang ada banyak yang sangat mirip. Hanya berbeda satu huruf, berbeda panjang pendeknya, dengan hal itu saja sudah dapat membuat arti yang berbeda.

Apa itu いえ (Ie)?

Tergantung konteksnya, kata ini bisa memiliki banyak arti. Jika penulisannya dengan Kanji 家 (Ie), maka berarti "Rumah". Jika penulisannya dengan Kanji 言え (Ie), maka berarti "Katakan!". Dan jika hanya ditulis dengan Hiragana いえ (Ie), artinya bisa bermacam. Kata 'Ie' ini juga bisa menjadi versi pendek dari 'Iie'. Pelafalan kata Ie ini seperti ie atau iye biasa saja.

Apa itu いいえ (Iie)?

Kata ini merupakan suatu ungkapan penolakan atau ketidaksetujuan terhadap suatu pendapat. Kalau diartikan sama seperti "Tidak", "Nggak", "Bukan", dan sebagainya. Pelafalan kata ini seperti I∼ye', dengan i yang agak panjang dan e terakhir agak ditekan. Contoh penggunaan kata nya adalah sebagai berikut.

入間 : あなたはクララですか？Iruma : Anata wa kurara desuka?Iruma : Apakah kamu Clara?アズ : いいえ、僕はアズです。Azu : Iie, boku wa azu desu.Azu : Bukan, aku Azu.

33. makna lagu ie ie yg berasal dari ende​


Jawaban:

Manusia mempunyai profil pribadi (individualitas) yang unik dan berbeda antara satu dengan yang lain. Hal ini juga berlaku bagi kelompok-kelompok manusia, suku-suku dan sebagainya. Itulah sebabnya mengapa kebudayaan yang diciptakan oleh manusia beranekaragam dan salah satunya adalah lagu daerah Ende “Ie” yang tetap bertahan sampai saat ini. Hal ini menjadi alasan bagi peneliti untuk melakukan penelitian dengan judul Makna Kekeluargaan Dalam Syair Lagu Daerah Ende “Ie” Menggunakan Semiotika Ferdinand De Saussure. Terdapat lima konsep yang penulis pakai dalam penelitian ini antara lain makna, semiotika, semiotika Ferdinand De Saussure, nyanyian rakyat dan keluarga. Sebelum menganalisis, penulis membagi lirik lagu ke dalam bentuk paradigmatik dan bentuk sintagmatik. Setelah itu penulis menggunakan model semiotika milik Saussure untuk menganalisis makna. Menurut Saussure tanda terdiri dari unsur pananda dan petanda. Penanda adalah “bunyi yang bermakna” atau “coretan yang bermakna”. Jadi penanda adalah aspek material dari bahasa : apa yang dikatakan atau didengar dan apa yang ditulis atau dibaca. Sedangkan Petanda adalah gambaran mental, pikiran, atau konsep. Jadi petanda adalah aspek mental dari bahasa. Lagu daerah Ende “Ie” tersusun atas pengantar (bait pertama), sampiran dan juga isi. Lagu ini memiliki sampiran dan isi karena lagu ini berisi tentang pantun. Lirik lagu dalam bait pengantar adalah Ie..ie..ie.. Bele wea seru molo mesa a..a..a..a. Lirik yang merupakan sampiran adalah Seru kai nonge nebu ola kobe one, Ie.. nonge ola baje wole a..a..a..a. Sedangkan lirik yang menjadi isi adalah Ana susa apa ema ngala laka role, Molo doa ana eda doa a..a..a..a. Lagu “Ie” menceritakan tentang wanita cantik dengan pakaiannya yang indah dan suaranya yang merdu menggoda para pria yang mendengarkan suaranya lewat lagu yang dia nyanyikan.
