3. apa arti dari san ichi no zero ichi no ni zero zero desu?


Arti dari angka san ichi no zero ichi no ni zero zero desu adalah 31-01-200.

Pembahasan

Angka Jepangnya : san ichi no zero ichi no ni zero zero desu.

Bentuk angkanya : 31-01-200

Cara baca dalam bahasa Indonesia : tiga satu nol satu dua nol nol.

_______________

san ichi no zero ichi no ni zero zero desu :

Masing-masing angka dalam bentuk kanjinya.

三 (san) = tiga = 3.一 (ichi) = satu = 1.零 (zero) = nol = 0.二 (ni) = dua = 2.

Adanya kata no yang terletak diantara tengah-tengah angka sebenarnya itu melambangkan tanda strip "—" dan tidak memiliki arti. Suka digunakan ketika kita menyebutkan suatu nomor, seperti nomor telepon, dll.

Adanya partikel です (desu) yang berada pada akhir setelah menyebutkan angka, karena memang fungsi partikel ini sebagai penekanan dan bentuk akhiran dalam kalimat.

_______________

Nama-nama angka dari 1 hingga 10 dalam bahasa Jepang, beserta dengan penulisan kanjinya sebagai berikut :

零 (rei/ree/zero) = nol = 1.一 (ichi) = satu = 1.二 (ni) = dua = 2.三 (san) = 3.四 (yon/shi) = empat = 4.五 (go) = lima = 5.六 (roku) = enam = 6.七 (nana/shichi) = tujuh = 7.八 (hachi) = delapan = 8.九 (kyuu/ku) = sembilan = 9.十 (juu) = sepuluh = 10.

PEMBAHASAN

Pada teks tersebut terdapat banyak kalimat yang menyatakan keberadaan sesuatu dan kebisaan melakukan sesuatu. Kuncinya terdapat pada kata 'Arimasu' yang berarti "Ada", dan 'Dekimasu' yang berarti "Bisa".

- - - -

POLA KALIMAT

1). Menyatakan keberadaan

=> [KB] ga arimasu.

    Ada [KB].

=> [Letak] ni arimasu.

    Ada di [Letak].

=> [Letak] ni [KB] ga arimasu.

    Di [Letak] ada [KB].

=> [KB] wa [Letak] ni arimasu.

    [KB] ada di [Letak].

2). Menyatakan kebisaan

=> [KK] koto/no ga dekimasu.

    Bisa [KK].

- - - -

CONTOH KALIMAT

1). Menyatakan keberadaan

- Puranbanan ga arimasu.

    Ada Prambanan.

- Jogujakaruta ni arimasu.

    Ada di Yogyakarta.

- Jogujakaruta ni puranbanan ga arimasu.

    Di Yogyakarta ada Prambanan.

- Puranbanan wa jogujakaruta ni arimasu.

    Prambanan ada di Yogyakarta.

2). Menyatakan kebisaan

- Shashin o toru koto ga dekimasu.

    Bisa mengambil foto.

- Keeki o tsukuru no ga dekimasu.

    Bisa membuat kue.

Romaji :

Artinya :

__________

Soal diatas, menanyakan mengenai arti 'menonton film' dalam bahasa Jepang, jawaban yang tepat adalah opsi D. Eiga o miru. Kemudian mengenai arti opsi lainnya di opsi A, B, dan C adalah :

A. Ryouri o miru「料理を見る」: melihat masakan.B. Sakka o suru「作家をする」: menjadi penulis.C. Hon o yomu「本を読む」: membaca buku.

__________

Watashi no shumi wa eiga o miru koto desu

「私の趣味は映画を見ることです」

Setiap terjemahan kosakata dengan penjelasannya juga sebagai berikut.

私・わたし (watashi) = saya, aku. Bentuk umum dan juga tergolong formal.

の (no) = *partikel yang dapat memberitahukan mengenai kepemilikan.

趣味・しゅみ (shumi) : hobi (kegiatan yang disukai).

は (wa) = *partikel yang dijadikan penanda subjek.

映画・えいが (eiga) = film.

を (o/wo) = *partikel yang penggunaannya sebagai penanda objek.

見る・みる (miru) = secara harfiah maknanya yaitu melihat, namun jika digabungkan dengan kata benda seperti 映画 (eiga) maka dapat diartikan menonton atau menyaksikan.

事・こと (koto) = tergantung konteks kalimatnya, kata ini bisa diartikan tentang, hal. Dapat dijadikan sebagai pola kalimat menyatakan hobi dalam bhs Jepang.

です (desu) = *bentuk kata bantu untuk menyatakan kesopanan/kehalusan. Bisa untuk penekanan.

35. Watashi no denwa bangou wa Zero go ni Ichi no san hachi nana kyuu no go ni roku go desu​


Jawaban:

Denwa bangou ( でんわ ばんご )

• 0 Zero ( ぜろ )

• 1 ichi ( いち )

• 2 ni ( に )

• 3 san ( さん )

• 4 yon ( よん )

• 5 go ( ご )

• 6 roku ( ろく )

• 7 nana ( なな )

• 8 hachi ( はち )

• 9 kyu ( きゅ )

• 10 juu ( じゅう )

• 11 juu ichi ( じゅう いち にち )

• 15 juu go ( じゅう ご )

• 20 ni juu ( にじゅう )

• 25 ni juu go ( に じゅう ご )

• 30 san juu ( さん じゅう )

Dan seterusnya

Pola kalimat (+) subjek (no) nomer telepon (wa) nomer telepon (desu).

Keterangan :

• WATASHI adalah subjek yang artinya saya

• Partikel (No) adalah partikel bentuk kepunyaan atau kepemilikan

• DENWA BANGO WA adalah nomer telepon

• Tanda (-) pada nomer di baca “NO” dalam bahasa jepang.

• DESU adalah partikel kalimat positif (+) yang ada di akhir kalimat dalam bahasa jepang

Jadi, kalimat "watashi no denwa bangou wa Zero go ni Ichi no san hachi nana kyuu no go ni roku go desu" atau 「わたしのでんわばんごわぜろごにいちのさんはちななきゅのごにろくごです」 kalau diterjemahkan ke dalam bahasa Indonesia adalah “nomor telepon saya adalah 0521-3879-5265”

Penjelasan:

angka = romaji = hiragana = kanji

1= ichi  = いち = 一

2= ni  = に = 二

3= san  = さん = 三

4= shi/ yon  = し、よん = 四  

5= go  =ご =五

6=roku  = ろく = 六

7=shichi / nana  = しち、なな = 七

8= hachi  = はち = 八

9=kyuu / ku  =  く、きゅう = 九        

10=juu = じゅう = 十

jika membaca nomor telepon,maka membaca angkanya harus satu-satu. berbeda dengan cara membaca bilangan biasa

contoh membaca nomor telepon:

0361-464452 = Zero san roku ichi no yon roku yon yon go ni desu.

Minta tolong di terjemahin tolongg bangett:(jogujakaruta wa jawa ni arimasu. kankouchi ga takusan arimasu. puranbanan ya jogujakaruta no oukyuu nado desu. puranbanan ni furui otera ga takusan arimasu. jawa no dansu o miru koto ga dekimasu. gamuran no enzou o kiku koto ga dekimasu. oukyuu ni hakubutsukan ga arimasu. ramayana no dansu o miru koto ga dekimasu. totemo kirei desu.


Terjemahan :

Yogyakarta ada di Jawa. Tempat wisata nya ada banyak. Prambanan, Keraton Yogyakarta, dan lain-lain. Di Prambanan ada banyak candi tua. Bisa melihat tari Jawa. Bisa mendengar pertunjukan gamelan. Di Keraton ada museum. Bisa melihat tari Ramayana. Sangat indah.

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

PEMBAHASAN

Pada teks tersebut terdapat banyak kalimat yang menyatakan keberadaan sesuatu dan kebisaan melakukan sesuatu. Kuncinya terdapat pada kata 'Arimasu' yang berarti "Ada", dan 'Dekimasu' yang berarti "Bisa".

- - - -

POLA KALIMAT

1). Menyatakan keberadaan

=> [KB] ga arimasu.

   Ada [KB].

=> [Letak] ni arimasu.

   Ada di [Letak].

=> [Letak] ni [KB] ga arimasu.

   Di [Letak] ada [KB].

=> [KB] wa [Letak] ni arimasu.

   [KB] ada di [Letak].

2). Menyatakan kebisaan

=> [KK] koto/no ga dekimasu.

   Bisa [KK].

- - - -

CONTOH KALIMAT

1). Menyatakan keberadaan

- Puranbanan ga arimasu.

   Ada Prambanan.

- Jogujakaruta ni arimasu.

   Ada di Yogyakarta.

- Jogujakaruta ni puranbanan ga arimasu.

   Di Yogyakarta ada Prambanan.

- Puranbanan wa jogujakaruta ni arimasu.

   Prambanan ada di Yogyakarta.

2). Menyatakan kebisaan

- Shashin o toru koto ga dekimasu.

   Bisa mengambil foto.

- Keeki o tsukuru no ga dekimasu.

   Bisa membuat kue.
