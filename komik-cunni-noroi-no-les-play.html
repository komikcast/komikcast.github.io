      Luas persegi panjang = p x l = 25cm x 18cm = 450cm²
      Luas trapesium           = 1/2 (a+b) t
                                          = 1/2 (25cm+35cm) (30cm-18cm)
                                          = 1/2 (60cm) x 12cm
                                          = 1/2 (720)
                                          = 360cm²
      Luas keseluruhan        = luas persegi panjang + luas trapesium
                                          = 450cm² + 360cm² = 810cm²


27. D. 312cm²
      Luas kertas pertama = s x s = 29cm x 29cm = 841cm²
      Luas kertas kedua     = s x s = 23cm x 23cm = 529cm²
      Selisih luas kertas     = Luas kertas kesatu - luas kertas kedua
                                        = 841cm² - 529cm² = 312cm²
