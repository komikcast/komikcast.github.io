1. christy has several $5 coins and $2 coins. it is given the total value of the coins is $38. if she uses two $5 coins to buy a pen, then the numbers of $5 coins and $2 coins will be the same. find the original number of coins


Jawaban:

The original number of $5 coins is 6.The original number of $2 coins is 4.

Penjelasan dengan langkah-langkah:

Number of $5 coins = x

Number of $2 coins = y

Total value is $38:

5x + 2y = 38 ... (1)

If two $5 coins used, then the number of $5 and $2 coins will be the same:

y = x - 2 ... (2)

Substitute the value of y to equation (1):

5x + 2y = 38

5x + 2(x-2) = 38

5x + 2x - 4 = 38

7x - 4 = 38

7x = 38 + 4

7x = 42

x = 42/7

x = 6

y = x-2

y = 6-2

y = 4

Therefore:

The original number of $5 coins is 6.The original number of $2 coins is 4.

Discovered:

Variable x = number of $2 coins

Variable y = number of $2 coins

Linear equation of two variables (1) = ... (1)

Asked:

 Original number of coins = ?

Answered:

 If two $5 coins used, then the number of $5 and $2 coins will be the same:

  y = x - 2 ... (2)

1. Substitute the value of y to equation:

 1.1. Search value of variable x

  [tex]5x + 2y = 38 \\5x + 2(x-2) = 38\\5x + 2x - 4 = 38\\7x - 4 = 38\\7x = 38 + 4\\7x = 42\\x = 42 \div 7x = 6[/tex]

 1.2. Search value of variable y

  [tex]y = x - 2\\y = 6 - 2\\y = 4\\\\ or\\\\5x + 2y = 38\\(5 \times 6) + 2y = 38\\30 + 2y = 38\\2y = 38 - 30\\y = 4[/tex]

Conclusion:

 So, original number of coins are $6 and $4

6. A bag contained some 20-cent and 50-cent coins. There were 14 fewer 50-cent coins than 20-cent coins. The total amount of money in the bag was $27.30. How many 50-cent coins were there?


The bag contained "several" 20-cent coins and "several" 50-cent coins. A 50-cent coin is 14 times less than a 20-cent coin. The total amount of cents is $27.30. Then the amount of 50 cents coins is 35 coins.

Discussion

First guess: 1 x 50 cent coin + (1+14) 20 cent coin

Value: (1 x $0.5) + (1+14 x $0.2) = $0.5 + $3.00 = $3.50

So, we still need $27.30 - $3.50 = $23.80

For every 50 cent coin added, a 20 cent coin will also be added so that the ratio of 20 cents and 50 cents remains 14.

Each additional coin will increase the number of cents by $0.70. So:

$23.80 : $0.70 = 34 times coin addition

Second guess: (1+ 34) 50 cent coin + (15+ 34) 20 cent coin

                             = 35 (50 cents) + 49 (20 cents)

                             = $17.50 + $9.80

                             = $27.30

So, the number of 50-cent coins in the bag is 35 coins and the number of 20-cent coins is 49 coins.

24. Your questionZaida has P coins and Wan has Q coins. If Zaida gives 20 coins to Wan then both of them have the same number of coins. If Wan gives 50 coins to Zaida, then the ratio of Zaida’s to Wan’s coin is 3:2. Find the number of coin P and coin Q​


Jawab:

Q = 330

P = 370

Penjelasan dengan langkah-langkah:

Zaida has P coins and Wan has Q coins.

If Zaida gives 20 coins to Wan then both of them have the same number of coins.

this can be written as: P - 20 = Q + 20 or P = Q + 40

If Wan gives 50 coins to Zaida, then the ratio of Zaida’s to Wan’s coin is 3:2.

this can be written as: P + 50 : Q - 50 = 3 : 2

P + 50 = 3/2 (Q - 50)

P = 3/2 Q - 125

Q + 40 = 3/2 Q - 125

Q = 165 x 2 = 330

P = 370
