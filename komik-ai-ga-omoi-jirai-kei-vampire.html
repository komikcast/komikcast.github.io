24. jira mengikuti lomba olimpiade matematika. Jira harus menyelesaikan 40 soal pilihan ganda. Jira akan mendapatkan nilai 4 jika menjawab benar, 0 jika tidak menjawab dan -2 jika jawabannya salah. Jira berhasil menjawab 32 soal dengan benar, 4 soal tidak dijawab dan sisanya salah. Berapakah nilai yang diperoleh Jira?Tolong disertakan dengan diketahui, ditanya nya, terimakasih​


- Matematika

Dik :

Dijawab Benar = 32 Soal

Dijawab Salah = 4 Soal

Tidak Dijawab = 4 Soal

Benar = 4 Poin

Salah = -2 Poin

Tidak Dijawab = 0 Poin

Nilai = ?

Nilai = Nilai Benar - Salah - Tidak Dijawab

Nilai = ( 32 × 4 ) - ( 4 × 1 ) - ( 4 × 0 )

Nilai = 124 - 4 - 0

Nilai = 120

Jawaban:

116

Penjelasan dengan langkah-langkah:

Diketahui =

Terdapat 40 soal pilihan ganda ( JS )Skor jika menjawab benar ( SkB ) = 4Skor jika menjawab salah ( SkS ) = - 2Skor jika tidak menjawab ( SkTJ ) = 0Jira berhasil menjawab 32 soal dengan benar ( SB ), 4 soal tidak dijawab ( STJ ) dan sisanya salah ( SS ) .

Ditanyakan = Nilai yang diperoleh Jira?

Penyelesaiannya =

Pertama, kita cari terlebih dahulu jumlah soal yang salah.

Soal Salah = JS - ( SB + STJ )

= 40 - ( 32 + 4 )

= 40 - 36

= 4

Kedua, kita mencari jumlah skor masing - masing yaitu skor benar, skor salah, dan skor tidak dijawab.

Jumlah Skor Benar ( JB ) = SB x SkB

= 32 x 4

= 128

Jumlah Skor Salah ( JS ) = SS x SkS

= 4 x -2

= - 8

Jumlah Skor Tidak dijawab ( JTJ ) = STJ x SkTJ

= 4 x 0

= 0

Terakhir, kita tinggal jumlahkan semua skornya.

Nilai Jira = JB + JS + JTJ

= 128 + ( - 8 ) + 0

= 116

Jadi, Nilai yang diperoleh Jira adalah 116.

38. kore ga boku no ai itu apa yaa ?


Kore ga boku no ai artinya dalam bahasa Indonesia adalah inilah cintaku.

Pembahasan

Jika ditulis dalam huruf Jepangnya maka menjadi これが僕の愛 yang dibaca kore ga boku no ai.

Terjemahan kata-perkata :

これが僕の愛

これ (kore)

Artinya adalah 'ini' atau 'inilah'. Merupakan salah satu kategori kata tunjuk dalam bahasa Jepang selain dari kata sore dan are.

が (ga)

Suatu partikel yang dimana menyatakan suatu penegasan atas subjek atau objek, dan juga berperan sebagai jenis keterangan dan keberadaan terhadap subjek atau objek.

僕 (boku)

Artinya adalah saya ataupun aku. Salah satu kata ini paling umum banyak digunakan oleh para lelaki. Boku merupakan salah satu jenis kata ganti orang yang pertama.

の (no)

Suatu partikel yang menyatakan jenis suatu kepemilikan yang dimilikinya yang terdapat dalam kalimat. Dimana dalam kalimat diatas, terdapat kata 愛 (ai) merupakan jenis yang dimiliki oleh kepemilikan tersebut

愛 (ai)

Dalam bahasa Indonesia artinya adalah cinta.
