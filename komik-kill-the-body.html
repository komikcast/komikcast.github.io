4. My cat didn't kill the bird Ubah ke kalimat pasif


MAPEL : BAHASA INGGRIS

MATERI : ACTIVE AND PASSIVE VOICE.


SOAL DAN PEMBAHASAN

========================

My cat didn't kill the bird. (Active simple past)


Rumus simple past

(+) Subject + V2 + Object

(-) Subject + Did + Not + V1 + Object

(?) Did + Subject+ V1 + Object


Karena kalimat di atas merupakan kalimat negatif, maka berikut adalah rumus kalimat pasive dalam bentuk negatif.

(-) Subject + To be (was/were) + not + V3 + By...


CATATAN !!!

Subject kalimat aktif berubah jadi object kalimat pasif.


THE ANSWER

============

My cat didn't kill the bird… (active)

Passive nya :

=> The bird was not killed by my cat.

The bird was not killed by my cat
