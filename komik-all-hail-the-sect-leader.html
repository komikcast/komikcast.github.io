10. bentuk sederhana dari sect -cost/tant​


Jawaban:

sin(t)

Penjelasan dengan langkah-langkah:

sec(t) = 1/cos(t)

( 1/cos(t) - cos(t) ) / tan(t)

= ( 1 - cos²(t) / cos(t) ) / tan(t)

karena, sin²(t) + cos²(t) = 1, maka

= ( sin²(t) / cos(t) ) / tan(t)

karena sin(t)/cos(t) = tan(t), maka

= ( sin(t) x tan(t) / tan(t) )

= sin (t)
