Jawaban:

D. The lady did not buy a pair of sporty shoes.

Penjelasan:

Bentuk kalimat adalah past tense.

Bought = verb 2 dari buy.

Untuk membuat kalimat past tense berbentuk negatif, maka tambahkan did not.

Did = verb 2 dari do.

[tex]f'(x) = 2kx - 2kh[/tex]
titik minimum terjadi ketika f'(x) = 0

25. can anybody help me? please trogonometry class XI ​


cos α = 7/8

Penjelasan dengan langkah-langkah:

tangen (tan) = sin/cos atau y/x

tan x = √15/7

cos adalah x/r

jika diketahui tan dan ditanya cos maka harus mencari nilai r

dengan menggunakan rumus pythagoras

r² = x² + y²

r = √(x² + y²)

r = √(7² + (√15)² )

r = √(49 + 15)

r = √64

r = 8

lalu masukkan r nya ke rumus cos

cos α = x / r

cos α = 7/8
