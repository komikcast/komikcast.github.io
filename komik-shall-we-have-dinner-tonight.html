5. 1)Have we … before? (meet) 2)I have … my dinner. (have) *​


[tex]{\bold{Answers:}}[/tex]

methad

[tex]{\bold{Explanation:}}[/tex]

Present Perfect :

Digunakan untuk menyatakan hal yang sudah/belum terjadi /dilakukan di masa lalu.Digunakan untuk menyatakan hal yang sudah terjadi berulang kali.Digunakan untuk menyatakan hal yang terdapat kata "since" (sejak).

Form :

(+) S + have/has + V3 + O/C

(–) S + have/has + not + V3 + O/C

(?) Have/has + S + V3 + O/C

Keterangan :

S = Subject (Subjek) ; I you, they, we, she, he, it, etc. O = Object (Objek) C = Compliment (Keterangan) V3 = Participle, e. g : eaten, driven, seen, etc.Have = digunakan untuk subjek I, you, they, we.Has = digunakan untuk subjek she, he, it.

Note :

Untuk V3 dari meet, adalah met.

Untuk V3 dari have, adalah had.
