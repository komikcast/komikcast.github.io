7. my big brother is .... than me. a. strong c. strongest b. stronger d. more stronger


b. stronger.......................Penjelasan dulu :

Ini materi Degree of Comparison (Tingkat Perbandingan)
Degree of Comparison ada 3 bagian, yaitu :

1. Positive Degree (Sifat sama)
Rumus : S + tobe + as + adj + as + object
Example : Putra is as tall as Feishal


2. Comparative Degree (Sifat lebih)
Aturan :
  *Jika satu/dua suku kata, maka rumus : S + tobe + Adj + er + than + object
Example : Alwi is taller than Iskandar

  *Jika tiga/lebih suku kata, maka rumus : S + tobe + more + Adj + than + object
Example : Ihvan is more diligent than Revi


3. Superlative Degree (Sifat paling)
Aturan :
  *Jika satu/dua suku kata, maka rumus : S + tobe + the + Adj + est + object
Example : Alya is the smartest student in our class

  *Jika tiga/lebih suku kata, maka rumus : S + tobe + the most + Adj + object
Example : Bintang is the most diligent student in our class



Soal :
My big brother is ... than me
--> Pertanyaan ini termasuk ke dalam Comparative Degree (Sifat lebih)

Jadi, jawabannya adalah b.stronger
