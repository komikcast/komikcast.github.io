❇Penjelasan:
↪しっかり (shikkari): sungguh-sungguh / tekun / rajin
↪ 勉強 (benkyou): belajar
↪ すれば (sureba): kalau / jika
↪ 早く (hayaku): cepat (bisa berubah menjadi Hayai, tetapi tergantung pola kalimatnya)

(しっかり勉強すれば、早く上手になるでしょう)

Terjemahannya adalah:
Jika belajar dengan giat, akan cepat menjadi pandai

Kosakata:
Shikkari (しっかり) ----->> Tekun / Giat 
Benkyou (勉強) ---------->> Belajar
Sureba (すれば) --------->> Jika / Kalau
Hayaku (早く ) ----------->> Cepat
Jouzu (上手) -------------->> Pandai
Ni naru (になる) --------->> Menjadi
Deshou (でしょう) ----->> Akan (Bentuk pengandaian)

日本の季節

日本の季節は春、夏、秋、冬があります。
春はあたたかいくて、綺麗な花がたくさん咲きます。桜も咲きます。
夏は一年で一番暑いです。七月と八月は夏休みで学校は休みです。海は山など絵遊びに行く人が多いです。
暑い 夏 が 終わる と, 涼しい 秋 に なります。 木ノ葉が色色な色に変わって, 綺麗 です。秋わ果物が大変美味しいです。
冷たい風が冬が来ます。冬になったら山 で わ スカイ が できます。

[Nihon no kisetsu
Nihon no kisetsu wa haru, natsu, aki, fuyu ga arimasu.
Haru wa atataka ikute, kireinahana ga takusan sakimasu. Sakura mo sakimasu.
Natsu wa ichinen de ichiban atsuidesu. sichigatsu to hachigatsu wa natsuyasumi de gakkō wa yasumidesu. Umi wa yama nado e asobiniiku hitogaōidesu. Atsui natsu ga owaru to, suzushī aki ni narimasu. Ki no wa ga iro-irona iro ni kawatte, kireidesu. Aki wa kudamono ga taihen oishīdesu. Tsumetai kaze ga fuyu ga kimasu. Fuyu ni nattara yamade wa sky ga dekimasu.]

Artinya :

Musim di jepang

Musim di jepang adalah musim semi, musim panas, musim gugur, dan musim dingin.
Musim semi sangat hangat. Banyak bunga-bunga indah bermekaran. Bunga sakura juga bermekaran.
Musim panas adalah musim yang paling panas dalam 1 tahun. Bulan Juli dan Agustus adalah liburan musim panas, Sekolah juga diliburkan. Baik pantai maupun gunung merupakan tempat banyak orang menghabiskan liburannya di musim ini.
Ketika panasnya musim panas berakhir, Akan berganti menjad dinginnya musim gugur. Dedaunan berubah warna menjadi berbagai macam warna yang indah. Buah di musim gugur sangat lezat.
Angin dingin di musim dingin datang. Saat musim dingin, anda dapat melakukan sky di atas gunung.

17. arti nihon jin ni naru


Nihon jin ni naru dalam bahasa Indonesia artinya adalah menjadi orang Jepang.

Pembahasan

Jika kalimat Nihon jin ni naru diubah ke dalam bentuk huruf Jepangnya adalah 日本人になる = menjadi orang Jepang.

Terjemahan kata-perkatanyajuga dengan penjelasannya :

日本人になる (Nihon jin ni naru)

日本 (Nihon)

Artinya yaitu Jepang. Sebenarnya kata 日本 ini tidak hanya bisa disebut sebagai Nihon, namun juga Nippon. Akan tetapi, kata Nihon lebih sering digunakan oleh orang-orang Jepang disana dibandingkan kata Nippon.

人 (hito/jin/nin)

Kalau dibaca kun'yomi (cara baca Jepang), maka dibaca hito. Sedangkan jika dibaca secara on'yomi (cara baca China), maka kita baca sebagai nin/jin. Kata dari 人 ini sendiri terbentuk dari huruf kanji. Dalam bahasa Indonesianya yang berarti orang, manusia.

になる (ni naru)

Jenis tata bahasa untuk bahasa Jepang. Termasuk kedalam golongan kata yang informal. Selain itu memiliki arti yakni menjadi atau akan menjadi.

23. Kaizoku oni ore wa naru​


Kalimat dari kaizoku ou ni ore wa ni naru artinya dalam bahasa Indonesia adalah aku akan menjadi raja bajak laut.

Pembahasan

Jika kalimat kaizoku ou ni ore wa ni naru ditulis dalam bentuk huruf Jepangnya, maka menjadi 海賊王に俺はになる

Tentunya, mungkin kalian pernah mendengar salah satu kalimat ini dari manga/anime yang kalian tonton yaitu One Piece. Kalimat ini pastinya suka diucapkan oleh kapten dari bajak laut Mugiwara, yaitu Monkey D Luffy.

__________________________________________

海賊王に俺はになる

(kaizoku ou ni ore wa ni naru)

Terjemahan kata-perkatanya, beserta dengan penjelasannya :

=> 海賊 (kaizoku)

Artinya dalam bahasa Indonesia adalah bajak laut.

=> 王 (ou)

Dalam bahasa Indonesia yang berarti raja. Umumnya digunakan sebagai pemanggilan suatu jabatan yang tertinggi. Kata raja dalam bahasa Jepang juga memiliki banyak kata sebagai berikut :

キング (kingu).国王 (kokuou).皇帝 (koutei).

=> Partikel に (ni)

Partikel ni ditulis dalam huruf hiragana dari に (ni). Memiliki jenis fungsi sebagai suatu keterangan terhadap kalimat dimaksud, menyatakan suatu tujuan ataupun target yang ingin dicapai.

=> 俺 (ore)

Artinya berarti gue atau aku (secara kasar). Karena ingin bukanlah termasuk jenis kata yang sopan, sehingga kata ini disebut sebagai kata yang informal atau tidak sopan. Kata ore adalah jenis kata ganti orang pertama tunggal.

=> Partikel は (wa)

Asalkan kalian tahu, bahwa partikel wa berasal dari salah satu huruf hiragana は yang seharusnya dibaca ha. Namun karena khusus untuk partikel maka boleh kita baca wa. Memiliki suatu fungsi untuk menjadikan subjek atau objek sebagai suatu topik kalimat.

=> になる (ni naru)

Artinya yang berarti menjadi atau akan menjadi. Suatu jenis kata yang tergolong tata bahasa dalam bahasa Jepang. Bisa kita gunakan untuk jenis kata benda maupun juga kepada kata sifat.

Dan juga kata ni naru ini merupakan suatu tata bahasa dalam bentuk untuk situasi yang digunakan secara informal.

24. Watashima ki ni shimashen itu apa ya?


私は気にしません (Watashi wa ki ni shimasen) artinya adalah "Saya tidak memikirkannya", "Saya tidak memedulikannya", "Saya tidak masalah", dan lain sebagainya.

私も気にしません (Watashi mo ki ni shimasen) artinya adalah "Saya juga tidak memikirkannya", "Saya juga tidak memedulikannya", "Saya juga tidak masalah", dan lain sebagainya.

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Pembahasan :

Sebelum ke pembahasan per-kata nya, saya ralat sedikit penulisan kalimatnya. Penulisannya bukan 'Watashima ki ni shimashen', melainkan 'Watashi wa ki ni shimasen' atau 'Watashi mo ki ni shimasen'

= Watashima ki ni shimashen ❌

= Watashi wa ki ni shimasen ⭕

= Watashi mo ki ni shimasen ⭕

Kedua kalimat tersebut sama-sama termasuk kalimat yang sopan. Adapun bentuk kasual dari kedua kalimat tersebut yaitu 'Watashi wa ki ni shinai' dan 'Watashi mo ki ni shinai'.

- - - - - - - -

Berikut pembahasan per-kata nya.

WATASHI「私・わたし」

Merupakan kata ganti orang pertama tunggal yang paling umum untuk digunakan sehari-hari. Penggunaannya sama seperti "Aku" atau "Saya". Kata ini termasuk sopan, dan bisa digunakan di situasi informal maupun formal, oleh laki-laki maupun perempuan, baik tua maupun muda.

WA「は」

Merupakan partikel atau kata bantu kalimat yang berfungsi untuk menandakan subjek atau topik dari sebuah kalimat. Kata yang terletak sebelum partikel ini ditandakan sebagai subjek atau topiknya, kemudian yang terletak setelah partikel ini merupakan keterangan dari subjek atau topiknya. Partikel ini biasanya tidak diartikan, tapi bisa berarti "Adalah".

MO「も」

Merupakan partikel atau kata bantu kalimat yang berfungsi menunjukkan hal yang sama dengan yang sudah ditunjukkan sebelumnya. Mudahnya, partikel ini dalam Bahasa Indonesia sama seperti kata "Juga".

KI NI SHIMASEN「気にしません」

Merupakan kata kerja bentuk negatif sopan yang berarti "Tidak memedulikan" atau "Tidak memikirkan". Satu kosakata ini dibagi menjadi beberapa kata yaitu Ki (Pikiran), Ni (Partikel penunjuk tujuan), Shimasen (Tidak melakukan). Bentuk awal dari kata kerja ini adalah 'Ki ni suru', yang berarti "Memedulikan" atau "Memikirkan".

KI NI SHINAI「気にしない」

Merupakan kata kerja bentuk negatif biasa yang artinya sama seperti 'Ki ni shimasen', hanya berbeda di tingkat kesopanannya saja. Kata ini adalah versi kasual dari 'Ki ni shimasen'.

32. apa arti ( o ki ni mesu mama) *sayonara minna:)


Arti dari お気に召すまま (Oki ni mesu mama) dalam Bahasa Indonesia adalah "Dengan sesukamu". Kalimat ini termasuk sopan tapi tidak biasa digunakan dalam percakapan sehari-hari.

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Pembahasan :

Berikut pembahasan per-kata nya.

お気に召す (Oki ni mesu)

Artinya adalah "Menyukai" atau "Memfavoritkan". Kata ini merupakan Sonkeigo (Bahasa penghormatan) dari kata 'Ki ni iru' yang memiliki arti serupa. Karena merupakan Sonkeigo, jadi hanya digunakan saat berbicara dengan lawan bicara yang derajatnya jauh lebih tinggi saja. Kalau bukan, lebih baik menggunakan 'Ki ni iru' saja.

まま (Mama)

Tidak ada arti khusus, namun makna nya seperti "Dengan dibiarkan / apa adanya". Kata ini sendiri merupakan salah satu tata Bahasa Jepang yang digunakan untuk menyatakan sesuatu dibiarkan atau dilakukan apa adanya saja. Kata ini diletakkan setelah predikat.

Kalimat 'Oki ni mesu mama' ini juga merupakan judul dari sebuah lagu yang dinyanyikan oleh salah satu penyanyi asal Jepang yang bernama Eve. Lagu ini merupakan lagu yang dibuatnya sendiri, dan awalnya dinyanyikan oleh vocaloid terkenal yaitu Hatsune Miku. Namun kemudian Eve mengcover lagunya sendiri, dan dirilis pada November 2017.
