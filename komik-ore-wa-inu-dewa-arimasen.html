1. Hiragana kore wa teburu dewa arimasen


これはてーぶるではありません 「これはテーブルではありません」Kore wa Teeburu dewa Arimasen.

Ini bukan Meja.

Teburu? Teeburu?

てぶる Teburu

てーぶる Teeburu

Penjelasan

Pola Kalimat:

Subyek/kata tunjuk + は ( Wa ) + Keterangan + ではありません ( Dewa Arimasen )

これはテーブルではありません 「 Kore Wa Teeburu Dewa Arimasen 」Ini Bukan Meja

言葉 「ことば」

• これ ( Kore ) Ini

• は ( Wa ) Partikel

• テーブル ( Teeburu ) Meja

• ではありません ( Dewa Arimasen ) Bukan

Kata ganti penunjuk

Wa merupakan partikel , Partikel wa ini berfungsi sebagai penunjuk subjek dari suatu kalimat , atau bisa disebut sebagai penunjuk dari pokok kalimat, Wa ditulis menjadi ( Ha ) bukan ( Wa ) , Tetapi tetap dibaca wa

5. iie, genki dewa arimasen (HIRAGANA) ​


Versi hiragananya adalah いいえ、げんきではありません (iie, genki dewa arimasen).

Pembahasan

いいえ、げんきではありません

Iie, genki dewa arimasen

________

いいえ (iie)

Yaitu artinya 'bukan', 'gak', atau 'tidak. Bentuk kata yang umum dipakai sebagai bentuk penolakan ataupun pertentangan.

元気 • げんき (genki)

Versi kanji : 元気Hiragana : げんき

Artinya tersebut bisa berarti 'kabar', 'sehat', atau 'bersemangat'.

ではありません (dewa arimasen)

Dalam bahasa Indonesia kalimat tersebut berarti 'bukan' atau 'tidak' dengan bentuk penggunaan yang formal. Kalimat tersebut merupakan jenis kalimat yang bermakna negatif.

Contoh penggunaannya adalah seperti berikut ini :

watashi wa issha dewa arimasen : aku bukan seorang dokter.watashi wa Nihon jin dewa arimasen : aku bukan orang Jepang.kore wa nootobukku dewa arimasen : ini bukan buku catatan.

7. apa arti dari "haha wa sensei dewa arimasen"?​


Arti dari 母は先生ではありません (haha wa sensei dewa arimasen) artinya adalah "ibu(ku) bukan seorang guru."

Pembahasan

Pola kalimat

(+) KB1 + partikel は (wa) + KB2 + です (desu).(-) KB1 + partikel は (wa) + KB2 + ではありません (dewa arimasen).(?) KB1 + partikel は (wa) + KB2 + ですか？(desuka?).

Kalimat 母は先生ではありません (haha wa sensei dewa arimasen) adalah bentuk pasif, sedangkan bentuk positif dan bentuk tanya adalah sebagai berikut :

(+) 母は先生です (haha wa sensei desu)

= ibu adalah seorang guru.

(?) 母は先生ですか？(haha wa sensei desuka?)

= apakah ibu adalah seorang guru?

___

母は先生ではありません。

Haha wa sensei dewa arimasen.

Ibu(ku) bukan seorang guru.

Haha「母・はは」: "ibu" pemakaian kata ini biasa ditujukan untuk menyatakan ibu sendiri.

Wa「は」: partikel yang tepatnya sebagai suatu penanda topik.

Sensei「先生・せんせい」: sebenarnya arti dari kata ini cukup banyak dan juga luas, biasa digunakan sebagai pemanggil. Kata ini digunakan sesuai dengan konteks kerjanya masing-masing. Artinya bisa berarti "guru," "dokter," "mangaka," "pengacara," dll.

Dewa arimasen「ではありません」: bentuk pasif atau negatif formal yang diartikan "bukan" atau "tidak.

29. Apa perbedaan dari dewa arimasen sama ja arimasen?


Untuk perbedaan diantara keduanya adalah dewa arimasen penggunaannya formal/resmi, tidak seperti ja arimasen yang dimana penggunaannya semi-formal atau informal.

Pembahasan

Kedua kalimat tersebut, untuk persamaannya adalah terletak diartinya. Karena arti diantara kedua kalimat tersebut adalah tidak/gak/bukan.

ではありません

dewa arimasen

Untuk menggunakan kalimat ini cocok dipakai pada saat situasi yang resmi atau formal. Kalimat ini mengungkapkan mengenai hal yang tidak termasuk/berkaitan atau bertentangan dengan hal lain.

_____________________

じゃありません

ja arimasen

Untuk penggunaan atau cara pengungkapannya sama seperti sebelumnya, namun penggunaan yang cocok dalam situasi untuk mengucapkan ja arimasen ini adalah bisa digunakan dalam situasi informal tapi juga bisa semi-formal (kurang resmi).

_____________________

Pola kalimat penggunaan ではありません • じゃありません (dewa arimasen/ja arimasen) adalah :

S/O + は (wa) + (....kalimat) keterangan + ではありません • じゃありません (dewa arimasen/ja arimasen)

Ada juga bentuk ungkapan lain untuk menyatakan 'tidak/gak/bukan' selain ではありません (dewa arimasen) dan じゃありません (ja arimasen) dalam bahasa Jepang :

じゃない (janai) –› bentuk informal.ではない (dewa nai) –› semi formal (kurang resmi).ではありませんでした (dewa arimasen deshlta) –› khusus digunakan untuk bentuk lampau (formal).じゃありませんでした (ja arimasen deshlta) –› juga termasuk bentuk lampau (semi-formal).

Perbedaan dari 'Dewa arimasen' dengan 'Ja arimasen' adalah 'Dewa arimasen' LEBIH FORMAL dari 'Ja arimasen'.

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Pembahasan :

Apa arti 'Dewa/Ja arimasen'?

Kalau diartikan, artinya seperti "Bukan...". Biasa diletakkan di akhir sebuah pernyataan, untuk menyatakan bahwa pernyataan tersebut tidaklah benar.

Contoh kalimat :

Ningen dewa arimasenNingen ja arimasen

Kedua kalimat tersebut sama-sama berarti "Bukan manusia". Namun tingkat keformalannya berbeda sedikit.

- - - - - - - - -

'Dewa arimasen' vs 'Ja arimasen'

Keduanya sama-sama sopan, tapi terdapat sedikit perbedaan pada tingkat keformalannya. 'Dewa arimasen' digunakan pada situasi formal, dan penggunaannya seperti ke bos, guru, dsb. 'Ja arimasen' digunakan pada situasi semi-formal, dan penggunaannya seperti ke keluarga, dsb.

- - - - - - - - -

Versi kasual dari 'Dewa/Ja arimasen'?

Kata 'Arimasen' merupakan bentuk negatif sopan dari kata kerja 'Aru', sedangkan bentuk negatif biasanya adalah 'Nai'. Dari sini dapat diketahui bahwa 'Arimasen' lebih sopan dari 'Nai'. Maka dari itu, versi kasual dari 'Dewa/Ja arimasen' adalah 'Dewa/Ja nai'. Kedua kata 'Dewanai' dan 'Janai' sama-sama merupakan versi kasualnya, tapi 'Dewanai' sedikit lebih formal dari 'Janai'.

35. Indo / dewa arimasen / jarjit / wa / jin / san Susunlah kata-kata diatas menjadi kalimat yang sempurna! *


Susunan kalimat yang benar adalah 'Jarjit-san wa Indoneshia jin dewa arimasen.'

Pembahasan

Jarjit-san wa Indoneshia jin dewa arimasen

「ジャージットさんはインドネシア人でわありません」

= Jarjit bukan orang Indonesia.

_____

Jarjit-san「ジャージットさん」

Kata ジャージット yang dibaca jaajitto, yang berarti Jarjit (merupakan nama seseorang). Ditulis dengan katakana karena bukan jenis kata/nama Jepang. Kemudian kata さん (san) yaitu semacam gelar atau panggilan yang tergolong sopan.

Wa「は」

Partikel yang dapat menyatakan atau penanda subjek dan juga objek.

Indoneshia「インドネシア」

Yaitu Indonesia, penulisannya dengan katakana. Selain untuk menulis nama seseorang yang merupakan nama asing, fungsi katakana juga dapat untuk menuliskan nama negara yang merupakan negara lain.

Jin「人 • ジン」

Arti 人 (jin) yang berarti orang.

Ditulis dengan kanji, tetapi dibaca secara On'yomi (cara baca China). Dalam satu kanji, cara bacanya ada beberapa yang tergolong kun'yomi dan on'yomi.

Dewa arimasen「でわありません」

Kalimat formal, merupakan pengganti desu namun bersifat negatif karena dalam bahasa Indonesia diartikan bukan atau tidak. Umumnya kata でわ ありません (dewa arimasen) terletak diakhir kalimat.
