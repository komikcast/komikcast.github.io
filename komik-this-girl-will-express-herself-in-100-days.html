33. The little girl fell in the field this morning. (-) (?)


Jawaban:

The little girl fell in the field this morning.

(-) The little girl did not fall in the field this morning.(?) Did The little girl fall in the field this morning?

PEMBAHASAN Past Tense

Digunakan untuk menyatakan Kegiatan yang dilakukan di masa lalu atau Peristiwa di masa lampau.

Ciri waktu Past Tense

=> Yesterday

=> This morning

=> Last night

=> Two weeks ago

Rumus Past Tense (+) S + Verb-2(-) S + did not + Verb-1(?) Did + S + Verb-1

Catatan Kalimat soal terdapat ciri Past Tense yaitu “this morning” maka pada kalimat Positive menggunakan Verb-2 “fell”Tapi pada kalimat Negative (-) dan Interrogative (?) menggunakan Verb-1 “fall” karena sudah ada DID

34. In a project, 10 tasks from A to J need to be completed. Each task can be done in specific number of days (as indicated in the diagram). Given this, how many days will it take to complete the project?


In a project, 10 tasks from A to J need to be completed. Each task can be done in specific number of days (as indicated in the diagram). From that, we could see that the project should be completed in 23 days.

Penjelasan dengan Langkah-langkah

Diketahui:

Possible paths:

A - B - E - G - H - JA - B - D - F - JA - C - I - J

Ditanyakan:

Butuh berapa hari untuk menyelesaikan projek tersebut.

Jawab:

Durasi path A - B - E - G - H - J = 3 + 3 + 2 + 2 + 4 + 5 = 19Durasi path A - B - D - F - J = 3 + 3 + 5 + 5 = 16Durasi path A - C - I - J = 3 + 6 + 9 + 5 = 23

Untuk menentukan butuh berapa hari untuk menyelesaikan projek tersebut, kita bisa melihat dari durasi maksimum. Jadi, ternyata butuh 23 hari untuk menyelesaikan projek tersebut.
