5. watashi no shuumi wa .... koto desu. (menonton film) a. ryouri o suru b. sakka o suru c. hon o yomu d. eiga o miru


Watashi no shumi wa eiga o miru (D) koto desu.

Pembahasan

私の趣味は映画を見る事です

Romaji :

Watashi no shumi wa eiga o miru koto desu.

Artinya :

Hobiku menonton film.

__________

Soal diatas, menanyakan mengenai arti 'menonton film' dalam bahasa Jepang, jawaban yang tepat adalah opsi D. Eiga o miru. Kemudian mengenai arti opsi lainnya di opsi A, B, dan C adalah :

A. Ryouri o miru「料理を見る」: melihat masakan.B. Sakka o suru「作家をする」: menjadi penulis.C. Hon o yomu「本を読む」: membaca buku.

__________

Watashi no shumi wa eiga o miru koto desu

「私の趣味は映画を見ることです」

Setiap terjemahan kosakata dengan penjelasannya juga sebagai berikut.

私・わたし (watashi) = saya, aku. Bentuk umum dan juga tergolong formal.

の (no) = *partikel yang dapat memberitahukan mengenai kepemilikan.

趣味・しゅみ (shumi) : hobi (kegiatan yang disukai).

は (wa) = *partikel yang dijadikan penanda subjek.

映画・えいが (eiga) = film.

を (o/wo) = *partikel yang penggunaannya sebagai penanda objek.

見る・みる (miru) = secara harfiah maknanya yaitu melihat, namun jika digabungkan dengan kata benda seperti 映画 (eiga) maka dapat diartikan menonton atau menyaksikan.

事・こと (koto) = tergantung konteks kalimatnya, kata ini bisa diartikan tentang, hal. Dapat dijadikan sebagai pola kalimat menyatakan hobi dalam bhs Jepang.

です (desu) = *bentuk kata bantu untuk menyatakan kesopanan/kehalusan. Bisa untuk penekanan.

8. gohan ni suru? ofuro ni suru? sore tomo..... ​


'Gohan ni suru? Ofuro ni suru? Sore tomo ...' artinya adalah (apa) mau makan? mau mandi? Ataukah ...

Pembahasan

Gohan ni suru?「ごはんにする?」

Arti kalimat tersebut ialah mau makan?

Kata ご飯 • ごはん (gohan) memiliki 2 arti yakni makanan atau nasi, beras (sesuai konteks dari kalimatnya). Pemakaian kata gohan ini terkesan sopan.

Ofuro ni suru「おふろにする?」

Arti kalimat tersebut ialah mau mandi?

Kata お風呂 • おふろ (ofuro) yang berarti mandi, terdapatnya ada huruf awalan お (o) itu menunjukkan agar kata tersebut memiliki kesan yang sopan. Sebenarnya kata ofuro ini dapat berarti berendam (di bak mandi).

Adanya kata akhiran 〜にする (ni suru) di kedua kalimat tersebut merupakan bentuk pola kalimat yang biasa dipakai untuk menyatakan atau menanyakan suatu pilihan yang ingin dipilih antara salah satunya. Bisa untuk pemilihan atau keputusan dnegan bentuk tanya maupun menyatakan.

POLA KALIMAT

1). Menyatakan keberadaan

=> [KB] ga arimasu.

=> [Letak] ni arimasu.

    Ada di [Letak].

=> [Letak] ni [KB] ga arimasu.

    Di [Letak] ada [KB].

=> [KB] wa [Letak] ni arimasu.

    [KB] ada di [Letak].

2). Menyatakan kebisaan

=> [KK] koto/no ga dekimasu.

    Bisa [KK].

    Di Yogyakarta ada Prambanan.

- Puranbanan wa jogujakaruta ni arimasu.

35. Jogujakaruta wa jawa ni arimasu. kankouchi ga takusan arimasu. puranbanan ya jogujakaruta no oukyuu nado desu. puranbanan ni furui otera ga takusan arimasu. jawa no dansu o miru koto ga dekimasu. gamuran no enzou o kiku koto ga dekimasu. oukyuu ni hakubutsukan ga arimasu. ramayana no dansu o miru koto ga dekimasu. totemo kirei desu.


Terjemahan :

Yogyakarta ada di Jawa. Tempat wisata nya ada banyak. Prambanan, Keraton Yogyakarta, dan lain-lain. Di Prambanan ada banyak candi tua. Bisa melihat tari Jawa. Bisa mendengar pertunjukan gamelan. Di Keraton ada museum. Bisa melihat tari Ramayana. Sangat indah.

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

PEMBAHASAN

Pada teks tersebut terdapat banyak kalimat yang menyatakan keberadaan sesuatu dan kebisaan melakukan sesuatu. Kuncinya terdapat pada kata 'Arimasu' yang berarti "Ada", dan 'Dekimasu' yang berarti "Bisa".

- - - -

POLA KALIMAT

1). Menyatakan keberadaan

=> [KB] ga arimasu.

   Ada [KB].

=> [Letak] ni arimasu.

   Ada di [Letak].

=> [Letak] ni [KB] ga arimasu.

   Di [Letak] ada [KB].

=> [KB] wa [Letak] ni arimasu.

   [KB] ada di [Letak].

2). Menyatakan kebisaan

=> [KK] koto/no ga dekimasu.

   Bisa [KK].

- - - -

CONTOH KALIMAT

1). Menyatakan keberadaan

- Puranbanan ga arimasu.

   Ada Prambanan.

- Jogujakaruta ni arimasu.

   Ada di Yogyakarta.

- Jogujakaruta ni puranbanan ga arimasu.

   Di Yogyakarta ada Prambanan.

- Puranbanan wa jogujakaruta ni arimasu.

   Prambanan ada di Yogyakarta.

2). Menyatakan kebisaan

- Shashin o toru koto ga dekimasu.

   Bisa mengambil foto.

- Keeki o tsukuru no ga dekimasu.

   Bisa membuat kue.
