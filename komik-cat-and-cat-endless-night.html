14. did you see my cat last night(?) ...................(-) ..................(+)


Jawaban:

(+) You saw my cat last night.(-) You did not see my cat last night.(?+) Did you see my cat last night?(?-) Didn't you see my cat last night?

Penjelasan:

Past Tense

The grammar used to describe something happening in the past.

Subject = You, verb₁ = see, verb₂ = saw, object = my cat

Past Tense rules:(+) Subject + Verb₂ + Object

Example: You saw my cat last night.

Kemarin malam, kamu melihat kucing saya.

(-) Subject + did not + Verb₁ + Object

Example: You did not see my cat last night.

Kemarin malam, kamu tidak melihat kucing saya.

(?+) Did + Subject + Verb₁ + Object ?

Example: Did you see my cat last night?

Kemarin malam, apakah kamu melihat kucing saya?

(?-) Didn't + Subject + Verb₁ + Object ?

Example: Didn't you see my cat last night?

Kemarin malam, tidakkah kamu melihat kucing saya?

33. Cat air merupakan salah satu warna berbasis air, dengan sifat bahan yang mudah larut dalam air. Selain cat air, contoh warna berbasis air lainnya seperti .... a. cat kayu, cat besi, cat tembok b. cat mobil, cat genting, cat besi c. cat kayu, akrilik, cat tembok d. cat tembok, akrilik, cat poster e. cat genting, cat kayu, cat besi​


Jawaban:

d. cat tembok, akrilik, cat poster

Penjelasan:

Jawaban d, "cat tembok, akrilik, cat poster," adalah yang benar karena semua jenis cat ini termasuk dalam kategori cat berbasis air yang mudah larut dalam air. Cat tembok digunakan untuk mengecat dinding, akrilik adalah jenis cat serbaguna yang digunakan untuk berbagai permukaan, dan cat poster digunakan dalam seni poster dan lukisan. Jadi, semuanya adalah contoh cat berbasis air.
