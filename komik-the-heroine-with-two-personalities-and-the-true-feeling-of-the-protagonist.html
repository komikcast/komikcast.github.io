9. Complete the rule. RULE : we use ... for things which happen regularly or which are always true. With I, ...., ...., .... and ..., we use the base form of the verb (V1) With He, ... and ..., we add ... (V1 + .../...)​


Jawaban:

RULE: We use the simple present tense for things which happen regularly or which are always true.

With I, we, you, and they, we use the base form of the verb (V1).

With he, she, and it, we add -s or -es (V1 + -s/-es).
