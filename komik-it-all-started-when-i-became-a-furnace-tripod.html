5. the complication started when?​


the complication started when?​

________________________

JAWABAN :

The complication started when the problem is arisen

PENJELASAN :

Complication  ( komplikasi ) termasuk bagian dari struktur narrative text yang memunculkan permasalahan-permasalahan dalam alur ceritanya.

NARRATIVE TEXT

=> Genre teks yang menjabarkan legenda/mitos/cerita fiksi karya manusia.

SOCIAL FUNCTION/COMMUNICATIVE PURPOSE  

Narrative text disajikan dengan memiliki tujuan untuk menghibur para pembaca { to amuse / entertain the readers }

GENERIC STRUCTURE

^ Orientation  

=> Paragraf ini pada umumnya berisi plot cerita sebagai awal pembukaan paragraf pertama . Ini mencakup pengenalan tokoh , waktu terjadi dan keterangan tempat yang ingin dipaparkan dalam cerita.  

^ Complication

=> Penulis menggunakan komplikasi untuk membangun emosi/mood para pembacanya . Bagian ini berisi  permasalahan -permasalahan yang muncul yang terjadi pada tokoh utamanya.

^ Resolution

=> Paragraf ini berisi tentang  penyelesaian permasalahan , apakah berakhir happy ending, sad ending dan atau memberikan efek suspended ( menggantung ) terhadap para pembacanya.

^ Coda

=> Berisi tentang pesan-pesan moral positive yang dapat disimpulkan dari kisah/cerita yang disajikan.

_______________________

KEYWORDS = NARRATIVE TEXT ; GENERIC STRUCTURE ; COMPLICATION

SUBJECT     = ENGLISH

MATERIAL    = NARRATIVE TEXT

GRADE         = JHS

CODE           = 5

9. When i arrived in the classroom, the lecture ... For 30 minutes, so i didn't find it easy to follow. A. Started B. Had started C. Had been starting D. Has been started +Alasannya


Subject : English

Category : Structure (Past Perfect Tense)

Level : SHS

Code : 5


Answer :

When I arrived in the classroom, the lecture HAD BEEN STARTED for 30 minutes, so i didn't find it easy to follow.

The past perfect is used to describe actions that had been completed at some point in the past or that had happened before another action in the past.

Structure : S + had + past participle

Kuliah dimulai sebelum saya tiba. Saya tiba di waktu lampau (menggunakan V₂ arrived) jadi gunakan past perfect untuk kejadian yang terjadi mendahului kejadian lain di waktu lampau.

Klausa menggunakan bentuk pasif karena kuliah dimulai, bukan memulai jadi pola yang digunakan adalah S + had + BEEN + V₃
