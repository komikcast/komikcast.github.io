penjelasan:

kalimat itu merupakan penggalan lirik dari lagu "kaoru amane-taiyou no uta"

yakni pada lirik=

“...kaze ga fuita

Waratte naite kimi to deaete

Mieru sekai wa kagayaki dashita

Himawari yureru Taiyoo no shita de

Kanjiteita kaze o kimi o”

karena merupakan terjemahan dari lagu, jadi pasti banyak kata kiasnya

ーーーーーーーーーーーー

“Kaze ga fuita”

kaze=angin

fuita= ditiup, berhembus

→angin berhembus

ーーーーーーーーーーーー

“waratte naite kimi to deaete ”

waratte=tertawa

naite=menangis

kimi=kamu

to= partikel yg berarti "dan atau dengan"

deaete= bertemu

→tertawa, menangis, dan kemudian bertemu denganmu

ーーーーーーーーーー

“mieru sekai wa kagayaki dashite”

mieru=lihat

sekai=dunia

kagayaki=berkilau, bersinar

dashite=fajat, mulai bersinar

→dunia yang ku lihat mulai bersinar

ーーーーーーーーーー

“himawari yureru taiyou no shita de ”

himawari=bunga matahari

yareru=bergetar, kocok, berayun

taiyou=mentari

shita= dibawah

→bunga matahari berayun di bawah mentari

ーーーーーーーーーー

“kanjiteita kaze wo kimi wo”

kanjiteita=aku merasakan

kaze=angin

kimi=kamu

→aku bisa merasakan angin, aku merasakanmu

ーーーーーーーーーーー

Penggunaan partikel pada lagu itu

●partikel ga=Menunjukkan subjek dan kata bantu dari kata kerja intransitif

●partikel to=berarti dengan, dan

●partikel wa=menunjukkan subjek

●partikel no=menunjukkan kepunyaan

●partikel de=berarti dengan,di;sebab

●partikel wo=kata bantu untuk kata kerja

8. Apa arti kata Guren no Uta (紅蓮の唄 )? ​


Arti dari 紅蓮の唄 (Guren no uta) adalah "Lagu teratai/lotus merah".

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Pembahasan :

Berikut pembahasan per-kata nya.

紅蓮 (Guren)

Artinya adalah "Teratai merah" atau "Lotus merah". Bisa dilihat dari Kanji nya sendiri, yaitu Gu yang berarti "Merah tua" dan Ren yang berarti "Teratai / Lotus".

の (No)

Pada kalimat ini tidak diartikan. Kata ini merupakan sebuah partikel atau kata bantu yang salah satu fungsinya adalah untuk menyatakan suatu kepemilikan. Misalkan pada kalimat 'A no B', maka artinya seperti "B milik A", "B nya A", atau hanya "B A".

唄 (Uta)

Artinya adalah "Lagu". Kata ini bisa digunakan untuk lagu pada umumnya, dan bisa juga dikhususkan untuk lagu-lagu daerah.

9. Apa arti kata dari Kaze ??


Pembahasan

Kata kaze ini dalam bahasa Indonesianya, memiliki 2 arti. Arti yang berbeda, sehingga penulisan kanjinya juga berbeda yaitu :

風 (kaze)

Dalam bahasa Indonesianya yang berarti angin. Kata kaze dalam bentuk hiragananya adalah かぜ

Kata 風 (kaze) ini memiliki cara baca on'yomi dan juga kun'yomi.

Jika dibaca on'yomi (cara baca China), maka cara bacanya yaitu fuu, fu.

Sedangkan, cara baca kun'yomi (cara baca Jepang), maka dibacanya kaza, kaze.

Contoh :

強い風

tsuyoi kaze

Angin kencang.

風船

fuusen

Balon.

風邪 (kaze)

Dalam bahasa Indonesianya yang berarti adalah :

Flu.Masuk angin.Pilek.

Umumnya untuk menyebutkan atau menyatakan suatu penyakit tubuh yang berasal dari masuknya angin ke dalam tubuh ataupun akibat tertularnya dari yang lain.

Contoh :

風邪気

kazeke

(Merasa agak/sedikit flu).

風邪気味

kazegimi

(Merasakan jenis-jenis gejala flu).

14. hiragana tegami,mizu,kaze,kazoku


Bentuk hiragana :

Tegami「てがみ」Mizu「みず」Kaze「かぜ」Kazoku「かぞく」

_____________

Pembahasan

Tegami「てがみ」

Versi kanji : 手紙

Artinya yaitu surat.

Terdiri dari huruf-huruf hiragana :

て (te).が (ga), bentuk hiragana tenten.み (mi).

Mizu「みず」

Versi kanji : 水

Artinya adalah air.

Terdiri dari huruf-huruf hiragana :

み (mi).ず (zu), bentuk hiragana tenten.

Kaze「かぜ」

Versi kanji : 風

Artinya adalah masuk angin, angin.

Terdiri dari huruf hiragana :

か (ka).ぜ (ze), bentuk tenten.

Kazoku「かぞく」

Versi kanji : 家族

Artinya adalah keluarga.

Terdiri dari huruf hiragana :

か (ka).ぞ (zo), bentuk tenten.く(ku).

Perbandingan Termometer Suhu Celcius, Reamur, Fahrenheit, dan Kelvin

Perbandingan Termometer Suhu Celcius, Reamur, Fahrenheit, dan Kelvin

Yang perlu kita ketahui adalah perbandingan suhu antara celcius, reamur, Fahrenheit dan Kelvin adalah 5 : 4 : 9 : 5. Khusus untuk Fahrenheit perlu ditambah 32 untuk perubahannya

A. Rumus merubah celcius ke kelvin= Celcius + 273,15

B. Rumus merubah celcius ke rheamur= Celcius x 0,8

C. Rumus merubah reamur ke celcius= Rheamur x 1,25

D. Rumus merubah celcius ke fahrenheit= (Celcius x 1,8) + 32

E. Rumus merubah Fahrenheit ke celcius= (Fahrenheit - 32) / 1,8

F. Rumus merubah rheamur ke fahrenheit= (Rheamur x 2,25) + 32

Percobaan :

￼

Pada thermometer jenis celcius mempunyai titik beku 0°c, titik didih 100°c, dan jumlah skala adalah 100 skala.

Pada thermometer jenis reamur mempunyai titik beku 0°c, titik didih 80°c, dan jumlah skala adalah 80 skala.

Pada thermometer jenis Fahrenheit mempunyai titik beku 32°c, titik didih 212°c, dan jumlah skala adalah 180 skala.

Pada thermometer jenis kelvin mempunyai titik beku 273°c, titik didih 373°c, dan jumlah skala adalah 100 skala.

Dari Gambar tersebut, diketahui bahwa 0°C = 32° F dan 100° C = 212° F, serta 100 skala Celsius = 180 skala Fahrenheit sehingga dapat dinyatakan persamaan sebagai berikut.

￼

Sehingga diperoleh hubungan antara skala Celcius dan skala Fahrenheit sebagai berikut.

￼

Dan juga, Dari Gambar diatas, telah diketahui bahwa titik tetap bawah skala Celsius dan skala Reamur adalah 0 °C dan 0 °R. Adapun titik tetap atas skala Celsius dan skala Reamur adalah 100° C dan 80 °R. Jadi, 100 skala Celsius = 80 skala Reamur. Sehingga dapat dinyatakan persamaan sebagai berikut.

￼

Sehingga diperoleh hubungan antara skala Celcius dan skala Reamur sebagai berikut:

￼

Contoh soal:

1. Di daerah pegunungan di pagi hari suhu mencapai 20°R , berapakah suhu tersebut jika dirubah menjadi celcius dan fahrenheit?

Penyelesaian :

Diketahui: Treamur : 20°R

Ditanya: T celcius dan T fahrenheit ?

Jawab:

TC = TC/TR x TR

= 5/4 x 20°

= 25°C

TF = TF/TR x TR + 32°

= 9/4 x 20° + 32°

= 45° + 32°

= 77° F

2. Air yang sedang dimasak selama 3 menit kemudian dicelupkan sebuah termometer yang kemudian menunjukkan angka 140 °F , apabila thermometer tersebut diganti dengan thermometer berskala celcius dan reamur, berapa angka yang ditunjukkan thermometer tersebut?

Penyelesaian :

Diketahui : TF = 140°F

Ditanya : TC dan TR?

Jawab:

TC = TC/TF x (TF-32°)

= 5/9 x (140°-32°)

= 5/9 x 108°

= 60°C

TR = TR/TF x (TF-32°)

= 4/9 x (140°-32°)

= 4/9 x 108°

= 48°R

31. Apa Arti Dari Apa arti kata dari Kaze wa Fuiteiru?


Bahasa Indonesianya dari kalimat kaze wa futeiru adalah angin bertiup.

Pembahasan

Kalimat kaze wa futeiru jika ditulis dalam bentuk kanjinya adalah 風は吹いている , sedangkan jika ditulis dalam bentuk huruf hiragananya adalah かぜはふいている

____________________________

Terjemahan kata-perkatanya :

風は吹いている

風 (kaze) = yang berarti angin. Dalam bentuk hiragananya adalah かぜ

Partikel は (wa) = yang berarti suatu partikel yang bertujuan agar subjek atau objeknya dijadikan topik utama pada kalimat.

吹いている (futeiru) = memiliki 2 arti yaitu hembusan, bertiup, atau tiupan. Dalam bentuk hiragananya ditulis sebagai ふいている

Sehingga arti dari 風は吹いている (kaze wa futeiru) bisa diartikan sebagai berikut :

Angin bertiup.Tiupan angin.Hembusan angin.
