4. fill the blank using "rather" or "not rather" to express preference Q :. ........... join sundanese class or Chinese class ? A : ........... join chinese class, because i cant write kanji


PART : STATING PREFERENCE - WOULD RATHER

--------------

Answer :

Q : Would you rather join sundanese class or chinese class?

A : I would rather not join chinese class, because i can't write kanji.

NOTE / DISCUSSION :

Would rather / 'd rather merupakan salah satu ungkapan perbandingan yang menunjukkan perasaan lebih suka terhadap sesuatu daripada yang lain. Untuk statement (Q) merupakan interrogative yang menyatakan pilihan perasaan lebih suka yang mana "kelas orang sunda" atau "kelas orang cina." Sedangkan statement (A) menunjukkan ungkapan kontras / negative , bisa diperhatikan pada cause/penyebab -nya "because i can't write kanji". Ini menunjukkan bahwa adanya perasaan tidak suka jika join di kelas orang cina dikarenakan tidak bisa menulis kanji. Penggunaan "would rather" harus diikuti dengan bare infinitive / base verb / verb 1 . Untuk statement (Q) memiliki struktur : Would+ Subject+rather + bare infinitive ? . Sedangkan untuk statement (A) memiliki struktur : Subject + would rather+not+ bare infinitive.

EXPLANATION :PREFERENCEUngkapan perbandingan perasaan lebih menyukai terhadap sesuatu/seseorang.Tiap preference memiliki konjungsi masing-masing yang menjadi pasangannya. Konjungsi ini sebagai perbandingan terhadap sesuatu/seseorang. So, harap dihafalkan konjungsi perbandingannya sehingga tidak salah penempatan pada masing-masing pereference.KINDS OF PREFERENCE :

Penggunaan "prefer"  ada 2 jenis yaitu dengan diikuti Gerund { the word "prefer" is one of common verbs should be followed by verb-ing  } dan TO-infinitive .

1. By using PREFER .........TO

PATTERN  :

S+prefer (s)+Gerund/noun +TO+Gerund/noun

Example :

1. I prefer writing to speaking ( by using Verb-ing )

{ saya lebih suka menulis daripada berbicara }

2. They prefer car to bus ( by using noun )

{ Mereka lebih suka mobil dari pada bis }

2. By using WOULD PREFER TO .............RATHER THAN

Penggunaan "prefer" dengan to-infinitive. " would prefer " bisa di kontraksi menjadi "  'd prefer ."

PATTERN :

(+) S+would prefer+to+verb 1+object+ rather than+verb1+object

(-) S+would prefer+not+to+verb1+object+rather than+to+verb 1+object

(?) Would+S+prefer+to+verb 1+object+rather than+to+verb1+object?

Example :

(+) He would prefer to eat fruit salad rather than (to) eat fried chicken.

(-)  He would prefer not to eat fruit salad rather than (to) eat fried chicken

(?) Would he prefer to eat fruit salad rather than (to) eat fried chicken? or we can say would he prefer to eat fruit salad or fried chicken?

3. By using  WOULD RATHER..............THAN

PATTERN :

(+) S+Would rather+verb1 + Object + than + verb1 + object

(-) S + would rather + not + verb 1 + object + than + verb 1 + object

(?) Would + subject + rather + verb1 + object + than + verb 1 + object?

Example :

(+) He would rather read some books than play outside.

(-)  He would rather not read some books than play outside

(?) Would he rather read some books than play outside? or we can say Would he rather read some books or play outside?

4. By using LIKE .............BETTER THAN

PATTERN :

Subject +like(s)+ noun/gerund+better than+noun/gerund.

Example :

1. I like eating the fruits better than (eating) the meat   ( by using verb-ing )

{ Aku lebih suka makan buah-buahan daripada daging }

2. I like an English grammar book better than a history book (by using noun/noun phrase)

{ Aku lebih suka buku grammar bahasa inggris daripada buku sejarah }

JAWAB:

PEMBAHASAN:

if conditional sentence merupakansebuah kalimat pengandaian yang digunakan untuk menunjukkan bahwa sesuatu benar-benar terjadi/nyata jika syarat yang lain terpenuhi. Kenapa harus begitu? Karena dalam pola kalimat if conditional sentence terdiri dari 2 klausa, yaitu main clause (kalimat utama) dan yang kedua sub clause (anak kalimat). Jadi kalimat If conditional sentence itu bisa benar-benar terjadi dan mungkin juga tidak akan pernah terjadi. Dan keduanya bisa saling bertukar tempat (dibolak balik). Dalam kata lain jika kalimat utama didepan, maka anak kalimatnya berada pada kalimat kedua (belakang) dan sebaliknya.

Ada tiga jenis kalimat if conditional:

1. If Conditional Tipe 1

If conditional tipe 1 yang juga disebut Future conditional sentence adalah sebuah kalimat pengandaian yang belum terjadi dan kemungkinan besar akan terjadi dimasa mendatang (future) jika syaratnya (anak kalimatnya) terpenuhi.

Contoh:  

If I go to Palembang I will meet Sarah  

Rumus/Pola kalimat:

If + S + V1 (simple present), S + will + Verb 1

2. If Conditional Tipe 2

If conditional tipe 2 yang juga disebut present conditional sentence adalah sebuah kalimat pengandaian yang digunakan untuk mengandaikan sesuatu yang tidak akan pernah terjadi karena syarat sub clausenya tidak mungkin terpenuhi, dan apa yang diandaikan bertolak belakang dengan kenyataan yang ada.

Contoh:  

If I were you, I would forgive him

Rumus/ Pola kalimat:

If + S + V2 (simple past), S + would + Verb 1

3. If Conditional Tipe 3

If conditional tipe 3 yang juga disebut past conditional sentence adalah sebuah kalimat pengandaian yang digunakan untuk mengandaikan sesuatu yang tidak akan pernah terjadi karena syarat sub clausenya tidak akan pernah terpenuhi, karena apa yang diandaikan sudah berlalu di masa lampau (masanya sudah terlewati).

Contoh :  

If Mimi had done her homework, she would have not got a punishment.

Rumus/ Pola kalimat:

If + S + V3 ( past perfect), S + would have + Verb3
