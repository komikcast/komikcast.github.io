Jawaban

Kore Wa Watashi No Enpitsu Desu

[ これは私の鉛筆です ]

Arti :

Ini Adalah Pensil Saya

Kosakata :

これ : Iniは : Partikel yang menunjukkan Subjek 私 : Sayaの : Partikel鉛筆 : Pensilです : Kalimat Yang Membantu Kalimat Lain Agar Lebih Sopan

Rumus :

Kore ( Kore / Sore / Are ) + Wa + Kalimat ( Orang ) +No ( の ) + Desu

Contoh kalimat :

これはマンゴです

[ Kore wa Mango Desu ]

Arti :

Ini Adalah Mangga

Kosakata :

これ : Iniは : Partikel マンゴ : Mangga です: Kalimat Yang Membantu Kalimat Lain Agar Lebih Sopan

Kore / Sore / Are

Kore :

Penulisan dalam Hiragana Adalah これ , Yang dalam Bahasa Indonesia Memiliki arti yaitu Ini ( Biasanya Ada Dalam Pembicaraan )

————

Sore :

Penulisan dalam Hiragana adalah それ , Yang dalam Bahasa Indonesia Memiliki arti Yaitu Itu ( Dekat )

Contoh Kalimat :

それは私の本です

( Sore wa Hon desu )

Arti :

Itu ( Dekat ) Adalah Buku ( Milik ) Saya

Kosakata :

それ : Itu ( Dekat )は : Partikel私 : Sayaの : Partikel本(ほん) : Bukuです : Desu

————

Are :

Penulisan dalam Hiragana Adalah あれ , Yang Dalam Bahasa Indonesia Memiliki arti Yaitu Itu ( Jauh )

Contoh kalimat :

あれはアニメです

[ Are Wa Anime Desu ]

Arti :

Itu ( Jauh ) Adalah Anime

Kosakata :

あれ : Itu ( Jauh )は : Partikelアニメ : Animeです : Desu

20. Apa arti dari watashi wa anata ni watashi no boifurendo o aishimasu ?


BAHASA JEPANG

Pembahasan :
Watashi wa anata ni watashi no boifurendo o aishimasu.

Jika kalimat seperti itu berarti dari sudut pandang Cewek.

Artinya :
"Bagi saya, kamu adalah Pacar saya yang saya cintai"

Tulisan Jepangnya :
" 私 は あなたに 私 の 簿いフレンド を あいします。"

Arti Per Kata :
Watashi = 私 = Saya (Formal)
wa = は (kata bantu)
Anata = あなた = Kamu (Formal)
Ni = に (Kata Bantu)
No = の (Kata Bantu)
Boifurendo = 簿いフレンド = Pacar Laki laki
O = を (Kata Bantu)
Aishimasu = あいします (Asal kata dari Aishiteru diubah kedalam bentuk Masu) = MencintaiWatashi wa anata ni watashi no boifurendo o aishimasu

Terjemahannya adalah:
Bagiku, kamu adalah seorang pacar yang kucintai

Note:
Kalimat ini diucapkan oleh seorang perempuan kepada laki laki yang dicintainya. Pernyataan ini dapat diketahui karena terdapat kata "boifurendo" yang artinya "teman laki laki / pacar"
