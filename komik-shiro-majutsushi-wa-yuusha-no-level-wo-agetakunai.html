Pembahasan

Partikel bentuk ます (Masu) adalah partikel yang berguna untuk memberikan keterangan kerja pada suatu kalimat (verb). Untuk menanyakan sesuatu yang dilakukan seseorang, maka tinggal menambahkan partikel (か) menjadi ますか (Masuka). Namun, harus diperhatikan bahwa partikel ini hanya digunakan pada pekerjaan yang sedang dilakukan atau akan dilakukan. Example :

______________________________

Ayu-san wamai asa nan jini asa gohan o tabemasuka berarti Ayu sarapan setiap pagi jam berapa?

あゆさん (Ayu-san) merupakan subjek dari kalimat tersebut yang diberi tambahan -san sebagai bentuk pengucapan yang hormat kepada seseorang. は (wa) adalah partikel penanda subjek. 毎 (Mai) berarti setiap dan 朝 (asa) berarti pagi, jadi artinya menjadi setiap pagi.何時に (nanjini) berarti jam berapa. ごはん (Gohan) berarti nasi namun karena latarnya dipagi hari maka lebih merujuk ke sarapan. 食べ (tabe) bentuk dasar dari taberu yang artinya makan. ますか adalah partikel pemberi pertanyaan pada suatu kerja. So, artinya adalah Ayu sarapan setiap pagi jam berapa?.

12. Watashi .......badominton .......dekimasu Wa,wo No,ga Wa,ga Ga,wa Wa,no


JAWABAN:

wa,ga

PENJELASAN:

私はバドミントンが出来ます。

わたしはばどみんとんができます。

Watashi wa badminton ga dekimasu.

Saya bisa (bermain) badminton.

文法(bunpou):

Tata bahasa:

Subjek + は + Objek + が 出来ます

Subjek+wa+objek+ga dekimasu

Saya bisa ~

Arti per kata:

1. 私(watashi): saya

digunakan oleh laki laki dan perempuan

2. は(ha, tapi disini dibuat partikel jadi cara membacanya menjadi wa): partikel yg menunjukkan topik suatu kalimat

3. バドミントン(badominton): Badminton

4. が(ga):

♧partikel yg menandai subjek,

♧bisa juga diartikan "tapi" di bunpou(kalimat+ga+kalimat ),

♧menyatakan kejadian alam yg dirasakan olh panca indra,

♧が(ga) juga sering diikuti beberapa kata kerja , antara lain:

a.出来ます(dekimasu): bisa, dapat

b. 上手(jouzu): pandai (dalam..)

c. 下手(heta): kurang pandai (dalam...)

d. 好き(suki) : suka

e. 嫌い(kirai): benci

f. 欲しい(hoshii): ingin ( suatu benda/barang)

g. 有ります(arimasu ): ada, benda mati(termasuk tumbuhan)

h. 居ます(imasu): ada, benda hidup(manusia dan hewan saja)

i. 一番(ichiban): (yg) paling...

5. 出来(deki): bisa, dapat

6. ます(masu): membuat kalimat menjadi formal, terkesan lbh halus

28. Waga na wa Megumin! Ākuwizādo wo nariwai toshi, saikyou no kougeki no mahou "bakuretsu mahou" wo ayatsuru mono!


Aku sendiri adalah Megumin! Seorang Arch Wizard, yang mempunyai serangan sihir terkuat yaitu "sihir ledakan" yang dapat memanipulasi.

我がなはメグミン! アークウィザードを 生業 とし , さいきょういの攻撃の魔法 "爆烈魔法" を操るモノ!

• • • • • • • • • • • • • • • • • • • • • • • • • • • • • • • • • • • •

Pembahasan

Kita akan bahas bersama sama tentang grammarnya.

Waga na (我がな) adalah frasa non formal yang berarti aku sendiri. Nah, Megumin harus ditulis dengan katakana menjadi メグミン. Partikel は digunakan sebagai penanda subjeknya yaitu Megumin.

Nariwai (kanji : 生業) merupakan frasa yang berarti mata pencaharian lalu toshi (とし) kata singkat dari tosh|te (として) yang artinya sebagai. Jadi, artinya menjadi ber mata pencaharian/bekerja sebagai... . Namun, karena ini didunia fiktif anime maka artinya menjadi "seorang" (sudah melambangkan pekerjaan). Arch Wizard juga merupakan kata serapan sehingga ditulis katakana menjadi アークウィザード .

Saikyou ( さいきょうい) merupakan frasa yang artinya pemulihan, tertinggi, terkuat. Partikel の (no) sebagai partikel kepemilikan. Kougeki (kanji : 攻撃) berarti penyerangan, serangan. Mahou (kanji : 魔法) berarti sihir. Jadi, arti saikyou no kougeki no mahou adalah mempunyai serangan sihir terkuat.

Bakuretsu Mahou (kanji : 爆烈魔法) berarti sihir ledakan dimana bakutetsu = ledakan dan mahou = sihir. Ayatsuru (操る) adalah kata jamak dari 操 (ayatsu) yang artinya manipulasi sehingga artinya menjadi memanipulasi. Mo (も) adalah partikel yang bersifat mempertegas. Jadi, artinya adalah "sihir ledakan" yang memanipulasi!

Kesimpulan

Waga na wa Megumin + Akuwizado wo nariwai toshi + saikyou no kougeki no mahou + bakuretsu mahou + ayatsuru

= Aku sendiri Megumin. Seorang Arch Wizard yang mempunyai serangan sihir terkuat yaitu sihir ledakan yang dapat memanipulasi

35. Bagaimana penggunaan partikel wa, wo/o, ni, no, mo, ga, dan ka dalam bahasa jepang?


Partikel bahasa Jepang.

Wa : kata bantu topik atau subjek.
O : kata bantu yang dikenakan tindakan atau objek yang dilakukan.
Ni : kata bantu menyatakan suatu keadaan disuatu tempat.
Mo : kata yang menunjukkan suatu hal sama dengan hal lain.
Ga : seperti kata bantu -wa- namun lebih menekan pada predikatnya.
Ka : kata bantu tanya yang diletakkan diakhir kalimat.

Ini menurut yang saya pelajari.

#goodluck1)Wa 「は」adalah partikel yang menyatakan bahwa A adalah B. atau bisa juga untuk menerangkan subjek adalah objek/ kata sifat. selalu diikuti 'desu' 「です」
contoh : Watashi wa Ahelva desu. 「わたしはAhelvaです」。
Artinya : Saya (A) adalah Aheva (B)
2)Wo (dibaca o)　「を」adalah partikel yang menghubungkan kata BENDA dengan kata KERJA. partikel ini menerangkan apa yang terjadi terhadap kata benda tsb.
contoh : Gohan o tabemasu. 「ごはんをたべます」。
Artinya : Makan nasi. Gohan = nasi, tabemasu = makan.
3) Ni 「に」adalah partikel yang menerangkan waktu dan tempat. "Ni" juga bisa berarti dari/ kepada. Arti "Ni" adalah di, pada.
Contoh : 1. Waktu, shigatsu ni kuni e kaerimasu「しがつにくにへかえります」。
Artinya : Bulan 4 pulang ke negara,
Penjelasan : Shigatsu = bulan April/ bulan 4, kuni = negara, e = partikel yg mempunyai arti "ke", kaerimasu = pulang/ kembali
2. Tempat
Isu no ue ni kaban ga arimasu. 「いすのうえにかばんがあります」。
Artinya : Di atas kursi ada tas. 
Penjelasan : Isu = kursi, no = partikel kepunyaan, ue = atas, kaban = tas, ga = partikel keterangan, arimasu = ada.
3. Kepada
Haha ni okashi o agemasu.　「ははにおかしをあげます」。
Artinya : Memberikan kue kepada ibu
Penjelasan : Haha = ibu,  okashi = kue, agemasu = memberikan

4. No 「の」adalah partikel kepunyaan. partikel ini juga bisa menyatakan bahwa A adalah milik B. Partikel ini juga menghubungkan kata benda dengan kata benda.
Contoh : Watashi no fudebako 「わたしのふでばこ」。
Artinya : Kotak pensil saya.
Penjelasan : Watashi = Saya, Fudebako = kotak pensil.
Contoh lain : Hon no kami 「ほんのかみ」。
Artinya : Kertas buku.
Penjelasan : Hon = buku, Kami = kertas.
5. Mo 「も」Adalah partikel yang memiliki arti Juga, biasanya menghubungkan keadaan positif dengan keadaan positif
Contoh :Chichi wa raamen ga suki desu. watashi mo raamen ga suki desu. 「ちちはラーメンがすきです。わたしもラーメンがすきです」。
Artinya : Ayah suka ramen. saya juga suka ramen
Penjelasan : Chichi = ayah, raamen = ramen, suki = suka.
6. Ga 「が」Adalah partikel yang menyatakan penegasan, bisa juga digunakan sebelum kata sifat, partikel ini menyatakan keadaan yang berbeda. memiliki arti tetapi.
Contoh : Nihongo wa omoshiroi desu ga, muzukashii desu. 「にほんごはおもしろいですが、むずかしいです」。
Artinya : Bahasa jepang itu menarik, tetapi susah.
Penjelasan : Nihongo = bahasa Jepang, omoshiroi = menarik, muzukashii = susah/sulit.
7. Ka 「か」adalah partikel tanya. digunakan untuk menanyakan sesuatu. terletak di akhir kalimat.
Contoh : Anata wa dare desu ka? 「あなたはだれですか。」。
Artinya : Kamu siapa ?
Penjelasan : Anata = kamu, Dare = siapa.


Sebagai bonus
8. He (dibaca e ) 「へ」adalah partikel yang memiliki arti Ke.
Contoh : Kaisha e ikimasu. 　「かいしゃへいきます」。
Artinya : Pergi ke perusahaan.
Penjelasan : Kaisha = Perusahaan , ikimasu = pergi.

Semoga membantu. kalau ada partikel lain yg ingin ditanyakan, jangan sungkan bertanya. ^^

37. shiroi (白い) artinya?


Jawaban:

白い = しろい = shiroi = putih

Penjelasan:

semoga membantu

白い (Shiroi) artinya adalah "Putih".

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Pendahuluan :

Kata sifat dalam Bahasa Jepang disebut dengan 形容詞 (Keiyoushi). Kata sifat Bahasa Jepang dibagi menjadi kata sifat-i dan kata sifat-na. Perbedaannya yaitu kata sifat-i menggunakan tambahan huruf i untuk menerangkan kata benda, sedangkan kata sifat-na menggunakan tambahan huruf na. Namun terdapat beberapa kata sifat yang menggunakan tambahan huruf i, tapi termasuk ke dalam kata sifat-na. Salah satu contohnya yaitu Kirai (Benci).

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Pembahasan :

白い (Shiroi) memiliki arti "Putih", dan kata ini termasuk ke dalam kelompok kata sifat-i. Kata sifat-i merupakan kelompok kata sifat yang terdapat tambahan huruf i di akhir kata nya. Contohnya seperti kata Shiroi, yang awalnya hanya Shiro kemudian ditambahkan huruf i. Menambahkan huruf i di akhir kata sifat-i ini fungsinya adalah untuk menerangkan kata benda nya. Contoh kalimatnya sebagai berikut.

白い鞄 (Shiroi kaban) = Tas putih白い服 (Shiroi fuku) = Baju putih白い髪 (Shiroi kami) = Rambut putih

Beberapa kata sifat-i jika tidak terdapat tambahan huruf i nya, akan menjadi kata benda. Contohnya seperti kata Shiroi ini. Shiroi merupakan kata sifat, dan Shiro merupakan kata benda. Contoh kalimat yang menggunakan kata Shiro sebagai berikut.

私の鞄は白 (Watashi no kaban wa shiro) = Tas saya putih私の服は白 (Watashi no fuku wa shiro) = Baju saya putih私の髪は白 (Watashi no kami wa shiro) = Rambut saya putih
