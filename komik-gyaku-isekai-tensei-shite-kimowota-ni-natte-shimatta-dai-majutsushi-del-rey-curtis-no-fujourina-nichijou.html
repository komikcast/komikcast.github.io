Jawaban:

Materi : EksponenKelas : 9Kode soal ; 2Kode kategorisasi ; 9.2.1

Kata kunci : Eksponen

==============

[tex]\it \frac{{rey}^{69} \div {rey}^{63} \times {rey}^{2} }{ {rey}^{6} } \\ \frac{rey {}^{69 - 63 + 2} }{rey {}^{6} } \\ \frac{rey {}^{8} }{rey {}^{6} } = rey⁸-⁶ = rey²✔[/tex]

⏩ Penyelesaian soal

= rey⁶⁹ ÷ r⁶³ × rey² ÷ rey⁶

= rey^( ⁶⁹-⁶³+²-⁶)

= rey²

Jadi, hasil dari rey⁶⁹ ÷ rey⁶³ × rey² ÷ rey⁶ = rey²
