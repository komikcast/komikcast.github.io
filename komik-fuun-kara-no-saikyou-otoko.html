3. Waga na wa Megumin! Ākuwizādo wo nariwai toshi, saikyou no kougeki no mahou "bakuretsu mahou" wo ayatsuru mono!


Aku sendiri adalah Megumin! Seorang Arch Wizard, yang mempunyai serangan sihir terkuat yaitu "sihir ledakan" yang dapat memanipulasi.

我がなはメグミン! アークウィザードを 生業 とし , さいきょういの攻撃の魔法 "爆烈魔法" を操るモノ!

• • • • • • • • • • • • • • • • • • • • • • • • • • • • • • • • • • • •

Pembahasan

Kita akan bahas bersama sama tentang grammarnya.

Waga na (我がな) adalah frasa non formal yang berarti aku sendiri. Nah, Megumin harus ditulis dengan katakana menjadi メグミン. Partikel は digunakan sebagai penanda subjeknya yaitu Megumin.

Nariwai (kanji : 生業) merupakan frasa yang berarti mata pencaharian lalu toshi (とし) kata singkat dari tosh|te (として) yang artinya sebagai. Jadi, artinya menjadi ber mata pencaharian/bekerja sebagai... . Namun, karena ini didunia fiktif anime maka artinya menjadi "seorang" (sudah melambangkan pekerjaan). Arch Wizard juga merupakan kata serapan sehingga ditulis katakana menjadi アークウィザード .

Saikyou ( さいきょうい) merupakan frasa yang artinya pemulihan, tertinggi, terkuat. Partikel の (no) sebagai partikel kepemilikan. Kougeki (kanji : 攻撃) berarti penyerangan, serangan. Mahou (kanji : 魔法) berarti sihir. Jadi, arti saikyou no kougeki no mahou adalah mempunyai serangan sihir terkuat.

Bakuretsu Mahou (kanji : 爆烈魔法) berarti sihir ledakan dimana bakutetsu = ledakan dan mahou = sihir. Ayatsuru (操る) adalah kata jamak dari 操 (ayatsu) yang artinya manipulasi sehingga artinya menjadi memanipulasi. Mo (も) adalah partikel yang bersifat mempertegas. Jadi, artinya adalah "sihir ledakan" yang memanipulasi!

Kesimpulan

Waga na wa Megumin + Akuwizado wo nariwai toshi + saikyou no kougeki no mahou + bakuretsu mahou + ayatsuru

= Aku sendiri Megumin. Seorang Arch Wizard yang mempunyai serangan sihir terkuat yaitu sihir ledakan yang dapat memanipulasi

7. anatawa otoko desuka artinya apa??​

Pembahasan

あなたは男ですか？

Anata wa otoko desuka?

Apakah kamu seorang laki-laki?

_____

あなた (anata)

Berarti 'kamu' atau 'anda'. Bentuk kata ini tergolong jenis formal atau bisa juga digunakan dalam situasi yang kasual (asalkan lebih baik jangan diucapkan ketika dalam situasi yang sudah sangat akrab) dan kata anata ini termasuk golongan kata ganti orang kedua tunggal.

は (wa)

Biasanya partikel ini terletak di dekat sebelah subjek atau tidak. Dalam kalimat ini, terdapatnya kata ganti atau subjek あなた (anata) kemudian ditambahkan dengan partikel は (wa) setelahnya, sehingga fungsi dari partikel は (wa) ini adalah menjadi penanda untuk subjek.

男 (otoko)

Jika dalam versi hiragana kata otoko ini ditulis dengan おとこ

Dan artinya itu berarti 'lelaki', 'pria', atau 'laki-laki'. Penggunaan kata 男 (otoko) ini bisa digunakan kepada laki-laki yang merupakan anak-anak, pemuda, orang dewasa, maupun orang tua.

ですか (desuka)

Kata tanya ini memiliki kesan yang sopan/formal, termasuk sopan/formal karena terdapatnya salah satu partikel tambahan yaitu です (desu), selain itu ini jenis partikel yang mungkin jika dalam bahasa Indonesia yaitu apakah. Namun kata tanya ini juga biasanya tidak memiliki arti dan berperan menjadikan kalimat tersebut sebagai kalimat tanya.

Arti dari 'Anata wa otoko desuka' adalah "Apakah anda laki-laki?"

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Pembahasan :

Pola Kalimat

== [Subjek] wa [Predikat] desuka

== Apakah [Subjek] (adalah) [Predikat]?

Fungsi dari pola kalimat ini adalah untuk menanyakan suatu hal, apakah si subjek merupakan 'predikat'.

- - -

Untuk pembahasan lebih lanjutnya, akan saya bahas masing-masing kata yang digunakan untuk membuat kalimat di soal.

-

1. 貴方 / あなた / Anata

Berarti "Anda" atau "Kamu". Merupakan salah satu kata ganti orang kedua dalam Bahasa Jepang. Kata ini termasuk sopan dan umum digunakan. Tapi jika sudah mengenal lawan bicara, tidak baik untuk menggunakan kata ini. Bisa juga berarti panggilan "Sayang", "Beb", dsb. Biasa digunakan perempuan untuk memanggil pacar atau pasangan laki-laki nya.

-

2. は / Wa

Tidak ada arti, namun kadang bisa diartikan "Adalah". Kata ini merupakan partikel atau kata bantu kalimat, yang fungsinya untuk menandakan subjek atau topik dari suatu kalimat tersebut. Partikel ini dibaca Wa, tapi penulisannya dalam Hiragana menggunakan Ha.

-

3. 男 / おとこ / Otoko

Berarti "Laki-laki" atau "Pria". Kata ini dari segi kedewasaan merujuk pada pria dewasa, sedangkan dari segi usia bisa digunakan untuk laki-laki muda maupun tua. Kata ini termasuk sopan, tetapi lebih sopan lagi jika menggunakan 'Dansei'. Tapi kata Dansei tidak bisa digunakan untuk anak laki-laki.

-

4. です / Desu

Tidak ada arti. Merupakan akhiran kalimat yang berfungsi untuk memperhalus dan mempertegas suatu kalimat tersebut. Kata ini diletakkan setelah predikat, tapi tidak bisa setelah kata kerja. Untuk kata kerja, dengan mengubahnya ke bentuk Masu. Adapun kata serupa yang sering dianggap sebagai 'bentuk kasual' dari Desu, yaitu 'Da'. Kata Da ini hanya mempertegas kalimat saja, tidak memperhalus.

Komik Detektif Kindaichi No. 17 berjudul "Pembunuhan Dewa Pencuri" (The God Thief Murder Case) mengikuti petualangan detektif remaja Hajime Kindaichi yang harus memecahkan misteri pembunuhan di sebuah museum seni.

Ketika sebuah patung bernama Dewa Pencuri dicuri dari museum, seorang karyawan bernama Ishida tewas terbunuh. Kindaichi, bersama dengan sahabatnya Miyuki Nanase dan Inspektur Kenmochi, mulai menyelidiki kasus ini.

Mereka menemukan bahwa ada beberapa orang yang memiliki motif untuk membunuh Ishida, termasuk mantan pacarnya yang gelap mata, seorang kolektor seni yang ingin memperoleh Dewa Pencuri, dan seorang penjahat yang ingin memanfaatkan kekacauan di museum untuk mencuri barang berharga lainnya.

Setelah serangkaian petunjuk dan kesaksian terungkap, Kindaichi menemukan pelaku sebenarnya adalah Ishida yang sebenarnya adalah seorang penjahat profesional yang bekerja sama dengan orang dalam untuk mencuri barang berharga dari museum. Namun, Ishida mengkhianati rekannya dan mencuri Dewa Pencuri untuk dirinya sendiri, sehingga dia dibunuh oleh orang dalam itu.

Dalam akhir yang menegangkan, Kindaichi berhasil mengungkapkan identitas pelaku dan memecahkan misteri di balik pembunuhan dan pencurian patung berharga tersebut.

Cerita komik detektif Kindaichi no 17 (Pembunuhan Dewa Pencuri) dimulai dengan legenda tentang harta karun di gua di pulau terpencil di Jepang. Seorang miliarder mengundang Kindaichi dan teman-temannya untuk mencarinya. Namun, ketika tiba di pulau tersebut, ada satu orang yang tewas dibunuh.

Akhirnya, Kindaichi mengungkapkan bahwa si pembunuh adalah salah satu dari tim yang diundang oleh Hoshikawa. Setelah memecahkan misteri dan mengumpulkan bukti, si pembunuh ditangkap oleh polisi.
