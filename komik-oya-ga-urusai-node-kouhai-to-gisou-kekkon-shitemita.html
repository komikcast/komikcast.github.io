2. Apa Arti Dari KEKKON OMEDETEO ​


Arti 'kekkon omedetou' artinya berarti selamat menikah.

Pembahasan

けっこん おめでとう (kekkon omedetou) adalah versi yang diucapkan secara informal, sedangkan untuk pengucapannya secara formal adalah けっこん おめでとう ございます (kekkon omedetou gozaimasu) yang artinya berarti :

Selamat menikah.Selamat atas pernikahanmu.Selamat sudah menikah.

___________

Berikut ini untuk terjemahan dan penjelasan setiap kata-perkatanya yaitu :

Kekkon「結婚 • けっこん」

Untuk artinya dapat diartikan yaitu menikah, pernikahan, perkawinan, atau nikah.

Omedetou gozaimasu 「おめでとう ございます」

Untuk arti おめでとう (omedetou) ini yaitu selamat, sehingga kata ini cocok digunakan sebagai bentuk ungkapan menyatakan selamat atas sesuatu.

Sedangkan kata ございます (gozaimasu) biasanya terdapat dalam ungkapan-ungkapan aisatsu yang tidak memiliki arti, namun dapat berfungsi agar kalimat terdengar secara halus/sopan.

13. Contoh Program Java DLinkedList tambahkan beberapa method Pada Kelas DLinkedList dengan Fungsi” berikut: 1.menambah node baru di depan list 2.menambah node baru setelah node tertentu 3.menambah node baru sebelum node tertentu 4.menghapus node pertama dalam list 5.menghapus node terakhir dalam list


public class DLinkedList {

private Node head;

private Node tail;

private int size;

private class Node {

int data;

Node next;

Node prev;

Node(int data) {

this.data = data;

}

}

public void addFirst(int data) {

Node node = new Node(data);

if (head == null) {

head = tail = node;

head.prev = null;

tail.next = null;

} else {

head.prev = node;

node.next = head;

node.prev = null;

head = node;

}

size++;

}

public void addAfter(Node node, int data) {

if (node == null) {

System.out.println("Node tidak dapat null");

return;

}

Node newNode = new Node(data);

newNode.next = node.next;

node.next = newNode;

newNode.prev = node;

if (newNode.next != null)

newNode.next.prev = newNode;

size++;

}

public void addBefore(Node node, int data) {

if (node == null) {

System.out.println("Node tidak dapat null");

return;

}

Node newNode = new Node(data);

newNode.prev = node.prev;

node.prev = newNode;

newNode.next = node;

if (newNode.prev != null)

newNode.prev.next = newNode;

size++;

}

public void removeFirst() {

if (head == null) {

System.out.println("List kosong");

return;

}

if (head == tail) {

head = tail = null;

size--;

return;

}

head = head.next;

head.prev = null;

size--;

}

public void removeLast() {

if (tail == null) {

System.out.println("List kosong");

return;

}

if (head == tail) {

head = tail = null;

size--;

return;

}

tail = tail.prev;

tail.next = null;

size--;

}

}

Metode `addFirst` menambahkan simpul baru di depan daftar. Metode `addAfter` menambahkan simpul baru setelah simpul tertentu. Metode `addBefore` menambahkan simpul baru sebelum simpul tertentu. Metode `removeFirst` menghapus simpul pertama dalam daftar. Dan metode `removeLast` menghapus simpul terakhir dalam daftar.

15. Topologi yang menghubungkan satu node ke node berikutnya dan node terakhir yang terhubung ke node awal merupakan pengertian dari topologi.


Topologi yang menghubungkan satu node ke node berikutnya dan node terakhir yang terhubung ke node awal adalah topologi Ring. Pada topologi Ring, setiap node terhubung dengan dua tetangga yang berdekatan dan data bergerak dalam satu arah sepanjang cincin atau lingkaran. Keuntungan dari topologi Ring adalah mudah dalam mengelola lalu lintas data dan dapat mengatasi kerusakan pada satu node, namun kelemahannya adalah jika ada dua node yang rusak dapat membuat seluruh jaringan terganggu. Topologi Ring umumnya digunakan pada jaringan LAN (Local Area Network) dengan jumlah node yang relatif sedikit.

36. つ ini huruf apa ya? tsu?kok di kata けっこん(kekkon)jadi "k" ? bukannya tsu?​


Huruf hiragana つ ini memiliki 2 fungsi yaitu diantaranya sebagai huruf yang dibaca tsu, tetapi juga berfungsi sebagai huruf yang menggandakan konsonan.

Pembahasan

Hiragana つ (tsu)

Berikut penjelasan mengenai kedua fungsi dari huruf hiragana つ :

1. Berfungsi sebagai 促音 • そくおん (sokuon), maksud 'sokuon' adalah golongan huruf Jepang yang dapat menggandakan huruf konsonan. Contohnya :

けっこん (kekkon) : menikah, pernikahan.まって (matte) : tunggu, sebentar.がっこう (gakkou) : sekolah.しっぱい (shippai) : gagal.

Umumnya dalam bentuk sokuon, huruf つ menjadi kecil dengan bentuk っ

Huruf-huruf Jepang yang tergolong sokuon adalah huruf hiragana つ dan huruf katakana ツ (tsu).

2. Seperti biasa, dibaca sebagai huruf tsu

Seperti huruf hiragana lainnya, huruf hiragana つ dibaca sebagai tsu. Contohnya :

ついて (tsuite) : tentang.つき (tsuki) : bulan.きせつ (kisetsu) : musim.けいさつ (keisatsu) : polisi.

38. struct node { int key; struct node *left; struct node *right; } *root; bool search(struct node *curr, int find){ //COMPLETE THE CODE! } int heightBinaryTree(struct node *node){ //COMPLETE THE CODE! }


Jawaban:

bool search(struct node *curr, int find){

   if (curr == NULL) { // base case: not found

       return false;

   }

   if (curr->key == find) { // base case: found

       return true;

   }

   if (find < curr->key) { // search in the left subtree

       return search(curr->left, find);

   } else { // search in the right subtree

       return search(curr->right, find);

   }

}

int heightBinaryTree(struct node *node){

   if (node == NULL) { // base case: empty tree

       return 0;

   }

   int leftHeight = heightBinaryTree(node->left); // height of left subtree

   int rightHeight = heightBinaryTree(node->right); // height of right subtree

   return 1 + max(leftHeight, rightHeight); // add 1 to the maximum height of the subtrees

}

Penjelasan:

Note that the search function performs a binary search on a binary search tree, where the key member variable is used to compare nodes. The heightBinaryTree function calculates the height of a binary tree recursively, by finding the height of the left and right subtrees and adding 1 to the maximum of those heights.
