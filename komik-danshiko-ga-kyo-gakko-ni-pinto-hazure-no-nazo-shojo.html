3. arti watashiwa getsuyoubi ni gakko e ikimasu​


Arti dari 私は月曜日に学校へ行きます

( watashi wa getsuyoubi ni gakko e ikimasu )

Saya pergi ke sekolah pada hari Senin

Pembahasan

Pada kalimat tersebut memiliki Ikimasu yang berarti pergi , berarti sudah terlihat arti dari kalimat tersebut yang memiliki arti berpergian

私は月曜日に学校へ行きます

( watashi wa getsuyoubi ni gakko e ikimasu )

Saya pergi ke sekolah pada hari Senin

Terjemah Per-Kata;

私 ( Watashi ) → Memiliki arti yaitu "Saya"

は ( Wa ) → Merupakan Partikel yang menunjukkan Subjek

月曜日 ( Getsuyoubi ) → Memiliki arti yaitu "Hari Senin"

に ( Ni ) → Merupakan Partikel yang menunjukkan suatu alasan

学校 ( Gakkou ) → Memiliki arti yaitu Sekolah

へ ( E ) → Merupakan Partikel

行きま ( Ikimasu ) → Memiliki arti yaitu "Pergi''

___________________________

Tidak hanya gakkou , berikut merupakan beberapa kalimat yang berhubungan dengan gakkou ( Sekolah);

学校 [ がっこう ]

[ Gakkou ] → Sekolah

幼稚園 [ ようちえん ]

[ Yoochien ] → Taman Kanak-kanak

小学校 [ しょうがっこう ]

[ Shougakkou ] → Sekolah Dasar

中学生 [ ちゅうがっこう ]

[ Chuugakkou ] → Sekolah menengah pertama

高校 [ こうこう ]

[ Kookoo ] → Sekolah menengah Atas

大学 [ だいがく ]

[ Daigaku ] → Universitas

幼稚園児 [ ようちえんじ ]

[ Youchienji ] → Anak TK

小学生 [ しょうがくせい ]

[ Shougakusee ] → Siswa Sekolah Dasar

駐学生 [ ちゅうがくせい ]

[ Chuugakusee ] → Siswa Sekolah menengah Pertama

高校生 [ こうこうがくせい ]

Pembahasan

Lirik lagu yang tertera dalam soal merupakan lirik lagu pembuka (opening) dari sebuah anime atau kartun Jepang berjudul Yarichin (ヤリチン☆ビッチ部). Anime ini sendiri dibuat berdasarkan sebuah manga berjudul sama yang ditulis oleh Ogeretsu Tanaka. Saat ini, manga Yarichin yang mulai beredar pada tanggal 10 Juni 2012 sudah memiliki empat buah volume.

Anime Yarichin sendiri dirilis pada 21 September 2018 dan sejauh ini hanya memiliki dua episode.
