Dialog diatas adalah contoh dialog conditional sentence type 1.

Conditional sentence type 1, menjelaskan pengandaian di masa mendatang yang belum/mungkin terjadi.

Rumusnya adalah:

If + subject + verb 1 + object, subject + will + verb 1 + object

Pada  conditional sentence type 1, kita bisa menggunakan modals sebagai klausa utama selain future tense untuk menyatakan degree of certainty, permission, or a recommendation about the outcome.

Contoh:

If you drop that glass, it will breakIf you don't enter the house now, you will be locked If you want to pass the exam, you should study moreYou can come over if you have time
