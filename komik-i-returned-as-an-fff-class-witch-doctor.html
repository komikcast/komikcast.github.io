26. It is called nurse. Nurse often called as co-doctor as they help the doctor curing the patient. arti dalam bahasa Indonesia​


Jawaban:

Translations:

Itu disebut suster / perawat. Suster / Perawat sering disebut sebagai asisten dokter karena mereka membantu dokter menyembuhkan pasien.

Penjelasan:

Jawaban dicetak tebal dan merupakan materi pembelajaran Translations (terjemahan) dari teks dalam Bahasa Inggris.

Kalimat dalam teks pada soal menggunakan bentuk kalimat Simple Present Tense karena menjelaskan fakta dalam Bahasa Inggris.

Simple Present Tense:

Nominal: (+) Subject + Tobe (is/am/are) + Adjective / Noun / Adverb

Verbal: (+) Subject + Verb-1 S/ES + Object

Passive: (+) Subject + Tobe (is/am/are) + Adverb (often) + Verb-3 + Object

Syifa menanyakan apakah Linda ngantuk saat pelajaran, ternyata Linda tidak ngantuk melainkan sakit perut, jadi Syifa menawarkan Linda untuk menemaninya, Jika dialog diterjemahkan:

Syifia: Kenapa kamu mengantuk saat pelajaran?

b.You could see the doctor  = kamu bisa menemui dokter

с.You should see the doctor  = Kamu perlu menemui dokter

d.You can see the doctor  = Kamu bisa menemui dokter

e.She should see the doctor  = Dia perlu menemui dokter
