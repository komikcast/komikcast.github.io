7. Tentukan panjang AB

Penyelesaian :

Perhatikan gambar yang ada pada lampiran II

Kita tarik garis merah DH yang sejajar dengan garis CB, agar terbentuk segitiga.

Cara I

Mencari panjang AH dan AB

Panjang AB = AH + HB

                   = 6 cm + 8 cm

                   = 14 cm

Jadi panjang AB adalah 14 cm

Cara II

Menentukan panjang AB

[tex]EF = \frac{(DE\times AB) ~ + ~(AE\times DC)}{AE~ + ~ED} \\ 9,8 = \frac{(3\times AB) ~ + ~(7\times 8)}{7~ + ~3} \\ 9,8 = \frac{3AB ~+ ~56}{10} \\ 9,8 \times 10 = 3AB+56 \\ 98 = 3AB +56 \\98 - 56 = 3AB \\42 = 3AB \\ AB = \frac{42}{2} \\ AB = 14 ~cm[/tex]

8. Diketahui titik E, F, dan G pada trapesium ABCD. Sisi FE sejajar dengan sisi AB. Jika AB = 7, DC = 14, DG = 8, FG = 4, GB = x, dan GE = y, maka nilai x + y adalah ....

Penyelesaian :

Perhatikan gambar yang ada pada lampiran III.

Menentukan nilai x pada Δ ABD

Garis AB bersesuaian dg garis FG dan garis BD dg garis DG

[tex]\displaystyle \frac{AB}{FG} = \frac{(DG + BG)}{DG}[/tex]

[tex]\displaystyle \frac{7}{4} = \frac{(8 + x)}{8}[/tex]

4 (8 + x) = 7 × 8

32 + 4x = 56

      4x = 56 - 32

      4x = 24

        x = [tex]\displaystyle \frac{24}{4}[/tex]

        x = 6

Jadi nilai x = 6

Menentukan nilai y pada Δ DCB

Garis CD bersesuian dg garis EG dan garis BD bersesuaian dg garis BG

[tex]\displaystyle \frac{CD}{EG} = \frac{(DG + BG)}{BG}[/tex]

[tex]\displaystyle \frac{14}{y} = \frac{(8 + 6)}{6}[/tex]

14 y = 14 × 6

14 y = 84

   y = [tex]\displaystyle \frac{84}{14}[/tex]

   y = 6

Jadi nilai y = 6

Nilai x + y

x + y = 6 + 6

       = 12

Maka nilai x + y adalah 12  

9. Diketahui Trapesium ABCD,dengan AB//DC//PQ.  Jika perbandingan AQ : QC = BP : PD = 3 : 2. Tentukan Panjang ruas garis PQ.

Penyelesaian :

Perhatikan gambar yang terdapat pada lampiran IV.

∆ ABD

[tex]\displaystyle \frac{AB}{x} = \frac{BD}{PD}[/tex]

[tex]\displaystyle \frac{10}{x} = \frac{(2 + 3)}{2}[/tex]

[tex]\displaystyle \frac{10}{x} = \frac{5}{2}[/tex]

5x = 10 × 2

5x = 20

x = [tex]\displaystyle \frac{20}{5}[/tex]

x = 4

∆ ADC

[tex]\displaystyle \frac{DC}{PQ + x} = \frac{AC}{AQ}[/tex]

[tex]\displaystyle \frac{20}{PQ + 4} = \frac{3 + 2}{3}[/tex]

5 (PQ + 4) = 20 × 3

PQ + 4 = [tex]\displaystyle \frac{60}{5}[/tex]

PQ + 4 = 12

PQ = 12 - 4

PQ = 8 cm

Jadi panjang ruas PQ adalah 8 cm

Pembahasan

あなたは男ですか？

Anata wa otoko desuka?

_____

あなた (anata)

Berarti 'kamu' atau 'anda'. Bentuk kata ini tergolong jenis formal atau bisa juga digunakan dalam situasi yang kasual (asalkan lebih baik jangan diucapkan ketika dalam situasi yang sudah sangat akrab) dan kata anata ini termasuk golongan kata ganti orang kedua tunggal.

は (wa)

Biasanya partikel ini terletak di dekat sebelah subjek atau tidak. Dalam kalimat ini, terdapatnya kata ganti atau subjek あなた (anata) kemudian ditambahkan dengan partikel は (wa) setelahnya, sehingga fungsi dari partikel は (wa) ini adalah menjadi penanda untuk subjek.

男 (otoko)

Jika dalam versi hiragana kata otoko ini ditulis dengan おとこ

Dan artinya itu berarti 'lelaki', 'pria', atau 'laki-laki'. Penggunaan kata 男 (otoko) ini bisa digunakan kepada laki-laki yang merupakan anak-anak, pemuda, orang dewasa, maupun orang tua.

ですか (desuka)

Kata tanya ini memiliki kesan yang sopan/formal, termasuk sopan/formal karena terdapatnya salah satu partikel tambahan yaitu です (desu), selain itu ini jenis partikel yang mungkin jika dalam bahasa Indonesia yaitu apakah. Namun kata tanya ini juga biasanya tidak memiliki arti dan berperan menjadikan kalimat tersebut sebagai kalimat tanya.

Arti dari 'Anata wa otoko desuka' adalah "Apakah anda laki-laki?"

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Pembahasan :

Pola Kalimat

== [Subjek] wa [Predikat] desuka

== Apakah [Subjek] (adalah) [Predikat]?

Fungsi dari pola kalimat ini adalah untuk menanyakan suatu hal, apakah si subjek merupakan 'predikat'.

- - -

Untuk pembahasan lebih lanjutnya, akan saya bahas masing-masing kata yang digunakan untuk membuat kalimat di soal.

-

1. 貴方 / あなた / Anata

Berarti "Anda" atau "Kamu". Merupakan salah satu kata ganti orang kedua dalam Bahasa Jepang. Kata ini termasuk sopan dan umum digunakan. Tapi jika sudah mengenal lawan bicara, tidak baik untuk menggunakan kata ini. Bisa juga berarti panggilan "Sayang", "Beb", dsb. Biasa digunakan perempuan untuk memanggil pacar atau pasangan laki-laki nya.

-

2. は / Wa

Tidak ada arti, namun kadang bisa diartikan "Adalah". Kata ini merupakan partikel atau kata bantu kalimat, yang fungsinya untuk menandakan subjek atau topik dari suatu kalimat tersebut. Partikel ini dibaca Wa, tapi penulisannya dalam Hiragana menggunakan Ha.

-

3. 男 / おとこ / Otoko

Berarti "Laki-laki" atau "Pria". Kata ini dari segi kedewasaan merujuk pada pria dewasa, sedangkan dari segi usia bisa digunakan untuk laki-laki muda maupun tua. Kata ini termasuk sopan, tetapi lebih sopan lagi jika menggunakan 'Dansei'. Tapi kata Dansei tidak bisa digunakan untuk anak laki-laki.

-

4. です / Desu

Tidak ada arti. Merupakan akhiran kalimat yang berfungsi untuk memperhalus dan mempertegas suatu kalimat tersebut. Kata ini diletakkan setelah predikat, tapi tidak bisa setelah kata kerja. Untuk kata kerja, dengan mengubahnya ke bentuk Masu. Adapun kata serupa yang sering dianggap sebagai 'bentuk kasual' dari Desu, yaitu 'Da'. Kata Da ini hanya mempertegas kalimat saja, tidak memperhalus.

-

5. か / Ka

Tidak ada arti, tapi perannya sama seperti tanda tanya. Kata ini merupakan partikel yang berfungsi untuk mengubah suatu kalimat biasa menjadi kalimat tanya. Biasa diletakkan setelah 'Desu / Masu'.

-

Kata-kata di atas bila digabung akan menjadi 貴方は男ですか (Anata wa otoko desuka), yang berarti "Apakah anda laki-laki?".
