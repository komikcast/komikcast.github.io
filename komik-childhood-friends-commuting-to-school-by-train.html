PEMBAHASAN Present Tense

=> tenses yg digunakan untuk menyatakan:

Kegiatan berulang/rutin,Kebiasaan (habitual) danhal-hal bersifat umum atau untuk kebenaran umum (fakta).

Ket. waktu Present TenseEveryday /Every morningalwaysusuallyOftensometimes

Rumus Present Tense

(+) S + V1 (-s/es) + O/C

(-) S + Do/Does + not + V1 + O/C

(?) Do/Does + S + V1 + O/C?

=> Subject + Auxiliary

I, you, we, they + doHe, She, it + does

KESIMPULAN

1. They do not go to Jakarta by train

=> Mereka tidak pergi ke Jakarta menggunakan kereta

2. I always go to school by bicycle

=> Saya selalu pergi ke sekolah menggunakan sepeda

23. fatih goes to school by train bag is expensive please write a wh question of statement above ?


please write a wh question of statement below

Fatih goes to school by train

The bag is expensive

jawaban :

1. How does fatih go to school? Fatih goes to school by train

2. How is the bag's price? The bag is expensive.

_________

Wh -questions disusun berdasarkan pola sebagai berikut :

WH ( quest ) + auxiliary verb + Subject + verb  ? ( bentuk verbal )

WH ( quest ) + linking verb + noun/adj/adv ?   ( bentuk nominal )

=> Auxiliary verb ( do/does/did/have/had/will/would, etc )

=> Linking verb ( is/am/are )

=> Noun ( kata benda )

=> Adjective ( kata sifat )

=> Adverb ( keterangan )

Mapel : Bahasa inggris

Bab : WH- Questions

Kelas : 7

1. How does fatih go to school? Fatih goes to school by train

2. How is the bag's price? The bag is expensive.
