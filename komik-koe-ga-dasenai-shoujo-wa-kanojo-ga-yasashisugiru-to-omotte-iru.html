3. mau nanya perbedaan "kanojo" sama "shoujo"" itu apa


Kanojo ( 彼女 ) = Dia ( perempuan )
Onna ( 女 ) = Perempuan ( universal )
Oujo ( 王女 ) = Tuan Putri
Shoujo ( 少女 ) = Perempuan ( muda )Perbedaannya:
↪Kanojo (かのじょ): bisa diartikan sebagai "pacar perempuan".
↪Shoujo (しょうじょ): Cewek / perempuan (bisa dikatakan ini bahasa nonformal)

Cara membedakan nya gampang, hanya perlu menghapal kosa katanya.
↪かれし >< かのじょ(Kareshi >< kanojo) : Pacar laki-laki - pacar perempuan
↪男の子 >< 女の子 (otokonoko >< onnanoko): anak laki2 - anak perempuan
おとこのひと >< おんなのひと (otokohito >< onnanohito): laki-laki - perempuan

10. apa arti nya nani ka dekiru watasi ga koko ni iru dalam bahasa indonesia


Mungkin artinya bisa jadi seperti ini:
↪なにかできるわたしがここにいる ( Aku di sini / berada di sini untuk menjadi sesuatu.)

❇Penjelasan:
↪ なにか / nanika: sesuatu. (Kalau misalakan jadi kalimat tanya "nani ka?" bisa diartikan apa)
↪できる / dekiru: dapat / menjadi
↪ わたし / watashi: saya
↪ が / ga: sebuah partikel kata sifat
↪ ここ / koko: disini (dekat)
↪ ここにいる / koko ni iru: berada di sini.
