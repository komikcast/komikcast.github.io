33. arti dari Anata wa watashi no koto o dō omotte imasu ka ​


'Anata wa watashi no koto o dou omotte imasu ka' artinya dalam Bahasa Indonesia adalah "Bagaimana pendapatmu tentang diriku?"

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Pembahasan :

= ANATA「貴方・あなた」

Berarti "Anda" atau "Kamu". Merupakan salah satu kata ganti orang kedua tunggal dalam Bahasa Jepang. Kata ini termasuk sopan dan umum digunakan. Tapi jika sudah mengenal lawan bicara, tidak baik untuk menggunakan kata ini. Kata ini juga bisa berarti panggilan "Sayang", "Beb", dsb. Biasa digunakan perempuan untuk memanggil pacar atau pasangan laki-laki nya.

= WA「は」

Tidak ada arti, namun kadang bisa diartikan "Adalah". Kata ini merupakan partikel atau kata bantu kalimat, yang fungsinya untuk menandakan subjek atau topik dari suatu kalimat tersebut. Partikel ini dibaca Wa, tapi penulisannya dalam Hiragana menggunakan Ha.

= WATASHI「私・わたし」

Berarti "Aku" atau "Saya". Kata ini merupakan salah satu kata ganti orang pertama tunggal yang paling umum dan termasuk sopan. Kata ini bisa digunakan oleh dan untuk siapa saja, baik cowo/cewe, tua/muda, berkedudukan tinggi/rendah, dsb.

= NO「の」

Tidak ada arti. Kata ini merupakan partikel atau kata bantu kalimat, yang berfungsi untuk menyatakan suatu kepemilikan. Contoh mudahnya yaitu misalkan pada kalimat 'A no B', maka artinya seperti "B milik A", "B nya A", "B dari A", "B A", dsb.

= KOTO「事・こと」

Berarti "Hal". Hal yang dimaksud di sini adalah hal yang bukan berbentuk fisik, melainkan non-fisik seperti perasaan, dsb. Kalau dalam kalimat '... no koto', maka artinya menjadi "Tentang ...".

= WO / O「を」

Tidak ada arti. Kata ini merupakan partikel yang berfungsi untuk menandakan objek atau kata benda. Biasanya partikel ini diletakkan di antara sebuah kata benda dan kata kerja.

= DOU「どう」

Berarti "Bagaimana". Merupakan salah satu kata tanya dalam Bahasa Jepang. Kata ini termasuk umum, tapi ada juga versi yang lebih sopannya yaitu 'Ikaga'.

= OMOTTE IMASU「思っています」

Berarti "Berpikir" atau "Berpendapat". Kata ini digunakan saat si pembicara sudah memikirkan suatu hal sejak lama.

= KA「か」

Tidak ada arti khusus, tapi kurang lebih artinya seperti "...kah?" atau bisa juga seperti tanda tanya. Kata ini merupakan partikel yang berfungsi untuk mengubah suatu kalimat biasa menjadi kalimat tanya. Biasa diletakkan setelah 'Desu / Masu'.

- - - - - - - - - -

Kesimpulan :

Kata-kata tersebut bila disatukan menjadi sebuah kalimat akan menjadi 貴方は私の事をどう思っていますか (Anata wa watashi no koto o dou omotte imasu ka), yang berarti "Bagaimana pendapatmu tentang diriku?"
