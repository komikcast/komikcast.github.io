1. arti mada kono sekai wa​


Kalimat 'mada kono sekai wa ...' diartikan sepertinya dunia ini masih ....

Pembahasan

Berikut terjemahan beserta juga dengan penjelasan kata-perkatanya.

Mada「まだ」: memiliki 2 arti yakni belum dan masih. Tergolong jenis na-keiyoushi (kata sifat -na) dengan kelompok yang ke-ll. Namun kata mada bisa tergolong menjadi kata keterangan.

Kono「この」: kata tunjuk dan jika dalam bahasa Indonesia yang berarti ini. Menunjuk ke benda yang dekat jaraknya.

Sekai「世界・せかい」: berarti dunia, bumi.

Wa「は」: partikel yang paling umum digunakan sebagai penanda dalam subjek.

___

Jika kata-kata diatas ini digabungkan maka : まだこの世界は 。。。 (mada kono sekai wa ...) yang artinya adalah sepertinya dunia ini masih ....

2. arti "mada kono sekai wa" apa?​


mada kono sekai wa dalam bahasa Indonesia arti kalimat tersebut adalah (sepertinya) dunia ini masih ....

Pembahasan

まだこの世界は ....

Mada kono sekai wa ...

(sepertinya) dunia ini masih ....

_______

–› Mada「 まだ 」

Memiliki 2 arti dalam bahasa Indonesia yaitu adalah masih atau belum (tergantung konteks/penggunaan).

Secara umum, jika kata まだ (mada) tersebut bertemu dengan kata kerja maka artinya dapat diartikan belum.

Bentuk kata まだ (mada) tersebut merupakan bentuk kata sifat yang termasuk golongan na-keiyoushi (akhiran -na) ll.

–› Kono「 この 」

Artinya yaitu ini. Jika ingin merujuk atau menunjukkan suatu subjek, benda/objek yang jaraknya tidak jauh alias berada di dekat diri kita maka kata tunjuk yang cocok digunakan untuk hal ini adalah この (kono).

–› Sekai「 世界 • せかい 」

Kata tersebut terdapat 2 arti yaitu bisa diartikan semesta ataupun dunia.

Jika kata sekai memiliki awalan huruf i, maka artinya menjadi lain yaitu 異世界 (isekai) yang artinya adalah dunia lain/dunia berbeda.

–› Wa「 は 」

Penggunaannya sekedar sebagai kata bantu yang penulisannya ditulis dengan huruf hiragana, kata bantu yang dimaksud yakni sebagai penanda dalam kalimat. は (wa) termasuk kata bantu ataupun juga merupakan jenis partikel.

'Mada kono sekai wa' dalam Bahasa Indonesia artinya adalah "Dunia ini masih..."

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Pembahasan :

Berikut pembahasan per-kata.

MADA「まだ」

Bisa berarti "Masih" atau "Belum".

KONO「この」

Berarti "Ini". Merupakan kata tunjuk untuk menunjukkan sesuatu yang dekat dengan pembicara. Kata ini tidak dapat berdiri sendiri, dan harus menerangkan sesuatu setelahnya. Contohnya pada kalimat ini, yang diterangkan adalah 'Sekai' (dunia). Kalau diartikan menjadi "Dunia ini".

SEKAI「世界・せかい」

Berarti "Dunia".

WA「は」

Tidak ada arti. Merupakan partikel yang memiliki fungsi untuk menandakan subjek atau topik dari sebuah kalimat. Dalam beberapa situasi, partikel ini dapat diartikan "Adalah". Penulisan partikel ini dengan huruf Ha, tapi dibaca Wa.

- - - - - -

Kalimat 'Mada kono sekai wa' ini kemungkinan merupakan potongan lirik dari lagu Sparkle oleh RADWIMPS. Kalau berdasarkan lirik lagunya, kalimat lengkapnya seharusnya 'Mada kono sekai wa boku wo kainarashιtetai mitai da'. Artinya adalah "Sepertinya dunia ini masih ingin mengendalikanku".

Berikut pembahasan setiap kata yang tersisa di kalimat lengkapnya.

BOKU「僕・ぼく」

Berarti "Aku". Kata ini merupakan kata ganti orang pertama tunggal yang biasa dipakai di situasi informal, tapi masih terkesan sopan. Umumnya digunakan laki-laki saja.

WO / O「を」

Tidak ada arti. Merupakan partikel untuk menandakan objek atau kata benda. Biasa diletakkan di antara kata benda dan kata kerja. Pada kalimat ini, yang ditandai sebagai kata benda adalah 'Boku' (saya). Kalau dalam Hiragana ditulis dengan huruf Wo, tapi dalam Romaji bisa ditulis dengan huruf Wo maupun O.

KAINARASHlTETAI「飼いならしてたい」

Berarti "(Sedang) ingin menjinakkan". Kata kerja pada bentuk ini menunjukkan suatu keadaan yang masih ingin berlanjut. Kadang diartikan sebagai "Sedang ingin...". Kalau melihat konteks kalimat, kata ini sebenarnya lebih tepat kalau diartikan sebagai "(Sedang) ingin mengendalikan".

MITAI「みたい」

Berarti "Kelihatannya", "Sepertinya", "Seperti", dsb. Kata ini merupakan kata sifat-i, atau kata sifat yang terdapat tambahan huruf i di akhir. Kata ini diletakkan setelah predikat, untuk menyatakan suatu kemiripan pada predikatnya.

DA「だ」

Tidak ada arti. Fungsinya sebagai penegas kalimat. Mirip seperti 'Desu', tapi 'Da' tidak memperhalus kalimatnya.

25. tolong bantu ya :) Tantangan dari guru terjemahkan : Sekai = Ichiban = Oujisama = Watashi = Hoshii = dijawab ya ^-^


Sekai : dunia, bumi.Ichiban : nomor satu, terbaik, yang paling.Oujisama : pangeran, putra kerajaan.Watashi : saya, aku.Hoshii : ingin, mau.

Pembahasan

Sekai「世界 • せかい」

Untuk arti yaitu bumi, atau dunia. Contoh penggunaan kata tersebut adalah :

Kono wa ore no sekai da (ini adalah duniaku).Kono sekai wa watashi no basho desu (bumi ini adalah tempatku).

Ichiban「いちばん」

Artinya ialah terbaik, yang paling, atau nomor satu. Kata ini termasuk salah satu pola kalimat perbandingan bentuk superlatif.

Memiliki 2 pola kalimat sebagai berikut :

KB : KB + partikel は/が (wa/ga) + いちばん + kt. sifat + だ/です (da/desu).

Contohnya :

- Eigo wa ichiban kantan desu (bahasa Inggris paling mudah).

- Anata wa ichiban utsukushii desu (kamu paling cantik).

KK : KK bentuk kamus + partikel が/は (wa/ga) + いちばん + kt. sifat + だ/です (da/desu).

Oujisama「王子様 • おうじさま」

Dalam bahasa Indonesia memiliki 2 arti yaitu pangeran, atau putra kerajaan.

Watashi「私 • わたし」

Yakni bermakna aku ataupun saya, kata ganti pertama tunggal dengan konteks semi formal dan formal.

Watashi wa Shafa desu (aku Shafa).Watashi to tomodachi gakkou ni ikimasu (aku dan teman pergi ke sekolah).

Hoshii「ほしい」

Maknanya ialah ingin, pengen atau mau, jenis ungkapan yang biasanya untuk menyatakan suatu hal kemauan atau keinginan untuk mendapatkan atau melakukan hal sesuatu. Merupakan salah satu golongan i-keiyoushi (kata sifat dengan akhiran -i).

Contoh penggunaan katanya adalah :

Ramen ga hoshii desu ((aku) mau ramen).Watashi wa anata ga hoshii desu (aku mau kamu).

34. Watashi wa sekai de mottomo muchina hitodesu artinya apa ya???


Bahasa jepang 
BAB 4 N5

Watashi wa sekai de mottomo muchi na hito desu
(わたしはせかいでもっともむちなひとです)

Terjemahannya adalah:
Saya adalah orang yang paling bodoh di dunia

Kosakata:
Watashi (わたし) : Aku/Saya
Wa (は) : partikel penanda subjek
Sekai (せかい) : Dunia
De (で) : Di
Mottomo (もっとも) : Sangat/Paling
Muchi (むち) : Tidak tahu / Bodoh
Hito (ひと) : Orang
Desu (です) : akhiran dalam sebuah kalimat

Nb:
"na" disini sebagai kata sifat. Pola kalimatnya : Kata sifat na + kata benda

37. Kono michi wa bokurano ai no akashi kono michi de boku wa anata o matsuyo


BAHASA JEPANG

Pembahasan :
Kono michi wa bokurano ai no akashi
kono michi de boku wa anata o matsuyo

→→ Arti ←←
Dijalan ini adalah bukti cerita cinta kita
Dijalan ini aku menunggu kamu

→→ Bahasa Jepangnya ←←
この みち は ぼくらの 愛 の あかし
この みち で ぼく は あなた を まつよ

→→ Arti Per Kata ←←
Kono = この = Kono
Michi = みち = Jalan
Wa = は = Partikel / Kata Bantu
Bokura no = ぼくら の = kita
Ai = 愛 = Cinta
No = の = Partikel / Kata Bantu
Akashi = あかし = Bukti
De = で = Partikel / Kata Bantu
Boku = ぼく = Saya (biasanya diucapkan oleh cowok, bahasanya sangat sopan)
Anata = あなた = Kamu
O = を = Partikel / Kata Bantu
Matsuyo = まつよ = Menunggu
