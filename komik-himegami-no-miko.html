7. Miko, Keddy dan Sharko mempunyai Total 586 Buah perangko Miko mempunyai perangko 3 kali lebih banyak daripada Kedi Tarko mempunyai 481 perangko lebih banyak daripada Miko Berapakah jumlah perangko yang dimiliki oleh miko?tolong di jawab mksii​


Jawaban:

asumsikan

Miko = M

Keddy = K

Sharko = S

maka model matematikanya adalah

M + K + S = 586 (pers. 1)

M = 3K (pers. 2)

S = M + 481

S = 3K + 481 (pers. 3)

substitusi pers. 2 dan pers. 3 ke pers. 1

3K + K + 3K + 481 = 586

7K = 586 - 481

7K = 105

K = 105/7

K = 15

substitusi nilai K untuk mencari nilai M

M = 3K

M = 3 x 15

M = 45

maka banyaknya prangko yang dimiliki Miko adalah

45 buah

18. Miko, Keddy dan Sharko mempunyai total 438 buah prangko. Miko mempunyai prangko 3 kali lebih banyak daripada Keddy. Sharko mempunyai 263 prangko lebih banyak daripada Miko. Berapakah jumlah prangko yang dimiliki oleh Miko?


Jawaban:

75 prangko

Penjelasan dengan langkah-langkah:

Misalkan

Miko = m

Keddy = k

Sharko = s

m + k + s = 438

3k + k + 263 + m = 438

4k + 263 + 3k = 438

7k + 263 = 438

7k = 438 - 263

7k = 175

k = 175 : 7

k = 25 prangko

m = 3k

= 3 × 25

= 75 prangko

Jadi, jumlah prangko yang dimiliki oleh Miko adalah 75 prangko

22. pls jawab salah report , betul dapat braainlest Miko, Keddy dan Sharko mempunyai total 349 buah prangko. Miko mempunyai prangko 3 kali lebih banyak daripada Keddy. Sharko mempunyai 202 prangko lebih banyak daripada Miko. Berapakah jumlah prangko yang dimiliki oleh Miko?


Jawaban:

x + y + z = 349 -----------------------> ( 1 )

x = miko

y = keddy

z = sharko

x = 3y ---------------> y = 1/3 x -----> ( 2 )

z = x + 202 ---------------------------> ( 3 )

ditanya x......???

selesaiannya :

x + y + z = 349

x + ( 1/3 x ) + ( x + 202 ) = 349

7/3 x = 349 - 202

7/3 x = 147

x = 147 × 3/7

x = 63

jadi perangko yg dimiliki miko berjumlah

63 buah.

» Penyelesaian «

Permisalan :

• a → Miko

• b → Keddy

• c → Sharko

a + b + c = 349

a = 3b

c = 202 + a

3b + b + 202 + 3b = 349

3b + b + 3b = 349 - 202

7b = 147

b = 21 perangko

a = 3b

a = 3(21)

a = 63 perangko

Kesimpulan

jadi Miko memiliki 63 perangko

37. Miko, Keddy dan Sharko mempunyai total 577 buah prangko. Miko mempunyai prangko 3 kali lebih banyak daripada Keddy. Sharko mempunyai 486 prangko lebih banyak daripada Miko. Berapakah jumlah prangko yang dimiliki oleh Miko? ​


Jawaban:

81 perangko

Penjelasan dengan langkah-langkah:

Diketahui:

Total seluruh perangko=577 perangko

Miko mempunyai perangko 3 kali lebih banyak dari Keddy,Sharko mempunyai 486 perangko lebih banyak dari Miko

Ditanyakan:

Jumlah perangko yang dimiliki oleh Miko

Jawab:

=Jumlah perangko - Jumlah perangko yang dimiliki oleh Sharko

=577 - 486

=81

Kesimpulan:

Jadi,jumlah perangko yang dimiliki oleh Miko adalah 81 perangko
