1. Future Tense

(+) S + WILL + VERB 1 + O

(+) S + WILL BE + VERB 3 + By ...

(+) S + is/am/are + VERB+ing + O

Active: The workers are finishing the work now.

(+) S + is/am/are + BEING + VERB 3

a. The work IS BEING FINISHED now ✅

(+) S + VERB 2 + O

Active: Dewa performed the songs amazingly

Passive:

(+) S + was/were + VERB 3 + By

b. The songs WERE PERFORMED by Dewa amazingly ✅

4. Past Continues-Passive

(+) S + was/were + VERB 3 + By

b. Our bookselves WAS BEING REPAIRED by the carpenter when you came ✅

(+) S + has/have + been + VERB 3 + By

6. the files are being repaired by me now ubah jadi active voice​


The active voice of the files are being repaired is...

Answer:I'm repairing the files.

Discussion:Passive voice

Kalimat pasif merupakan pernyataan suatu obyek sebagai subyek dengan diletakkan pada awal kalimat dan diperankan oleh subyek yang bertukar sebagai obyeknya.

Merubah kalimat aktif menjadi pasif cukup mudah dengan:

menukarkan subyek ke obyek dan obyek ke subyeksetiap subyek yang berperan sebagai obyek perlu diawali dengan bykata kerja terdapat perubahan dengan mengamati penggunaan tense-nya

Kalau kita perhatikan pada soal, itu merupakan bentuk dari present continuous.

O + to be (is/am/are) + be-ing + V3 + by + S + ...dengan to be merupakan kunci dari pernyataan kalimat progresif dan dapat digunakan salah satu dengan sesuai obyeknya.setelah S adalah sisa kalimatnya.

Jika diubah dalam bentuk aktif:

S + to be (is/am/are) + V-ing + O + ...

Dengan:

pronoun: me (obj) → I (subj)to be: is (the files | it | obj) → am (I | subj)verb-ing: repairing (repaired in passive)noun: the filesadverb: now (the rest of the sentence)

Jadi kalimat aktifnya:

I am repairing the files now.

PASSIVE VOICE  

Kalimat aktif adalah kalimat yang subjeknya melakukan aksi, sementara kalimat pasif adalah kalimat dimana subjeknya tidak melakukan aksi tetapi menerima aksi. Ciri khas dari kalimat passive adalah TOBE+V3(Past Participle). Syarat-syarat perubahan kalimat aktive ke kalimat pasif adalah:  

Subjek pada kalimat aktif berubah menjadi objek pada kalimat pasif.   Sebaliknya, objek pada kalimat aktif berubah menjadi subjek pada kalimat pasif.

Selalu dibentuk dengan menggunakan pola TOBE+ VERB 3 (Past particile).

Tobe yang digunakan sesuai dengan pola tenses dari kalimat aktif.

Terkadang menggunakan kata by jika yang melakukan aksi/perbuatan pada kalimat aktif perlu dan sangat penting untuk diketahui.  

Passive Voice pada beberapa tense:

1. Simple Present Tense  

Aktif : S + V1+(S/ES)+O  

Pasif : S + to be (am, are, is) + V3 + by + O  

2. Present Continuous Tense  

Aktif : S + to be (am, are, is) + V1-ing + O  

Pasif : S + to be (am, are, is) + being + V3 + by + O  

3. Present Perfect Tense  

Aktif : S + have/has + V3 + O  

Pasif : S + have/has + been + V3 + by + O  

4. Present Perfect Continuous Tense  

Aktif : S + have/has + been + V1-ing + O  

Pasif : S + have/has + been + being + V3 + by + O  

5. Simple Past Tense  

Aktif : S + V2 + O  

Pasif : S + to be (was, were) + V3 + by + O  

6. Past Continuous Tense  

Aktif : S + to be (was, were) + V1-ing + O  

Pasif : S + to be (was, were) + being + V3 + by + O  

7. Past Perfect Tense  

Aktif : S + had not + V3 + O  

Pasif : S + had + been + V3 + by + O  

8. Past Perfect Continuous Tense  

Aktif : S + had + been + V1-ing + O  

Pasif : S + had + been + being + V3 + by + O  

9. Simple Future Tense  

Aktif : S + will + V1 + O  

Pasif : S + will be + V3 + by + O  

10. Simple Future Past Tense  

Aktif : S + would + V1 + O  

Pasif : S + would + be + V3 + by + O

Jawabannya yang benar adalah E. The packages are now being delivered to the customers

Jawaban:

E. The packages are now being delivered to the customers.

Penjelasan:

Subject + to be + verb(ing) + adverb

Subject: the packages

To be: are

Verb(ing): being

Adverb: to the customers

Delivered di sini berfungsi sebagai "diantarkan", bukan "mengantarkan" versi past tensenya, karena di belakangnya terdapat adverb "to the customers".

Kita harus memilih present continuous tense karena ada kata "now" yang berarti sekarang atau sedang terjadi.

35. Ubah kedalam kalimat aktif right now the novel is being written by deni


Jawaban:

PEMBAHASAN

Right now the novel is being written by deni

Structure kalimatnya:right now => ciri Present Continuous The novel => Subjek kalimat Passive is => Tobe untuk Subjek tunggal being + written => Be + V3 (pola Passive)by deni => By Agent

Berdasarkan keterangan di atas:Kita bisa mengidentifikasi bahwa kalimat tersebut adalah kalimat Passive dari Present Continuous. Sebab terdapat keterangan waktu Present Continuous yaitu Right Now (saat ini)

Berikut Berdasarkan Rumus

Passive-Present Continuous

(+) S + is/am/are + being + V3 + By Agent

➡ right now the novel is being written by deni

Saat ini Novel sedang ditulis oleh Deni

Active-Present Continuous

(+) S + is/am/are + V-ing + O

➡ Deni is writing the novel right now
