Maaf Kalau SalahSemoga MembantuJadikan Jawaban Terbaik Jika PerluTerima Kasih ✨☄️

a. Has

→ Mengapa jawabannya a.has ???

Karena Subject nya ialah perempuan ( my little sister ). My Little sister memiliki arti adik perempuan saya sedangkan she ialah dia perempuan, intinya setiap subject nya ada kalimat bahwa dia perempuan seperti sister,mother dll ia termasuk menggunakan subject she. Penggunaan have/has dibagi sesuai subject nya.

✔Subject yang using have :

I → using have

You → using have

They → using have

We → using have

✔Subject yang using has :

She → using has

He → using has

It → using has

Jadi, My little sister has a pink tricycle

maaf klo salah

Any adalah sebuah quantifier(kata yang digunakan untuk menyatakan jumlah).

Any digunakan untuk kalimat negatif atau kalimat interogatif(tanya).

Misalnya:

(Negative) I have't got any friends.

(Interrogative) Have you got any friends?

Jadi, jawabannya adalah: a. any
