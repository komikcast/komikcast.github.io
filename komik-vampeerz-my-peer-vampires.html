Jawaban:

Peer to peer ialah jaringan yg menghubungkan 1 komputer dg komputer lain, dimana tidak ada peran server dalam jaringan tersebut.

Peer to server ialah jaringan yg menghubungkan 1 komputer dg komputer lain, dimana ada 1 komputer yg berperan sebagai server pada jaringan itu.

Penjelasan:

Keuntungan jaringan peer to peer:

- implementasinya mudah (karena sederhana)

- tidak membutuhkan sistem operasi/ aplikasi tertentu khusus server

Keuntungan jaringan peer to server:

- lalu lintas data dalam jaringan lebih terjamin keamanannya (karena ada komputer server yg mengamankan data dalam jaringan).

- pengaturan jaringan cukup mudah, karena administrasi (pengelolaan) jaringan dipusatkan di server.
