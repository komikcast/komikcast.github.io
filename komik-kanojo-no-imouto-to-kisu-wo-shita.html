8. Watashi 5-nin kazokudesu Chichi to naga to imouto to watashidesu Chichi wa 50-sai Desy


Mungin Maksud dari "Naga" adalah "Haha"

Keluarga saya Berjumlah 5 Orang. Ayah , Ibu, Adik Perempuan dan Saya. Ayah Berumur 50 Tahun.

Pembahasan

Sakubun / 作文 / Karangan

Terjemahan:

Keluarga saya Berjumlah 5 Orang. Ayah , Ibu, Adik Perempuan dan Saya. Ayah Berumur 50 Tahun.

Kanji , Hiragana:

私5人家族です。父と母と妹と私です。父は50さいです。

Romaji:

Watashi 5-nin kazoku desu Chichi to Haha to imouto to watashi desu Chichi wa 50-sai Desu.

＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿

Penjelasan

Karangan / Sakubun diatas Ini, Bisa di Bilang sakubun yang Belum jadi, Atau Mungkin ada Lanjutannya tetapi tidak Engkau kirim.

Kita Pecah Menjadi 3 Bagian:

私5人家族です。( Watashi 5 Nin Kazoku Desu ): Keluarga Saya Berjumlah 5 Orang父と母と妹と私です ( Chichi to Haha to Imouto To Watashi Desu ) Ayah , Ibu , Adik ,Dan Saya父は50さいです。( Chichi wa 50 Sai Desu ) Ayah Berumur 50 Tahun

言葉 「ことば」

• 私 ( Watashi ) Saya

• 人 ( Nin ) Orang

• 家族 ( Kazoku )

• 父 ( Chichi ) Ayah

• と ( To ) Partikel

• 母 ( Haha ) Ibu

• と ( To ) Partikel

• 妹 ( Imouto ) Adik

• と ( To ) Partikel

• 私 ( Watashi ) Saya

• 父 ( Chichi ) Ayah

• は ( Wa ) Partikel

• さい ( Sai ) Umur

~ です ( Desu ) Tidak memiliki arti berfungsi sebagai penghalus kalimat

Keluarga saya Berjumlah 5 Orang. Ayah , Ibu, Adik Perempuan dan Saya. Ayah Berumur 50 Tahun.

Kanji , Hiragana:

私5人家族です。父と母と妹と私です。父は50さいです。

Romaji:

Watashi 5-nin kazoku desu Chichi to Haha to imouto to watashi desu Chichi wa 50-sai Desu.

Karangan / Sakubun diatas Ini, Bisa di Bilang sakubun yang Belum jadi, Atau Mungkin ada Lanjutannya tetapi tidak Engkau kirim.

私5人家族です。( Watashi 5 Nin Kazoku Desu ): Keluarga Saya Berjumlah 5 Orang父と母と妹と私です ( Chichi to Haha to Imouto To Watashi Desu ) Ayah , Ibu , Adik ,Dan Saya父は50さいです。( Chichi wa 50 Sai Desu ) Ayah Berumur 50 Tahun

• 私 ( Watashi ) Saya

• 人 ( Nin ) Orang

• 家族 ( Kazoku )

• 父 ( Chichi ) Ayah

• と ( To ) Partikel

• 母 ( Haha ) Ibu

• と ( To ) Partikel

• 妹 ( Imouto ) Adik

• と ( To ) Partikel

• 私 ( Watashi ) Saya

• 父 ( Chichi ) Ayah

• は ( Wa ) Partikel

• さい ( Sai ) Umur

~ です ( Desu ) Tidak memiliki arti berfungsi sebagai penghalus kalimat
