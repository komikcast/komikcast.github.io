17. perbedaan want dan wont


Penulisan yang benar adalah "won't" yg merupakan singkatan dari "will not" 
Dari artinya sendiri, won't adalah "tidak akan" 

cth: 
- I won't buy this book. bisa ditulis juga I will not buy this book, yang artinya sendiri "saya tidak akan membeli buku ini" 
- I won't tell you my secret, yang artinya "saya tidak akan memberitahu anda rahasia saya" 

Sedangkan "want" sendiri artinya mau, dan bila digunakan untuk menyatakan keinginan melakukan suatu pekerjaan, penggunaan want menjadi "want to" 

cth: 
- I want it, yang artinya "saya menginginkannya" 
- I want to buy this book, yang artinya "saya ingin/mau membeli buku ini". Perhatikan, sebelum kata kerja "buy" terdapat artikel "to" 
- I want to tell you my secret, yang artinya "saya mau memberi tahu anda rahasia saya"

PEMBAHASAN :CONDITIONAL SENTENCES

↪ Kalimat pengandaian yang mana hal itu akan terjadi jika syaratnya terpenuhi.

Contoh :

If you study hard, you will pass the exam. (Jika kamu belajar keras, kamu akan lulus ujian)

Syaratnya : Harus belajar keras. Jika syarat ini tidak terpenuhi maka tidak akan lulus ujian.

Conditional Sentences terbagi menjadi 3 macam, yaitu :

Conditional Type 1

Rumus :

If + Simple present, conditional (=will + Verb-1)

Contoh :

➡ If you study hard, you will pass the exam.

If clause-nya bentuk simple present. Ciri-ciri simple present kata kerjanya menggunakan verb-1 (s/es) jika kalimatnya positif. Tapi jika kalimat negatif menggunakan do/does + not + Verb-1.Conditional sentencesnya menggunakan "will + Verb-1".

Conditional Type 2

Conditional Type 2Rumus :

If + Simple past, conditional (= would+ Verb-1)

Contoh :

➡ If you studied hard, you would pass the exam.

If clause-nya bentuk simple past. Ciri-ciri simple past kata kerjanya menggunakan verb-2 jika kalimatnya positif. Tapi jika kalimat negatif menggunakan did + not + Verb-1.Conditional sentencesnya menggunakan "would + Verb-1".

Conditional Type 3

Conditional Type 3Rumus :

If + Past perfect, conditional (= would + have + Verb-3)

Contoh :

➡ If you had studied hard, you would have passed the exam.

If clause-nya bentuk past perfect. Ciri-ciri past perfect "had + Verb-3".Conditional sentencesnya menggunakan "would + have + Verb-3".

KESIMPULAN

Pada kalimat "We.... Paris if we... Enough money" diketahui bahwa :

Kalimat conditionalnya adalah "We.. Paris". Kalimat if clausenya "If we... Enough money".

Pada pilihan jawaban diatas termasuk conditional type 1. Kenapa? Karena pilihan auxiliary verb-nya adalah bentuk simple present yaitu "don't/doesn't". Maka untuk melengkapi kalimat diatas, kita harus ingat rumus conditional type 1 yang dijelaskan diatas bahwa :

Conditional sentence menggunakan "will + Verb-1". Karena pilihan jawabannya bentuk negatif, maka "Won't + Verb-1".

If clausenya menggunakan "simple present". Karena kalimatnya bentuk negatif, maka auxiliary verb yang digunakan harus sesuai subjectnya. Dimana subjectnya adalah "we" maka pakai "don't".

Jadi jawabannya adalah :

➡ We won't visit Paris if we don't save enough money.
