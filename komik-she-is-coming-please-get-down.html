Jawaban:

PEMBAHASAN Simple Present Tense

Digunakan untuk menyatakan Kegiatan yang dilakukan secara berulang, kebiasaan, rutin dan fakta.

Adverb of FrequencyAlwaysUsuallyOftenNever ✔️

Rumus Present Tense (+) S + Adv. freq + Verb-1 (-s/es)(-) S + do/does + not + Adv. freq + Verb-1 (-s/es)(?) Do/does + S + Adv. freq + Verb-1 (-s/es)?

Catatan Subject jamak menggunakan Verb-1Subject tunggal menggunakan Verb+s

KESIMPULAN

She never ... late.

a. come => Verb-1b. is coming => Be “is” + V-ing “coming” c. came => Verb-2d. comes => Verb+s ✅

Jadi, Jawaban yg D karena Subjek tunggal “He” maka menggunakan Verb+s “comes”
