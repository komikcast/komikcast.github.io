fasilitas word option terdiri dari sembilan macam kelompok, (catatan : setiap masing - masing versi beda jumlah kelompok, disini saya menggunakan microsoft office word 2007). diantaranya terdiri dari: popular, display, proofing, save, advanced, customize, add-ins, trust center, dan resources.

save terdiri dari tiga kelompok yaitu save document berfungsi sebagai tempat documen disimpan sementara, offline editing options for document management server file berfungsi sebagai tempat document disimpat ketika tiba - tiba keluar sebelum di simpan. preserve fidelity when sharing this document berfungsi untuk membagikan document ini.
