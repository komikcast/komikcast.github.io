12. My maid has washed those clothes.In passive is............​


(active voice)

My maid has washed those clothes

↓

(passive voice)

those clothes have been washed by my maid

PENJELASAN :Present Perfect Tense

→ rumus active voice :

(+) s + have/has + v3

(–) s + have/has + not + v3

(?) have/has + s + v3

→ rumus Passive voice :

(+) s + have/has + been + v3 + by + o

(–) s + have/has + not + been + v3 + by + o

(?) have/has + s + been + v3 + by + o
