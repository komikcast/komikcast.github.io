1. apa perbedaan chichi dan otou san???


Perbedaannya

Chichi「父・ちち」: ayah (cocoknya digunakan ketika sang pembicara sedang membicarakan mengenai ayahnya sendiri).Otousan「お父さん・おとうさん」: ayah (cocoknya ketika sang pembicara sedang membicarakan mengenai ayahnya orang lain).

_____________________

_____________________

Pendahuluan

KAZOKU「家族・かぞく」

Memiliki artian keluarga kedalam bahasa Indonesia. Dalam bahasa Jepang, ada banyak sekali mengenai frasa-frasa tentang penyebutan nama-nama keluarga dalam bahasa Jepang. Hampir setiap frasanya memiliki fungsi yang berbeda, tingkat keformalannya pun juga berbeda.

Di pembahasan kali ini saya hanya membahas mengenai perbedaan dan persamaan mengenai kedua kata chichi dan otousan.

Pembahasan

Persamaan yang terletak diantara kedua kata tersebut adalah bahwa kedua kata itu bisa kita gunakan sebagai panggilan untuk sang ayah sendiri. Bisa untuk panggilan terhadap sang ayah kandung, tiri, ataupun angkat.

Chichi「父・ちち」

= ayah/bapak.

Ketika sang pembicara sedang berbicara kepada orang lain misalnya berbicara kepada teman, saudara, rekan, dll. Kemudian sang pembicara pun membicarakan ayahnya sendiri kepada sang lawan bicara, maka kata yang tepat adalah menggunakan kata 父 (chichi).

Contoh :

Watashi no chichi wa isha desu : ayahku seorang dokter.Watashi no chichi wa hansamu desu : ayahku ganteng.

---------------

Otousan「お父さん・おとうさん」

= ayah/bapak.

Ketika sang pembicara sedang berbicara terhadap lawan bicara, kemudian membicarakan mengenai ayah orang (bukan ayah sendiri), maka kata yang tepat adalah menggunakan kata お父さん (otousan) agar memberikan kesan yang sopan.

Kata お父さん (otousan) ini memiliki awalan huruf お (o) agar terdengar baik/sopan, kemudian diakhiri dengan kata 〜さん (-san) yang merupakan salah satu panggilan hormat dalam bahasa Jepang.

Contoh :

Kare wa Udin no otousan desu : dia (lk) adalah ayahnya Udin.Rafi no otousan wa yasashii hito desu : ayah Rafi orang yang baik.

Perbedaan Chichi dan Otou-san adalah Chichi umumnya digunakan saat membicarakan ayah sendiri sebagai orang ketiga dalam percakapan formal, sedangkan Otou-san untuk ayah orang lain. Namun Otou-san juga bisa digunakan untuk memanggil ayah sendiri secara langsung, atau sebagai orang kedua.

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Pendahuluan :

家族 (Kazoku)

Merupakan istilah untuk "Keluarga" yang umum digunakan di Jepang. Penyebutan anggota keluarga dalam Bahasa Jepang ada yang untuk keluarga sendiri, dan ada yang untuk keluarga orang lain. Selain itu penyebutannya juga ada yang sopan, dan ada juga yang akrab.

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Pembahasan :

Bahasa Jepang dari "Ayah", beberapanya yaitu Chichi dan Otou-san. Kedua kata ini sama-sama berarti "Ayah", tapi penggunaannya berbeda. Perbedaannya yakni sebagai berikut.

CHICHI「父・ちち」

Merupakan sebutan untuk ayah sendiri, sebagai orang ketiga, dalam percakapan formal. Bisa diartikan dengan "Ayah saya" atau "Ayahku". Kata ini tidak bisa digunakan untuk memanggil ayah secara langsung, sebagai orang kedua. Hanya bisa digunakan sebagai sebutan untuk orang ketiga. Contoh penggunaannya adalah sebagai berikut.

1. この鞄は父が買った

= Kono kaban wa chichi ga katta

= Tas ini ayahku yang beli

2. 父は優しくてハンサムです

= Chichi wa yasashikute hansamu desu

= Ayah saya baik hati dan tampan

OTOU-SAN「お父さん・おとうさん」

Merupakan sebutan untuk ayah orang lain sebagai orang ketiga, dan juga untuk ayah sendiri sebagai orang kedua. Bisa diartikan dengan "Ayahmu" maupun "Ayah" saja. Saat membicarakan ayah orang lain, kata yang digunakan adalah Otou-san. Saat membicarakan ayah sendiri, kata yang digunakan adalah Chichi. Saat memanggil ayah sendiri, bisa menggunakan Otou-san. Dan saat memanggil ayah orang lain, biasa dengan nama yang ditambahkan gelar '-San'. Contoh penggunaan kata Otou-san adalah sebagai berikut.

1. その鞄はお父さんが買ったの？

= Sono kaban wa otou-san ga katta no?

= Tas itu ayahmu yang beli?

2. お父さんはどんな人ですか？

= Otou-san wa donna hito desu ka?

= Ayahmu orang yang bagaimana?

3. お父さん、これ見て！

= Otou-san, kore mite!

= Ayah, lihat ini!

- - - - - - - - - - - - -

Pada kata Otou-san, selain menggunakan gelar '-San', kalian bisa menggunakan beberapa gelar lainnya. Gelar '-San' sendiri digunakan untuk memberi kesan sopan dan hormat. Untuk gelar-gelar lainnya yang bisa digunakan, fungsinya adalah sebagai berikut.

'-Chan' : Gelar ini digunakan untuk memberi kesan akrab, imut, sekaligus manja. Biasa hanya digunakan anak-anak, untuk memanggil ayahnya sendiri. (Otou-chan)

'-Sama' : Gelar ini digunakan untuk memberi kesan yang sangat amat sopan, sekaligus rasa hormat yang sangat amat tinggi, jauh melebihi gelar '-San'. Biasa hanya digunakan saat menyebut ayah dari seseorang berkedudukan tinggi, seperti bos. (Otou-sama)

Terjemahan kata-perkata dari kalimat

ゆみこさんは日本人です :

=> ゆみこ (Yumiko)

Artinya adalah Yumico, hurufnya dalam bahasa Jepang ditulis dengan menggunakan huruf hiragana.

=> さん (san)

Merupakan salah satu jenis panggilan umum yang bertujuan untuk menyebutkan seseorang terhadap orang yang terhormat, lebih tua, atau belum akrab. Kata -san selalu diletakkan di akhir setelah nama.

Tanda akhir pada kalimat, yang merupakan salah satu partikel.

23. Desu / wa / Ali-san / kaban / no / koreSusunan kalimat yang benar adalaha. Kore no Ali san wa kaban desu.b. Kore wa kaban no Ali-san desu.c. Kore desu wa Ali san no kaban.d. Kore wa Ali san no kaban desu.​


Susunan kalimat yang benar adalah d. Kore wa Ali-san no kaban desu.

Pembahasan

Dalam soal tersebut, tentunya terdapat kalimat yang acak kemudian sang penjawab harus menyusun kalimat tersebut agar menjadi kalimat yang benar.

Tentu saja kalimat yang benar adalah これはアリーさんかばんです (kore wa Ali-san no kaban desu), opsi (D).

Sedangkan untuk opsi lainnya itu tidak sesuai.

-----

これはアリーさんのかばんです

Kore wa Ali-san no kaban desu

Ini adalah tas Ali

これ (kore)

Jika dalam bahasa Indonesianya berarti ini (salah satu kata yang memiliki kegunaan untuk menunjuk sesuatu). Kata ini biasa diucapkan ketika terdapat suatu benda yang ditunjuk tersebut berada tepat di dekat.

Biasanya benda yang berada di dekat tersebut ada di dekat yang berbicara.

~~

は (wa)

Untuk memberitahukan bagian suatu topik. Biasa juga disebut sebagai kata bantu untuk subjek.

~~

アリーさん (Arii (Ali)-san)

Nama アリー ini dibentuk dengan menggunakan katakana.

ア (a).リ (ri) : dikarenakan dalam katakana tidak memiliki huruf L, maka dialternatifkan menggunakan huruf リ (ri).ー : berfungsi untuk memanjangkan huruf katakana.

Sehingga nama アリー ini dibaca arii (Ali).

Terdapat kata さん (san) yang gunanya sebagai jenis panggilan kepada seseorang yang lebih tua berdasarkan usia ataupun jabatan dari kita sendiri.

~~

の (no)

Dalam konteks kalimat tersebut, partikel の (no) memiliki fungsi dalam kalimat ini yang berperan sebagai menyatakan kepemilikan. Seperti yang terdapat dalam kalimat アリーさんのかばん (Arii-san no kaban) yang berarti adalah tas (milik) Ali.

~~

かばん (kaban)

Ditulis dalam bentuk hiragana, yang dalam bahasa Indonesia artinya adalah tas.

~~

です (desu)

Salah satu partikel akhir.

~~

penjelasan partikel は (ha dibaca wa)

mengikuti topik yang ingin dibicarakan oleh pembicara. Oleh karena itu, ha (dibaca wa) sering disebut partikel penanda topik. “Topik” seringkali merupakan subjek gramatikal, tetapi dapat berupa apa saja (termasuk objek tata bahasa, dan terkadang) kata kerja), dan mungkin juga mengikuti beberapa partikel lainnya.        

pola kalimat = [ A ] wa [ B ] desu.

contoh =

昨日は雨だった。

Kinou wa ame datta.

の (no)

menunjukkan kepemilikan . Dalam struktur A no B, B milik A, namun, banyak kata benda bertindak seperti kata sifat jika diikuti oleh no.

Ini secara langsung mengikuti kata benda dan frase kata benda.
