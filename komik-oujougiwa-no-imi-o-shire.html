Pembahasan

Bahasa Jepang terbagi menjadi 3 Tulisan

- Hiragana

- Katakana

- Kanji

Soal :

Konnichiwa no imi wa nan desuka?

Jawaban

Konnichiwa no imi wa nan desu ka ?

[こんにちはのいみはなんですか? ]

--> Arti :

Halo , Apa itu ?

Keterangan -

こんにちは [ Konnichiwa ]

--> Konnichiwa artinya :

-> Halo

-> Selamat Siang

-> selamat Sore

なん { 何 } [ Nan ] -

--> Nan artinya apa

-> ' Apa ' Dalam bahasa Jepang :

--> なん [ 何 ] [ Nan ]

--> なに [ 何 ] [ Nani ]

Keduanya arti nya tetap sama

です [ Desu ]

• Termasuk dalam bentuk formal

か [ Ka ]

--> Partikel '' Tanya ''
