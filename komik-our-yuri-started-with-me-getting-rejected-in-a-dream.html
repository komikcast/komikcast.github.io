7. chage the active voice into passive voice1. she has rejected me,but I'm still in love


Bentuk passive daru kalimat "she has rejected me,but I'm still in love" adalah...

Karena kalimat aktif nya termasuk present perfect :
Subject + Have/has + V3 + object
=> she has rejected me,but I'm still in love.

Maka rumus passive nya :
Subject + Have/has + been + V3 + by...

Ingat !!!
Subject dikalimat aktif berubah jadi object di kalimat pasif, begitu pula sebaliknya.

Jadi jawabannya adalah...
=> I HAVE BEEN REJECTED by her. But I'm still in love.
================================
Mapel : Bahasa Inggris
Materi : Passive Voice
Kelas : 9
Kode Soal : 9.5
Kode Kategorisasi : 9.5.4

Rules:

Wh pronoun + has + he/she/it/(satu orang/benda) + V₃?

V₃ dari "read" = read

Nama orang = 'Alan'

Example: What has 'Alan' read recently?

Wh pronoun + have + I/you/we/they/(dua orang/benda) + V₃?

V₃ dari "reject" = rejected

Subject = 'they'
