Pembahasan Present Perfect Tense

Digunakan untuk menyatakan Kegiatan yg sudah selesai tapi masih ada kaitannya dengan masa sekarang.

Rumus Present Perfect

1. Nominal Sentence, kalimat yg TIDAK ADA kata kerja

(+) S + has/have + been + O/Complement (-) S + has/have + not + been + O/Complement (?) has/have + S + been + O/Complement?

2. Verbal Sentence, kalimat yang ada kata kerja

(+) S + Has/have + V3 + O/Complement (-) S + Has/have + not + V3 + O/Complement(?) Has/have + S + V3 + O/Complement?

Tambahan

Rumus Passive Voice dari Present Perfect(+) S + has/have + been + V3 (-) S + has/have + not + been + V3(?) has/have + S + been + V3?
