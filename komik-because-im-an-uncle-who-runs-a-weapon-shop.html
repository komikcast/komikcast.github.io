tenses: simple present

Kita identifikasi kalimatnya

Mrs.yusi runs the shop every weekend

mrs.yusni: subyek

runs: verb atau predikat

the shop : object

every weekend : keterangan waktu

ubah obyek menjadi subyek

the shop menjadi Mrs.yusi

jadi

the shop duluan lalu ditambahkan Tobe present is/ am/ are

the shop is run

run verb 3

lalu by subyek menjadi obyek

by mrs.yusi

keterangan waktu diikuti dibelakangnya

jadilah

the shop is run by Mrs. yusi every weekend

Penjelasan dengan langkah-langkah:

Bicycle has 2 wheels

Tri cycle has 3 wheels

41 bicycle means 41*2 = 82 wheels

let tricycle wheel = x

total wheels = 100

82+x = 100

x = 100-82

x = 18 wheels

since 1 tricycle has 3 wheels, then

18/3 = 6 tricycle
