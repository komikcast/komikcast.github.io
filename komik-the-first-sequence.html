18. the first term of arithmetic sequence is -4 and the 21th term is 116. 31th term of arithmetic sequence is...​


Jawaban:

176

Penjelasan dengan langkah-langkah:

Use this formula :

a_n = a_1 + (n - 1) * d

In this case, you're given:

• The first term (a_1) = -4

• The 21st term (a_21) = 116

Calculate the common difference (d) using the information given for the 1st and 21st terms, then solve the d.

a_21 = a_1 + (21 - 1) * d

116 = -4 + 20d

20d = 116 + 4

20d = 120

d = 120 / 20

d = 6

Now you can find the 31st term (a_31):

a_31 = a_1 + (31 - 1) * d

a_31 = -4 + 30 * 6

a_31 = -4 + 180

a_31 = 176

So, the 31st term of the arithmetic sequence is 176.

Arithmetic Sequence

• The nth term (Un)

Un = a + (n - 1) b

• The sum of the first n terms (Sn)

Sn = n/2 × (a + Un)

Sn = n/2 × (2a + (n - 1) b)

a = the first term

b = the common difference

==================================

S₁₀ = 125

U₃ = 5

a = ?

b = ?

S₁₅ = ?

S₁₀ = 125

10/2 × (2a + (10 - 1) b) = 125

5 × (2a + 9b) = 125

2a + 9b = 25 ... eq (1)

U₃ = 5

a + (3 - 1) b = 5

a + 2b = 5 ... eq (2)

To find a and b, use the elimination/substitution

▪︎Find the first term

Eliminate variable b to find a

2a + 9b = 25 (×2)

a + 2b = 5 (×9)

4a + 18b = 50

9a + 18b = 45

___________ -

-5a = 5

a = -1

The first term is -1

▪︎Find the common difference

To find the common difference, substitute a = -1 to eq (2)

a + 2b = 5

-1 + 2b = 5

2b = 5 + 1

2b = 6

b = 3

The common difference is 3

▪︎Find the sum of the first 15 terms

S₁₅ = 15/2 × [2(-1) + (15 - 1)(3)]

S₁₅ = 15/2 × [-2 + (14)(3)]

S₁₅ = 15/2 × (-2 + 42)

S₁₅ = 15/2 × 40

S₁₅ = 15 × 20

S₁₅ = 300

The sum of the first 15 terms is 300.
