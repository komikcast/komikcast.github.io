4. quiz bahasa jepang coba diartiin1.omae wa nan nin kanojo/kareshi desuka? 2.omae wa kanji o dekiru ka? 3.ore wa kanji o dekinai yo huaaaaaaa. ​


お前は何人彼女/彼氏ですか? (omae wa nan-nin kanojo/kareshi desuka?) : lu punya berapa banyak pacar?お前は漢字おできるか? (omae wa kanji o dekiru ka?) : lu bisa menggunakan kanji?俺は漢字できないよ (ore wa kanji o dekinai yo) : gue gak bisa menggunakan kanji lho.

Pembahasan

お前 (omae) dan 俺 (ore) merupakan kata ganti orang kedua tunggal, dan kata ganti orang pertama tunggal.

Kedua kata tersebut bersifat informal dan juga termasuk kata yang tidak sopan. Diartikan お前 (omae) : kamu, elu/lu. 俺 (ore) : aku, gue.

___________

___________

お前は何人彼女/彼氏ですか?

Omae wa nan-nin kanojo/kareshi desuka?

Lu punya berapa banyak pacar?

何人 ini ditulis dengan bentuk kanji, yang bisa dibaca nan-nin atau nanijin. Jika diartikan dalam bahasa Indonesianya itu berarti berapa banyak/berapa orang.

Adanya kata 彼女/彼氏 (kanojo/kareshi) dalam bahasa Indonesianya adalah kekasih/pacar. 彼女 (kanojo), sebenarnya kata ini tidak hanya diartikan kekasih tetapi juga bisa diartikan dia (pr).

Biasanya 彼女 (kanojo) ini digunakan oleh para lelaki yang memiliki pacar perempuan.

Sedangkan kata 彼氏 (kareshi) ini artinya pun juga sama seperti sebelumnya, namun kareshi adalah kebalikan dari kanojo. 彼氏 (kareshi) digunakan untuk para perempuan yang memiliki pacar lelaki.

Kata 彼氏 (kareshi) berasal dari kata 彼 (kare) yang berarti dia (lk).

----

お前は漢字おできるか?

Omae wa kanji o dekiru ka?

Lu bisa menggunakan kanji?

Kata できる (dekiru) dalam konteks kalimat ini diartikan sebagai bisa. Adanya penambahan か (ka) setelah できる (dekiru) menjadikan kalimat tersebut menjadi kalimat tanya yang menanyakan mengenai kemampuan/kesanggupan.

---

俺は漢字できないよ

Ore wa kanji o dekinai yo

Gue gak bisa menggunakan kanji lho

Terdapat kata できない (dekinai) yang artinya adalah tidak/gak bisa (menyatakan rasa tidak kesanggupan atau tidak mampu). できない (dekinai) merupakan kategori kata yang informal.

Selain itu terdapat partikel よ (yo) yang biasanya terletak di sekitar akhir kalimat yang umumnya memiliki arti lho atau juga lah yang berfungsi agar kalimat tersebut memiliki penekanan.

7. watashi : ara ara : gomenasai :dame : ​


Watashi (私), dalam bahasa Indonesia artinya "Saya"

Contohnya: "Saya adalah orang Jepang" dalam bahasa Jepangnya "Watashi wa nihon-jin desu"

___________________________________

Ara ara (あらあら), dalam bahasa Indonesia bisa diartikan sebagai "Oh", "Wah", "Astaga", "Ya ampun", "Hah" dan "Wow".

Biasanya menunjukkan ekspresi keterkejutan, ketidakpercayaan, kagum dan kegelian.

Contohnya: "Ya ampun, maaf" atau "Wah, maaf" dalam bahasa Jepangnya "Ara ara, gomen"

___________________________________

Gomenasai (ごめんなさい), adalah bentuk formal dari kata gomen yang artinya permintaan maaf.

Contohnya: "Maafkan saya, senior" diartikan "Gomenasai, senpai"

___________________________________

Dame (だめ), bisa diartikan "Tidak boleh"

Ini adalah kata informal. Anda bisa menggunakannya kepada kawan, orang yang lebih muda atau bawahan Anda.

Adapun dalam bahasa gaul ini bisa diartikan "Jangan" atau bisa jadi "Payah".

Contohnya: "Tidak boleh makan!" diartikan "Tabete wa dame!"

17. mau nanya perbedaan "kanojo" sama "shoujo"" itu apa


Kanojo ( 彼女 ) = Dia ( perempuan )
Onna ( 女 ) = Perempuan ( universal )
Oujo ( 王女 ) = Tuan Putri
Shoujo ( 少女 ) = Perempuan ( muda )Perbedaannya:
↪Kanojo (かのじょ): bisa diartikan sebagai "pacar perempuan".
↪Shoujo (しょうじょ): Cewek / perempuan (bisa dikatakan ini bahasa nonformal)

Cara membedakan nya gampang, hanya perlu menghapal kosa katanya.
↪かれし >< かのじょ(Kareshi >< kanojo) : Pacar laki-laki - pacar perempuan
↪男の子 >< 女の子 (otokonoko >< onnanoko): anak laki2 - anak perempuan
おとこのひと >< おんなのひと (otokohito >< onnanohito): laki-laki - perempuan

20. apa arti "DAME TU COSITA"?


DAME TU COSITA sebenarnya adalah lagu dari penyanyi Panama bernama El Chambo. Lagu Dame Tu Cosita ini dirilis pada tahun 2011 namun baru viral di tahun 2017/2018.

Dame Tu Cosita dalam arti harfiahnya adalah GIVE ME YOUR (LITTLE) THING atau Berikan aku hal kecilmu. Namun dalam artian kiasan, kalimat ini memiliki makna yang cenderung menjurus ke hal dewasa. Dame Tu Cosita digunakan untuk meminta izin menyentuh bagian tubuh yang dalam ukuran kecil.

Dalam Urban Dictionary, disebutkan bahwa COSITA dalam bahasa pergaulan Latin diartikan sebagai organ seksual wanita. Meski artian harfiahnya adalah CUTE LITTLE THING, namun kata ini umum digunakan untuk menunjuk pada organ vital. 

Oleh sebab itu, meski video ini viral dan terdengar ‘ramah’ di telinga, namun kita wajib untuk kritis dan tidak menyanyikannya sembarangan. Terlebih saat kita tahu makna lagu ini menjurus ke hal dewasa yang bertentangan dengan nilai ketimuran yang kita anut.

'Dame dame' artinya adalah...

Jangan, jangan Ga boleh, ga boleh Ngga, ngga

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Pembahasan :

DAME「駄目・だめ・ダメ」

Memiliki arti "Jangan", "Ngga boleh", "Ngga bagus", "Payah", dsb.

Kanji = 駄目 Hiragana = だめ Katakana = ダメ

Kata ini jarang ditulis dalam Kanji karena terlalu sulit, dan lebih sering ditulis dalam Hiragana atau Katakana saja yang lebih mudah. Penulisan dalam Katakana memiliki kesan kasual dan informal. Sedangkan penulisan dalam Hiragana sopan atau biasa saja.

Contoh kalimat :

だめです (Dame desu) : Tidak boleh絶対ダメだ！ (Zettai dame da!) : Pokoknya ngga boleh!これはだめだね (Kore wa dame da ne) : Ini ngga bagus俺なんてダメだ (Ore nante dame da) : Aku ini payah俺じゃダメかな (Ore ja dame ka na) : Apa aku tidak cukup untukmu

Selain 'Dame', ada satu kata lagi yang memiliki arti dan penggunaan yang sama, yaitu adalah あかん (Akan). Kata ini merupakan versi dialek Kansai dari 'Dame'. Dialek Kansai digunakan di daerah barat Jepang, seperti Osaka, Kyoto, Kobe, dll.

- - - - - - - - -

Arti 'Dame dame'?

Arti dari 'Dame' sendiri sudah saya jelaskan di atas. Namun jika lebih dari satu, seperti 'Dame dame' atau 'Dame dame dame', ini lebih seperti ke ungkapan larangan. Kalau diartikan seperti "Jangan, jangan", "Ga boleh, ga boleh", dan sebagainya.

38. Dame Dame Dame Yo...Artinya apa​


'Dame dame dame yo' artinya dalam bahasa Indonesia berarti jangan, jangan, jangan lho.

Pembahasan

Dame「駄目・だめ・ダメ」

Jika dalam bahasa Indonesia memiliki arti sebagai berikut ini :

Jangan.Gak.Gak bagus.Payah.Gak boleh.

Kata dame ini lebih umum ditulis dengan bentuk versi hiragana atau juga katakana, sehingga jarang menggunakan versi kanji. Kemudian untuk penggunaannya sesuai situasi tergantung dari versi penulisannya :

Versi hiragana : だめ (dame) -› informal.Versi katakana : ダメ (dame) -› sopan/halus.

yo「よ」

Partikel yang dapat berfungsi ketika ingin menegaskan atau menekankan kalimat di akhir, sehingga jika partikel yo diterjemahkan dalam bahasa Indonesia yang berarti kok, lho, atau lah.
