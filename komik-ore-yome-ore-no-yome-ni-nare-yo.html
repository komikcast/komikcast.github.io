13. Apa Artinya "Omae wa, ore no nakamada!"​


Arti kalimat bahasa Jepang "omae wa ore no nakama da" adalah "elu adalah rekan gue."

Pembahasan

Omae wa ore no nakama da

「お前は俺の仲間だ」

╰➝ = elu adalah rekan gue

Kalimat diatas tersebut pernah diucapkan oleh salah satu karakter utama dalam serial One Piece yaitu Monkey D Luffy.

Berikut penjelasan mengenai setiap kata-perkatanya dibawah ini :

Omae「お前」

Ini merupakan kata ganti kedua tunggal dengan versi yang kasar, jika dalam bahasa Indonesianya kata omae setara dengan kata "elu."

Wa「は」

Partikel untuk penanda dalam kalimat.

Ore「俺」

Kata ganti pertama tunggal dengan versi yang kasar, setara dengan kata "gue" dalam bahasa Indonesia.

No「の」

Partikel penghubung atau menyatakan hal kepemilikan.

Nakama「仲間」

Artinya tersebut ialah rekan, rekan yang membantu satu dengan yang lainnya dalam hal yang berhubungan pekerjaan.

Da「だ」

Partikel dengan fungsi penegas, yang kerap digunakan hanya untuk bagi lelaki saja.

15. arti dari Ore no atama ga okashi?​


ore no atama ga okashii dalam bahasa Indonesia diartikan gue gila.

Pembahasan

俺の頭がおかしい

Ore no atama ga okashii

Gue gila.

Sebenarnya, kalimat 頭がおかしい (atama ga okashii) ini kalau diartikan dalam setiap kata-katanya yaitu :

頭 (atama) : kepala.おかしい (okashii) : aneh.

Digabungkan menjadi kepalanya aneh. Kemudian kalimat ini dijadikan suatu istilah-istilah yang merujuk ke arti gila. Dikarenakan seperti ibaratnya seseorang yang memiliki kepala (pemikiran) yang aneh (tidak jelas, tidak masuk akal).

俺の頭がおかしい (ore no atama ga okashii) jika dalam artiannya tidak sesuai dengan istilah, maka seperti kepala gue aneh.

Jika sesuai dengan istilahnya diartikan seperti gue gila.

________

Kebanyakan orang Jepang lebih banyak mengartikan 頭がおかしい (atama ga okashii) dengan istilah gila.

________

おかしい (okashii)

Artinya adalah aneh, tetapi juga bisa diartikan lucu (sesuai konteks). Salah satu kata ini merupakan jenis kategori golongan kata i-keiyoushi.

20. Q: Apa bedanya Watashi dan Ore?


Apa bedanya Watashi dan Ore?

Kedua kata ini sama-sama merupakan kata ganti orang pertama. Namun, terdapat perbedaan besar pada tingkat kesopanan dan penggunaannya, sebagai berikut.

私 (Watashi) : Termasuk sopan, dapat digunakan ke siapa saja, dan oleh siapa saja. Kalau dalam Bahasa Indonesia, penggunaannya mirip seperti "Saya" atau "Aku".俺 (Ore) : Termasuk kasar, biasa digunakan laki-laki, dan hanya digunakan ke teman saja. Kalau dalam Bahasa Indonesia, penggunaannya mirip seperti "Gua".

27. Apa bedanya “ore” sama “boku”


Ore : gue, aku. Jenis kata ganti yang termasuk bentuk kasar (tidak sopan).Boku : aku, saya. Kata ganti yang termasuk jenis sopan dan juga halus.

Pembahasan

Untuk persamaan diantara kedua kata diatas yaitu ore dan boku adalah bahwa kedua kata ini adalah golongan kata ganti orang pertama yang tunggal.

Ore「俺・おれ」

Termasuk kata ganti yang tergolong bentuk kasar, sehingga penggunaannya hanya digunakan dalam situasi yang sangat informal. Tidak bisa diucapkan dalam situasi yang formal-semi formal, kemudian kata 俺 (ore) ini hanya digunakan oleh pria, tidak digunakan untuk perempuan.

Contoh penggunaannya dalam bentuk kalimat yaitu :

Ore wa tensai da : gue jenius.Ore wa maketa : gue kalah.Omae wa ore no nakama da : elu adalah rekan gue.

-----------------

Boku「僕・ほく」

Walau penggunaannya hanya digunakan ketika dalam situasi yang semi formal-non formal, akan tetapi kata ini merupakan bentuk kata yang bersifat sopan serta halus.

Pengucapan kata 僕 (boku) sering digunakan oleh para lelaki, khususnya anak muda bisa juga untuk pria dewasa. Akan tetapi kata ganti ini juga bisa diucapkan bagi perempuan yang merasa dirinya tomboy.

Contoh penggunaannya dalam bentuk kalimat yaitu :

Boku wa kimi ga suki dakara : karena aku suka sama kamu.Boku wa hansamu desu ne : aku ganteng ya.Boku no shumi wa supootsu desu : hobi aku berolahraga.

Perbedaan 俺 (Ore) dan 僕 (Boku) terdapat pada tingkat kesopanannya. Kata Ore terkesan kasar, sedangkan Boku sopan. Namun kedua kata tersebut sama-sama biasa digunakan di situasi informal.

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Pendahuluan :

人称代名詞 (Ninshou Daimeishi)

Berarti "Kata ganti orang". Merupakan kata-kata yang digunakan untuk menunjukkan orang. Sama hal nya dengan yang Bahasa Indonesia, dalam Bahasa Jepang kata ganti orang juga dibagi menjadi kata ganti orang pertama, kedua, dan ketiga, yang masing-masingnya dibagi lagi menjadi tunggal (satu) dan jamak (lebih dari satu).

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Pembahasan :

Yang akan saya bahas kali ini hanya kata ganti orang pertama tunggal saja. Kata-kata yang saya sebutkan ini hanya beberapa saja, yang cukup sering didengar dan digunakan, sebagai berikut.

WATASHI「私・わたし」

Penggunaannya seperti "Saya" dan "Aku". Kata ini termasuk sopan, dan paling umum digunakan. Bisa digunakan di situasi formal maupun informal. Bisa digunakan oleh laki-laki maupun perempuan. Bisa digunakan oleh orang yang sudah tua maupun yang masih muda.

BOKU「僕・ぼく」

Penggunaannya seperti "Aku". Kata ini termasuk sopan, tapi hanya digunakan di situasi semi-formal sampai informal. Umumnya digunakan oleh laki-laki saja. Kalau digunakan perempuan, biasanya perempuan tersebut tomboy. Kata ini tidak biasa digunakan oleh orang yang sudah tua, jadi biasa digunakan oleh orang yang masih muda saja.

ORE「俺・おれ」

Penggunaannya seperti "Gua". Kata ini termasuk kasar, dan jelas hanya digunakan di situasi informal saja. Sama seperti Boku, kata ini umumnya digunakan oleh laki-laki saja, dan kalau digunakan perempuan, biasanya perempuan tersebut tomboy. Kata ini juga tidak biasa digunakan oleh orang yang sudah tua, jadi biasa hanya digunakan oleh orang yang masih muda saja.

ATASHI「あたし」

Penggunaannya seperti "Aku". Kata ini tidak sopan, namun juga tidak kasar. Kata ini hanya digunakan di situasi informal saja. Kata ini merupakan semacam plesetan dari kata Watashi. Tujuan memplesetkannya yaitu agar terkesan imut. Karena itu umumnya digunakan oleh perempuan saja. Jika digunakan laki-laki, maka dia kesannya seperti banci. Kata ini biasa hanya digunakan oleh orang yang masih muda, khususnya gadis SMA.

私(わたくし) watakushi = Ini yang paling formal. Jarang digunakan untuk percakapan verbal. Hanya digunakan untuk pidato/ceramah yang sangat formal, seperti pidato Presiden, dll.

私(わたし) watashi = Penggunaannya sepenuhnya "saya" dalam bahasa Indonesia. Formal hingga semi formal. Baik pria maupun wanita dapat menggunakannya.

僕(ぼく) boku = Penggunaannya sama dengan "aku" dalam bahasa Indonesia. Informal tapi masih cukup sopan. Terutama untuk pria.  aneh jika wanita menyebut diri mereka "boku". Ini Sering digunakan dalam percakapan sehari-hari.

俺(おれ) ore =Penggunaannya sama dengan "gue". Agak kasar. Haram digunakan perempuan.  Itu hanya untuk digunakan oleh teman atau teman yang sudah dekat, meskipun ada juga beberapa orang Jepang yang tidak suka menggunakan "ore", dan bahkan lebih suka untuk tidak menggunakannya.

私’(あたし)atashi : ini adalah "僕(ぼく) boku" tapi versi perempuannya.kanji atashi sama dengan watashi dan watakushi.

ada juga yang lain untuk "kata ganti orang pertama"

自分(じぶん) jibun = Mirip dengan "boku".

うち uchi: Mirip dengan "boku" tapi untuk perempuan.

当方(とうほう) touhou = Sangat formal seperti  私(わたくし) watakushi. biasanya untuk  surat menyurat bisnis.

わし washi = digunakan oleh orang-orang tua (kakek nenek).

33. Apa perbedaan dari ore, boku, dan watashi? Apakah ore hanya diucapkan oleh laki2? Berikan alasanya


Jawaban:

1.Watashi

Pengguna : Cowok/Cewek

Analogi bahasa Indonesia : Saya

Penjelasan :

Walo kata ini adalah kata yang paling umum untuk menyatakan kata "saya", tapi untuk konteks kalimat "watashi mo" yang berarti "aku juga", kata tersebut bersifat cewek banget. Jadi, buat cowok pengguna kata watashi, baiknya diganti katanya jadi pilihan yang lain khusus buat kalimat itu ajah.  

2.Boku

Pengguna : Cowok/Cewek tomboy

Anologi bahasa Indoneisia : Gua dan Lu

Penjelasan :

Kata ini digunakan cowok sebagai sapaan sehari-hari. Kesan yang disampaikan sopan, ramah dan gak terlalu formal. Kata boku sebenernya punya arti harfiah "manservant" (maksudnya pelayan), yang membuat pemakainya seolah merendahkan diri dari lawan bicara tanpa menghilangkan kesan maskulin dari yang berbicara. Bisa dibilang ini soft-masculine buat dipake ama cowok.

3.Ore/Oira

Pengguna : Cowok

Anologi bahasa Indonesia : Gue

Penjelasan :

Walaupun ini setara ama boku di kalangan cowok, yang menimbulkan suasana santai, gak resmi dan kesan keakraban, tapi sapaan ini bukan kata yang sopan sebenernya. Kalo dianalogiin di bahasa sunda, ini mirip ama kata "aing" (kata teman dari bandung, demi apapun, kamu jangan sekali-kali menyebut kata ini di tanah sunda).

Kondisi terlarang buat memakai kata ini, yaitu pas lagi bicara sama atasan atau sama yang kamu hormati. Kecuali kalo kamu mau langsung dipecat atau gak di-respect-in lagi. Bisa dibilang ini versi hard-masculine dari kata boku.

Kalo "Oira" itu logat lain dari ore. Banyak digunakan sama orang tokyo lama (adanya nih kata2 muncul di zaman edo, tapi masih ada yang ngegunain sampe sekarang).

40. bokuorewatashi............​


Kata boku, ore, dan watashi artinya adalah aku, gue, dan saya.

Pembahasan

Untuk mengetahui perbedaan antara boku, ore, dan watashi bisa dilihat sebagai berikut :

=> 僕 (boku)

Artinya adalah aku, namun salah satu kata ini hanya umumnya digunakan oleh para lelaki. Dan termasuk ke dalam kategori kata ganti dalam bentuk yang kurang formal (informal), tapi masih dibilang cukup sopan.

Sebenarnya untuk perempuan juga bisa menggunakan kata ini, dengan tujuan jika perempuan itu ingin terlihat tomboy.

=> 俺 (ore)

Merupakan salah satu bentuk kata yang tergolong tingkatan agak tidak sopan atau kasar. Dikarenakan kata ini artinya dalam bahasa Indonesia adalah gue.

Gue dalam bahasa Indonesia termasuk kata tidak baku dan tidak sopan, begitupun juga 俺 (ore) dalam bahasa Jepang.

=> 私 (watashi)

Bentuk kata ganti yang bisa diartikan aku/saya. Karena kata watashi ini termasuk jenis kata yang tergolong termasuk kategori formal atau sopan, sehingga boleh kita gunakan dalam situasi formal atau resmi.
