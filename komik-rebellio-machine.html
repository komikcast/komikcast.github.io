16. Apa itu Machine Learning? Apa kaitannya berpikir komputasional dengan Machine Learning?


Machine learning adalah penerapan kemampuan manusia akan cara belajar dan berpikir komputasi pada sebuah mesin komputer. Berpikir komputasional berkaitan dengan machine learning karena machine learning ditrain untuk menyelesaikan masalah sehingga mereka akan mampu belajar dari masalah-masalah tersebut dan dapat menemukan solusinya.

Penjelasan:

Pembelajaran mesin adalah bidang kecerdasan buatan (AI) dan ilmu komputer, dengan fokus pada penggunaan data dan algoritme untuk meniru metode pembelajaran manusia  dan secara bertahap meningkatkan akurasi. Pembelajaran mesin dapat didefinisikan sebagai proses pembelajaran mesin  dengan menganalisis sejumlah besar data. Misalnya, Anda tidak perlu lagi  menulis program baru untuk mengetahui  wajah kucing dan manusia. Anda dapat melatih mesin Anda dengan beberapa gambar untuk membuatnya sadar akan objek. Tujuannya adalah untuk dapat mengambil keputusan atau membuat prediksi  berdasarkan data yang tersedia. Semakin baik algoritma pembelajaran mesin, semakin baik akurasi keputusan dan  prediksi sistem.

32. → Which machine will create the highest value improvement to MyCompany's productivity? * (assuming the machine lasts for 3 years and they use all machine capacity) Labor cost per month (in Rp) Capacity per month Average value per items (in Rp) Machine price (in Rp) Old Machine Machine A Machine B Machine C 5,000,000 5,000,000 5,000,000 5,000,000 200 410 385 280 80,000 80,000 80,000 80,000 470,000.000 400.000.000 150,000.000​


Machine A will create the highest value improvement to MyCompany's productivity, with a net benefit of Rp 780,800,000 over the 3-year lifespan.

Pembahasan

To determine which machine will create the highest value improvement to MyCompany's productivity, we need to calculate the net benefit for each machine. The net benefit is obtained by subtracting the machine price from the total value generated during its lifespan.

Given the information provided:

Lifespan: Assuming the machine lasts for 3 years.

Labor cost per month: Rp 5,000,000.

Capacity per month: The number of items produced per month with each machine.

Old Machine: 200 items per month.

Machine A: 410 items per month.

Machine B: 385 items per month.

Machine C: 280 items per month.

Average value per item: The average value generated per item produced.

Old Machine: Rp 80,000 per item.

Machine A: Rp 80,000 per item.

Machine B: Rp 80,000 per item.

Machine C: Rp 80,000 per item.

Machine price:

Old Machine: Rp 470,000,000.

Machine A: Rp 400,000,000.

Machine B: Rp 150,000,000.

Machine C: Rp 80,000,000.

Now, let's calculate the net benefit for each machine:

Net Benefit for Old Machine:

Total value generated in 3 years = (200 items/month * Rp 80,000/item) * 12 months/year * 3 years = Rp 576,000,000.

Net Benefit = Total value generated - Machine price = Rp 576,000,000 - Rp 470,000,000 = Rp 106,000,000.

Net Benefit for Machine A:

Total value generated in 3 years = (410 items/month * Rp 80,000/item) * 12 months/year * 3 years = Rp 1,180,800,000.

Net Benefit = Total value generated - Machine price = Rp 1,180,800,000 - Rp 400,000,000 = Rp 780,800,000.

Net Benefit for Machine B:

Total value generated in 3 years = (385 items/month * Rp 80,000/item) * 12 months/year * 3 years = Rp 1,108,800,000.

Net Benefit = Total value generated - Machine price = Rp 1,108,800,000 - Rp 150,000,000 = Rp 958,800,000.

Net Benefit for Machine C:

Total value generated in 3 years = (280 items/month * Rp 80,000/item) * 12 months/year * 3 years = Rp 806,400,000.

Net Benefit = Total value generated - Machine price = Rp 806,400,000 - Rp 80,000,000 = Rp 726,400,000.
