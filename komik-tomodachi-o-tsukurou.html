4. Susunlah menjadi kalimat tepatde-tomodachi-kantin-tabemasu-o-to-nasi goreng


Jawaban:

友達とカンテインでナシゴレンを食べます

Tomodachi to kantin de nasi goreng o tabemasu

(saya) makan nasi goreng bersama teman di Kantin

Penjelasan:

友達（Tomodachi): teman-teman

カンテイン (kant(e)in): Kantin

atau dalam bahasa Jepang nya食堂(shokudou)

ナシゴレン(nashi goren):nasi goreng

atau dalam bahasa Jepang nya 炒飯(chāhan) atau serapan dari bahasa Inggris フライドライス (furaido raisu)

を(o): partikel dalam bahasa Jepang yg menunjukkan objek suatu kalimat

で(de): partikel dalam bahasa Jepang yg artinya bisa di dan dengan

食べます(tabe-masu):makan(formal)

と(to): partikel dalam bahasa Jepang yg menunjukkan (bersama) dengan seseorang

juga bisa diartikan (dan)

semoga membantu

Susunan kalimat yang tepat adalah :

Kantin de tomodachi to nasi goreng o tabemasu.

PENJELASAN :

Kotoba :

友達 (Tomodachi) = Teman食べます (Tabemasu) = Makan

Partikel :

で (De) = Bisa berarti "di", "dengan", dllと (To) = Bisa untuk kata hubung "dan", "dengan", dllを (O) = Menunjukkan objek dari kata kerja

Jadi, bila kalimat tersebut diterjemahkan, artinya adalah :

= Makan nasi goreng dengan teman di kantin.

Dan bila ditulis dalam huruf Jepang menjadi :

= Kantinで友達とNasi gorengを食べます。

35. 4. Susunlah kata-kata berikut agar menjadi kalimat yang benar! Tomodachi/o/hiru-gohan/to/tabemasu/yoku​


Jawaban :

Susunan kata-kata tersebut agar menjadi kalimat yang benar adalah Yoku tomodachi to hirugohan o tabemasu. Kalimat ini dalam Bahasa Indonesia artinya "(Saya) sering (makan) makan siang dengan teman".

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Pembahasan

Struktur kalimat dalam Bahasa Jepang sedikit berbeda dengan Bahasa Indonesia. Urutan struktur kalimat dalam Bahasa Jepang bisa bermacam-macam. Dalam kalimat Bahasa Jepang, predikat selalu diletakkan di akhir.

Dalam kalimat Bahasa Jepang juga sering terdapat kata bantu atau biasa disebut partikel. Partikel ini jumlahnya ada banyak dan masing-masing memiliki fungsinya tersendiri.

-

PEMBAHASAN SOAL

Urutan struktur kalimat :

S ➞ K ➞ O ➞ P

S : SubjekK : KeteranganO : ObjekP : Predikat

*Orang Jepang jarang menyebutkan subjek jika subjeknya sudah diketahui oleh lawan bicara, jadi bagian S (Subjek) bisa dihilangkan.

Kosakata :

Tomodachi = TemanHirugohan = Makan siangTabemasu = MakanYoku = Sering

Partikel :

Partikel O : Untuk menandakan objek. Biasa diletakkan di antara kata benda atau objek dengan kata kerja.Partikel To : Untuk menunjukkan kata yang dimaksud sebagai lawan yang diajak. Kalau diartikan seperti "Dengan" atau "Bersama". Misalkan pada kalimat 'A to', maka artinya seperti "Dengan A" atau "Bersama A".

===

Untuk membuat kalimatnya, dimulai dari subjek. Tapi karena subjeknya tidak ada, maka langsung ke keterangannya. Yang berperan sebagai keterangannya adalah Yoku (sering) dan Tomodachi to (Dengan teman). Kemudian untuk objek, yaitu Hirugohan (makan siang). Dan predikatnya adalah Tabemasu (makan).

Jika disusun menjadi kalimat, akan menjadi :

=> Yoku tomodachi to hirugohan o tabemasu.

=> Sering makan siang bersama teman.

===

Jawaban dan Penjelasan:

1). sinonim dari bahasa jepang "tomodachi" yaitu?

友達 (tomodachi) = *dijelaskan dibawah

仲間(nakama) = *dijelaskan dibawah

友人 (yuujin) = artinya teman (cuma 1 orang)

親友 (shinyuu) = teman yang sudah sangat sangat sangat dekat

dan apa arti tomodachi?

友達 (tomodachi) artinya teman-teman

2). perbedaan dari "tomodachi" dan "nakama"  

Ini masalah derajat pertemanan.友達 (tomodachi) adalah siapa saja yang sering kamu ajak bergaul;仲間(nakama) adalah hubungan yang lebih dekat, hampir seperti 'seseorang yang kamu pilih untuk berbagi hidup'. 仲間(nakama) juga tampak memiliki pengertian 'melakukan sesuatu bersama-sama' atau 'bekerja menuju tujuan bersama' yang tidak ada pada 友達 (tomodachi) . penjelasan lebih lengkapnya =

友達 (tomodachi) = Seorang teman. tidak perlu melakukan sesuatu yang istimewa dengan mereka. Mereka bisa menjadi seseorang yang sudah lama tidak kamu temui.

仲間(nakama) = Orang-orang yang memiliki tujuan yang sama dan bekerja/berjuang/berjuang bersama dalam suatu kelompok atau organisasi(temen sekolah,les,pokonya  gitu). Mereka sering kali bisa menjadi teman dekat juga, tetapi itu tidak perlu. Seseorang yang tidak kamu sukai secara pribadi, atau yang bahkan tidak kamu kenal, terkadang bisa menjadi kamu :)
