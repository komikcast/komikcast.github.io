15. Apa Arti Wo Ai Ni Dan Wo Bu Ai Ni


Arti dari Wǒ ài nǐ [ 我爱你 ] adalah Aku mencintaimu.

Arti dari Wǒ bù ài nǐ [ 我不爱你 ] adalah Aku tidak mencintaimu.

PEMBAHASAN

Dalam Bahasa Mandarin, kita akan mengenal istilah [ 肯定句 ] Kěndìng jù, [ 否定句 ] Fǒudìng jù, dan [ 疑问句 ] Yíwènjù. [ 肯定句 ]. Kěndìng jù secara umum berarti Kalimat Positif, yaitu kalimat yang menyatakan persetujuan terhadap suatu pernyataan. [ 否定句 ] Fǒudìng jù secara umum berarti Kalimat Negatif, yaitu kalimat yang menyatakan penentangan terhadap suatu pernyataan. [ 疑问句 ] Yíwènjù secara umum berarti Kalimat Interogatif, yaitu kalimat yang menanyakan suatu pernyataan. Rumus pembentukan kalimat dari ketiga konsep diatas adalah:

[+] 代词 + 形容词 / 动词 + 宾语 + 副词

[-] 代词 + 不 + 形容词 / 动词 + 宾语 + 副词

[?] 代词 + 形容词 / 动词 + 宾语 + 副词 + 吗？

Keterangan:

代词 : Subjek.形容词 : Kata Sifat.动词 : Kata Kerja.宾语 : Objek.副词 : Kata Keterangan (Jika ada)不 : tidak (menyatakan pertentangan).吗？: Apakah? (menyatakan pertanyaan).

CONTOH:

[+] 我爱你 (Wǒ ài nǐ )

Aku mencintaimu.

[-] 我不爱你 (Wǒ bù ài nǐ)

Aku tidak mencintaimu.

[?] 我爱你吗？(Wǒ ài nǐ ma?)

Terjemahan ↴

Rumah saya besar. Di rumah adalah 3 kamar. Di rumah ada ruang tamu, ruang keluarga, dapur, dan kamar mandi. Juga ada taman. Di kamar saya ada 1 tempat tidur. Saya belajar di kamar. Pembahasan

Berikut beberapa kosakata yang ada di teks tersebut, beserta artinya dalam Bahasa Indonesia.

Uchi = RumahHeya = KamarKyakuma = Ruang tamuIma = Ruang keluargaDaidokoro = DapurBasuruumu = Kamar mandiNiwa = Taman

Teks tersebut menjelaskan tentang rumah dari si penulis. Kalimat-kalimat pada teks kebanyakan menyatakan keberadaan suatu benda. Untuk menyatakan keberadaan suatu benda dalam Bahasa Jepang, dapat menggunakan pola kalimat berikut.

=> [...] ga arimasu.

=> Ada [...].

Contoh kalimat :

Heya ga arimasu (Ada kamar)Niwa ga arimasu (Ada taman)Reizouko ga arimasu (Ada kulkas)Terebi ga arimasu (Ada TV)

言葉（kotoba：kosakata）

Watashi : saya

uchi : rumah (untk menyebut rumah sendiri)

ookii : besar (kata sifat i)

heya : kamar

mittsu : tiga buah

arimasu : ada (konteks formal, ada untuk benda mati termasuk tumbuhan)

kyakuma : ruang tamu

ima : ruang keluarga

daidokoro : dapur

basuru-mu : kamar mandi

beddo : tempat tidur

hitotsu : satu buah

benkyou : pelajaran, belajar ( kata kerja ke 3)

shimasu: melakukan (konteks formal)

eakon : AC

terebi : tv

ichidai : satu unit/mesin

Reizouko: kulkas

suto-bu : kompor (stove)

Kazoku : keluarga

gohan : nasi

tabemasu: makan (formal, kata kerja ke 2)

kuruma : mobil

___________

助詞（joshi：partikel）

は(ha, tp disini ditulis jadi partikel jadi cara bacanya wa): menunjukkan topik suatu kalimat

を(wo, karna dijadikan partikel jadi bacanya jadi o): menunjukkan objek

と(to): dan, bersama(orang dan hewan)

へ(he, karna dijadikan partikel jadi bacanya e): menunjukkan arah perginya

に: di, pada, kepada

で(de): di, dengan(benda, kendaraan)

も(mo): juga

-------++---

Pola Kalimat

KB¹ + は(wa) + KB²(tempat) + に(ni) + あります(arimasu)

: KB¹ ada di KB²

KB¹ + に(ni) + KB² + が(ga) + あります(arimasu)

: Di KB¹ ada KB²
